## 修复内容

### 1. 🍅 番茄钟 (pomodoro.js)

- 确保与系统整体UI风格一致
- 数据保存到 localStorage

### 2. 🧮 计算器 (calculator.js) - 新建
- 实现基础的加减乘除计算器功能

- UI风格与系统保持一致
- 包含返回按钮

### 3. 📝 记事本 (notepad.js) - 新建
- 实现简单的本地存储记事本
- 支持添加标题和内容

- 笔记列表按时间倒序显示
- 所有数据保存到 localStorage
- 包含返回按钮

### 4. 📊 AI使用统计
- 保留原有的模态框显示方式
- 添加 renderUsageStats 函数支持全屏页面路由
- 确保功能正常可用

## 路由配置 (ui.js)

在 `openFullscreenPage` 函数中添加了以下路由：
```javascript
case 'pomodoro': renderPomodoro(contentEl); break;
case 'calculator': renderCalculator(contentEl); break;
case 'notepad': renderNotepad(contentEl); break;
case 'usage-stats': renderUsageStats(contentEl); break;
```

## 按钮调用 (my-page.js)

更新了实用工具模块的按钮点击函数：
- `openPomodoro()` → 调用 `openFullscreenPage('pomodoro')`
- `openCalculator()` → 调用 `openFullscreenPage('calculator')`
- `openNotepad()` → 调用 `openFullscreenPage('notepad')`
- `openUsageStats()` → 保持原有模态框方式

## 文件加载顺序 (index.html)

在 JS 加载顺序中添加了新模块：
- `js/modules/calculator.js`
- `js/modules/notepad.js`
