## 🎮 第1次训练：数学图形扫描

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
屏幕会快速显示一组数学图形，你需要：
1. 快速扫视所有图形
2. 说出指定形状的数量
3. 记录你的正确率

### 今日题目

```
┌─────────────────────────────────────────────────┐
│  观察下面的图形，数一数有多少个圆形？            │
│                                                 │
│     ●    △    ○    □    ●    △    ◇    ○       │
│     □    ●    ◇    △    ○    ●    □    △       │
│     △    ○    ●    ◇    ●    △    ○    ●       │
│                                                 │
│  ⏱️ 限时：15秒                                  │
│  你的答案：_____                                │
└─────────────────────────────────────────────────┘
```

**答案**：圆形（●和○）共有 8 个

### 挑战升级
- 🌟 基础：正确数1种图形
- ⭐⭐ 进阶：正确数2种图形
- ⭐⭐⭐ 大师：说出每种图形的数量

## 🧠 第2次训练：数字密码记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
1. 记忆一串数字密码
2. 回忆并默写出来
3. 挑战更长的密码

### 今日密码

**Level 1（入门）**
```
📱 手机密码：6 → 3 → 8 → 1 → 9
记忆时间：30秒
尝试回忆并写下：_______________
```

**Level 2（挑战）**
```
📱 保险柜密码：4 → 7 → 2 → 9 → 5 → 1
记忆时间：45秒
尝试回忆并写下：_______________
```

### 记忆技巧
💡 **联想记忆法**：把数字变成故事！
- 例如：6像哨子，3像耳朵，8像葫芦...
- 试着把你的密码编成一个小故事！

## 🃏 第6次训练：数学符号配对

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
记忆数学符号和它们的含义，然后配对。

### 今日记忆卡

**符号记忆表**
```
┌─────────┬──────────────────┬────────────┐
│  符号   │      含义        │   例题     │
├─────────┼──────────────────┼────────────┤
│    +    │     加法         │  3 + 5 = ? │
│    -    │     减法         │  9 - 4 = ? │
│    ×    │     乘法         │  6 × 2 = ? │
│    ÷    │     除法         │  8 ÷ 4 = ? │
│    =    │     等于         │  5 = ?     │
│    >    │     大于         │  7 > 3     │
│    <    │     小于         │  2 < 8     │
└─────────┴──────────────────┴────────────┘

⏱️ 记忆时间：60秒
```

### 配对挑战
根据含义，写出对应符号：

1. "把东西合起来" 用 _____ 表示
2. "去掉一些东西" 用 _____ 表示
3. "很多个一样的加在一起" 用 _____ 表示
4. "平均分东西" 用 _____ 表示
5. "两边一样多" 用 _____ 表示

**答案检查**：+  -  ×  ÷  =
