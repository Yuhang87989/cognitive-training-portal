## 🔬 第1次训练：定量实验设计思维

**模块**：科学方法论  
**时长**：约2.5分钟

### 从定性到定量

科学的本质是用数学语言描述自然规律。今天我们强化定量实验的设计能力。

### 定量实验的关键要素

**1. 自变量的精确控制**
- 不仅改变变量，还要量化
- 例："增加温度"→"将温度从20°C升至30°C"

**2. 因变量的准确测量**
- 选择合适的测量工具
- 明确测量精度
- 重复测量取平均值

**3. 控制变量的系统管理**
- 列出所有可能影响结果的变量
- 逐一分析哪些需要控制
- 确定控制方法

### 误差分析基础

**系统误差 vs 随机误差**

| 类型 | 特点 | 来源 | 减小方法 |
|-----|------|------|---------|
| 系统误差 | 单向偏移，可重复 | 仪器偏差、方法缺陷 | 校准仪器、改进方法 |
| 随机误差 | 散布在真值两侧 | 环境波动、测量不确定 | 多次测量取平均 |

**有效数字规则**
- 测量值：最后一位是可疑数字
- 计算结果：与测量值中精度最低的保持一致
- 例：3.2cm + 0.456cm = 3.7cm（保留一位小数）

## 💧 第2次训练：用毛细管法测定表面张力

**模块**：表面张力测定  
**时长**：约2.5分钟

### 测定液体的表面张力系数

表面张力是液体的基本性质，可以用毛细管上升法精确测定。

### 实验原理

**毛细上升公式**
h = 2γcosθ / (ρgr)

其中：
- h：液柱上升高度
- γ：表面张力系数（N/m）
- θ：接触角（对于水和玻璃≈0°）
- ρ：液体密度（kg/m³）
- g：重力加速度（9.8 m/s²）
- r：毛细管半径（m）

**简化公式**（当θ≈0°）
γ = ρgrh / 2

### 实验设计

**实验目的**：测定水的表面张力系数

**实验材料**
- 毛细玻璃管（内径0.5-1.0mm）
- 烧杯
- 蒸馏水
- 温度计
- 游标卡尺
- 读数显微镜

### 实验步骤

**步骤1**：测量毛细管内径
- 用游标卡尺在不同位置测量3次
- 取平均值：d = ______ mm

**步骤2**：安装毛细管（垂直）

**步骤3**：测量上升高度
- 从液面开始测量到凹液面最低点
- 测量3次取平均值：h = ______ mm

**步骤4**：记录水温（影响ρ值）

### 数据记录与计算

| 测量次数 | 内径d(mm) | 上升高度h(mm) | 水温T(°C) |
|---------|----------|--------------|----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 平均值 | d = | h = | T = |

**计算过程**：
ρ水(20°C) = 998 kg/m³
r = d/2 = ______ m
γ = ρgrh/2 = 998 × 9.8 × ______ × ______ / 2
γ = ______ N/m

**标准值**：γ水(20°C) = 72.8 × 10⁻³ N/m

**相对误差**：|γ测量 - γ标准| / γ标准 × 100% = ______%

## 🧪 第3次训练：酸碱反应速率的定量研究

**模块**：反应动力学  
**时长**：约2.5分钟

### 反应速率与反应物浓度的关系

化学反应速率受多种因素影响，其中浓度是最重要的因素之一。

### 反应速率的定义

**平均速率**：v = -Δc/Δt = Δn/V·Δt

- Δc：反应物浓度变化
- Δt：时间间隔
- 单位：mol/(L·s) 或 mol·L⁻¹·s⁻¹

### 实验设计

**实验目的**：测定不同浓度下酸碱反应产生CO₂的速率

**实验原理**：
NaHCO₃ + CH₃COOH → CH₃COONa + H₂O + CO₂↑

**实验方法**：排水集气法测定CO₂体积

### 数据记录

**实验条件**：
- 乙酸体积：50mL（过量）
- 反应温度：室温（25°C）
- 小苏打浓度变化

| 实验组 | 小苏打浓度(mol/L) | 收集10mL CO₂所需时间(s) | 速率(mL/s) |
|-------|------------------|----------------------|-----------|
| A组 | 0.1 | | |
| B组 | 0.2 | | |
| C组 | 0.5 | | |
| D组 | 1.0 | | |

### 速率方程分析

对于本反应，速率方程可近似写为：
v = k[NaHCO₃]

**作图分析**：
- 横坐标：[NaHCO₃]
- 纵坐标：v
- 斜率 = k（反应速率常数）

## ⚖️ 第4次训练：阿基米德原理的实验验证

**模块**：密度与浮力  
**时长**：约2.5分钟

### 浮力大小的定量测定

阿基米德原理：浸在液体中的物体受到向上的浮力，浮力等于物体排开液体的重力。

**公式**：F浮 = ρ液 · g · V排

### 实验设计

**实验目的**：验证阿基米德原理，测定物体的平均密度

**实验材料**
- 弹簧测力计（量程5N，精度0.1N）
- 溢水杯
- 烧杯
- 水
- 金属块
- 细线

### 实验步骤

**步骤1**：用弹簧测力计测量金属块在空气中的重力G
G = ______ N

**步骤2**：将金属块浸入水中（不触底），读数F拉
F拉 = ______ N

**步骤3**：用小烧杯收集溢出的水，测量其质量m
m = ______ g

**步骤4**：计算浮力和排开液体重力，比较

### 数据记录

| 测量项目 | 数值 |
|---------|------|
| 金属块重力G (N) | |
| 金属块在水中拉力F (N) | |
| 浮力F浮 = G - F (N) | |
| 溢出水质量m (g) | |
| 溢出水重力G排 = mg (N) | |
| F浮 / G排 | |

### 原理验证

**计算验证**：
- 若F浮 ≈ G排，则验证阿基米德原理

**物体密度计算**：
ρ物体 = ρ水 × G / (G - F)
= ______ kg/m³

### 小任务

**任务：拓展分析**

| 问题 | 分析 |
|-----|------|
| 如果金属块只浸入一半，F浮如何变化？ | 减半，因为V排减半 |
| 如果把水换成盐水，F浮如何变化？ | 增大，因为ρ液增大 |
| 物体的沉浮条件是什么？ | ρ物 > ρ液则沉，ρ物 < ρ液则浮 |

## 🌈 第5次训练：用最小偏向角法测折射率

**模块**：折射率测定  
**时长**：约2.5分钟

### 精确测定透明介质的折射率

光的折射遵循斯涅尔定律，n₁ sinθ₁ = n₂ sinθ₂。我们可以用最小偏向角法精确测定折射率。

### 实验原理

**最小偏向角法**：
当光线以特定角度入射时，偏向角达到最小值。此时：
n = sin[(δmin + α)/2] / sin(α/2)

其中：
- δmin：最小偏向角
- α：棱镜顶角

### 实验材料

- 三棱镜（玻璃，n≈1.5）
- 光学实验台
- 平行光源（单色光）
- 旋转台
- 量角器或分光计

### 实验步骤

**步骤1**：测量棱镜顶角α
- 将棱镜放在分光计上
- 测量两个折射面的夹角
- α = ______°

**步骤2**：测定最小偏向角
- 让光从一侧入射
- 缓慢旋转棱镜，观察出射光方向
- 找到偏向角最小的位置
- δmin = ______°

**步骤3**：计算折射率

### 数据计算

**代入公式**：
n = sin[(δmin + α)/2] / sin(α/2)
= sin[(__ + __)/2] / sin(__/2)
= sin[__] / sin[__]
= ______

**查表对比**：玻璃折射率标准值约1.50-1.52

**相对误差** = |n测量 - 1.52| / 1.52 × 100% = ______%

### 小任务

**任务：误差来源分析**

| 误差来源 | 影响 | 改进措施 |
|---------|------|---------|
| 顶角测量误差 | 系统误差 | 多次测量取平均 |
| 偏向角读数误差 | ______ | ______ |
| 单色性不好（光源） | ______ | ______ |
| 棱镜不均匀 | ______ | ______ |

## 🌱 第6次训练：探究蒸腾拉力的数学模型

**模块**：植物蒸腾模型  
**时长**：约2.5分钟

### 植物体内水运输的定量分析

植物通过蒸腾作用产生拉力，将水从根部运输到顶部。这个拉力有多大？

### 模型建立

**基本假设**：
1. 导管是均匀的圆柱形管道
2. 水在导管中形成连续的水柱
3. 蒸腾拉力是主要动力

**关键参数**：
- 导管半径：r
- 导管数量：N
- 蒸腾速率：Q（单位时间失水质量）
- 水柱高度：H

### 实验测量

**实验目的**：测定蒸腾拉力

**实验材料**：
- 带叶的枝条
- 装水的玻璃管（带刻度）
- 水槽
- 密封圈

### 数据记录

| 测量项目 | 数值 |
|---------|------|
| 枝条导管总横截面积A | mm² |
| 导管数量N | 根 |
| 平均导管半径r | μm |
| 蒸腾失水速率Q | g/h |
| 水柱上升高度H | cm |

### 模型计算

**蒸腾拉力的理论值**：
- 水流速度 v = Q / (ρ · N · π · r²)
- 所需拉力 F = ρ · g · H

**验证实验**：
将枝条下端连接水银压力计，可以直接测出蒸腾拉力。

**植物导管vs人类血管**：
- 植物导管直径：10-500 μm
- 人体毛细血管直径：5-10 μm
- 但人体有心脏泵血，植物只能靠蒸腾

## 🔌 第7次训练：混联电路的设计与计算

**模块**：复杂电路分析  
**时长**：约2.5分钟

### 串并联混合电路的分析方法

实际电路往往是串并联的组合，需要系统分析。

### 分析方法

**步骤1：识别电路结构**
- 找到等势点（用相同符号标记）
- 简化电路图

**步骤2：确定等效电阻**
- 并联部分先计算
- 再与串联部分合并

**步骤3：计算总电流和电压**
- I总 = U源 / R总
- 用分压、分流公式求各部分

### 例题分析

**电路图**：
```
    R1
U ─┤├──┬──┤R2
         │
        R3
         │
        GND
```

R1 = 10Ω, R2 = 20Ω, R3 = 30Ω, U = 12V

**分析**：
- R2和R3先并联
- R并 = (20×30)/(20+30) = 12Ω
- R总 = R1 + R12 = 10 + 12 = 22Ω
- I总 = 12/22 = 0.55A
- U1 = 10 × 0.55 = 5.5V
- I2 = 12/20 = 0.6A
- I3 = 12/30 = 0.4A
