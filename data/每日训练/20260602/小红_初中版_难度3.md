## 🎯 第2次训练：SMART暑假目标

**模块**：目标设定  
**时长**：约2分钟

### SMART目标法则

设定有效目标需要遵循SMART原则：

| 字母 | 含义 | 说明 | 示例 |
|-----|------|------|------|
| S | Specific | 具体 | 不是"学英语"而是"背完中考词汇1600" |
| M | Measurable | 可衡量 | 有明确的数字或里程碑 |
| A | Achievable | 可达成 | 踮脚能够到的目标 |
| R | Relevant | 相关 | 与你的成长方向一致 |
| T | Time-bound | 有期限 | 8月31日前完成 |

## ⭐ 第3次训练：重要vs紧急

**模块**：优先级排序  
**时长**：约2分钟

### 重要-紧急矩阵

将暑假要做的事分类到四个象限：

```
              重要
                │
                │
    ┌───────────┼───────────┐
    │   Ⅱ象限   │   Ⅰ象限   │
    │  规划成长  │  紧急执行  │
    │ （应该做） │ （必须做） │
紧急├───────────┼───────────┤
    │   Ⅳ象限   │   Ⅲ象限   │
    │  浪费时间  │  享受放松  │
    │ （少做）   │ （适度）   │
    └───────────┴───────────┘
                │
                │不重要
```

### 学习时间分配建议
