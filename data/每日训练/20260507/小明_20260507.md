## 模块一：注意力训练（2次）

### Session 1：审题专注力训练 ⏱️2分钟

**任务名称**：关键词小侦探

**训练说明**：
做数学题时，第一步不是计算，而是找到题目的"眼睛"——关键词。今天训练你快速识别关键词的能力。

**训练内容**：
```
题目：用加法和减法解决问题
"小明有8颗糖果，给了小红3颗，小明还剩几颗糖果？"

找出这句话中的关键词：
□ 小明有8颗糖果    □ 给小红3颗    □ 还剩几颗
□ 小红有糖果

正确关键词：小明有8颗糖果、给了小红3颗、还剩几颗
```

**判断标准**：
- 找到全部3个关键词 = ★★★
- 找到2个关键词 = ★★
- 找到1个关键词 = ★

**小明挑战**：
```
题目：小华有5支铅笔，用掉了2支，又买了3支，小华现在有几支铅笔？

这道题的关键词是：
① 小华有5支铅笔   ② 用掉了2支   ③ 买了3支   ④ 小华有几支

你选：_______
```

**训练说明**：

**训练内容**：

```
你的解题步骤：
步骤1：先算加法        → 6 + 3 = ___

答案：____
```

**小明练习**：
```
计算：10 - 3 + 4 = ？

步骤1：先算什么？（减法还是加法？）→ 先算 10 - 3 = ___
步骤2：再算什么？→ ___ + 4 = ___

你的最终答案：_______
```

## 模块二：记忆力训练（2次）

### Session 3：审题要点记忆 ⏱️2分钟

**任务名称**：三秒小记忆

**训练说明**：
考试时没时间反复看题，要在最短时间内记住关键信息。今天训练你的快速记忆能力。

**训练内容**：
```
现在有10秒钟记忆这道题：

"小红有10个苹果，给了小明4个，
又从小刚那里得到3个，
小红现在有多少个苹果？"

闭眼回忆，在心里说出：
1. 小红原来有几个苹果：____个
2. 给小明几个：____个
3. 又得到几个：____个
4. 要解决的问题：________________

记忆完成后再睁开眼睛！
```

**记忆小技巧**：
- 把数字和生活联系起来（比如10个苹果 = 10根手指）
- 用"原-给+得"记住变化过程

### Session 4：解题思路记忆 ⏱️2分钟

**任务名称**：步骤小口诀

**训练说明**：
每做完一道题，要把解题步骤记住，下次遇到类似的题就能快速做出来。

**训练内容**：
```
【简单应用题解题步骤】

第一步：看清楚（题目问什么？）
第二步：找条件（有哪些数字？）
第三步：想一想（用什么方法？加法还是减法？）
第四步：算出来（写出答案）
第五步：检查一下（算对了吗？）

小口诀："看、找、想、算、查，五步完成顶呱呱！"
```

**小明挑战**：
```
题型：简单增减问题

题目："图书角有12本书，借走了5本，又还回来3本，现在有几本书？"

解题步骤：
第一步：看清楚 → 问现在有几本书
第二步：找条件 → 有12本、借走5本、还回3本
第三步：想一想 → 先减后加
第四步：算出来 → 12 - 5 = ___，___ + 3 = ___
第五步：检查 → 答案对吗？

你的答案：_______
```

## 模块三：学习方法应用（4次）

### Session 5：真题思维四步法 ⏱️2分钟

**任务名称**：审题→破题→解题→检查

**训练说明**：
建立完整的解题思维流程，每道题都按这个流程走。

**训练内容**：
```
【真题思维四步法】

第一步 审题（看清楚）
- 题目问什么？_____________
- 有哪些数字？_____________

第二步 破题（想明白）
- 这道题考什么？（加法/减法/乘法/除法）
- 需要几步计算？

第三步 解题（算准确）
- 写出解题步骤
- 每一步都要写清楚

第四步 检查（验一验）
- 答案合理吗？
- 再算一遍验证
```

**小明练习**：
```
用四步法解这道题：

"小刚有15颗玻璃球，丢了3颗，又捡到5颗，现在有几颗？"

第一步审题：问____________，有____________

第二步破题：用的是____________法

第三步解题：
第一步：15 - 3 = ___
第二步：___ + 5 = ___

第四步检查：____________

答案：______颗
```

### Session 7：难题攻略 ⏱️2分钟

**任务名称**：难题不怕怕

**训练说明**：
遇到不会的题不要慌，按这个顺序尝试。

**训练内容**：
```
【难题攻略】

遇到不会的题，按这个顺序尝试：

第1步 跳过（1分钟）
先做会做的题，难题留到最后
不要在一道题上卡太久

第2步 回头看（2分钟）
- 再读一遍题目
- 圈出关键词
- 想想老师讲过类似的题吗

第3步 画一画（2分钟）
- 用手指指着数字
- 在草稿纸上画一画
- 把大问题变成小问题

第4步 求助（标记）
- 问老师或同学
- 查看参考答案
- 弄懂后一定要自己再算一遍
```

**小明自测**：
```
一道数学题看了3分钟还是不会做，你怎么做？
□ 继续死磕，直到想出来
□ 先跳过，最后再回来做
□ 直接看答案
□ 问同学或老师

正确做法：先跳过，最后再回来

如果最后还是不会，应该：
□ 算了，不做了
□ 看答案，然后自己再做一遍
□ 问老师，但不用搞懂

正确做法：看答案弄懂后，自己再算一遍
```

### Session 8：易错题防范 ⏱️2分钟

**任务名称**：避坑小能手

**训练说明**：
很多同学不是不会做，而是掉进了"陷阱"。今天训练你识别陷阱的能力。

**训练内容**：
```
【常见易错陷阱】

陷阱1：看错符号
例：6 + 3 = ___
易错：看成 6 × 3 = 18
正确：6 + 3 = 9

陷阱2：算错顺序
例：10 - 3 + 2 = ___
易错：10 - (3+2) = 5
正确：(10-3) + 2 = 9

陷阱3：漏掉数字
易错：做一半忘一半
正确：读一题做一题，圈出关键词

陷阱4：粗心算错
易错：3+4=6（错）
正确：3+4=7
```
