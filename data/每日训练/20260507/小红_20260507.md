## 模块一：注意力训练（2次）

### Session 1：审题专注力训练 ⏱️2分钟

**任务名称**：关键词猎人（进阶版）

**训练说明**：
做数学题时，第一步不是计算，而是找到题目的"眼睛"——关键词。今天训练你快速识别关键词并挖掘隐含条件的能力。

**训练内容**：
```
题目：综合应用题
"某商场销售一种电器，每台进价为2000元。
如果按标价的8折出售，每台可获利200元。
为扩大销售，商场决定调整价格：
每降价50元，销量增加10台。
已知降价前每月可卖出50台，
问：每月获得最大利润时，每台应定价多少元？"

找出这句话中的关键词：
□ 进价2000元    □ 标价未知    □ 8折出售    □ 获利200元
□ 降价50元销量增10台    □ 降价前每月50台    □ 求最大利润时定价

隐含条件挖掘：
1. 标价 = (进价 + 利润) ÷ 0.8 = (2000+200) ÷ 0.8 = ____元
2. 销量与价格的关系式：y = ____ - ____x
```

**判断标准**：
- 找到全部7个关键词 + 2个隐含条件 = ★★★
- 找到5-6个关键词 + 1个隐含条件 = ★★
- 找到4个以下关键词 = ★

**小红挑战**：
```
题目：一艘船在河中航行，已知船在静水中的速度为15km/h，
水流速度为5km/h。船从A码头顺水航行到B码头后立即返回。
已知A、B两码头之间的距离为40km，
问：船往返一次的平均速度为多少km/h？

这道题的关键词和隐含条件：
① 静水速度   ② 水流速度   ③ 距离40km   ④ 往返
⑤ 顺水速度 = 静水速度 + 水流速度
⑥ 逆水速度 = 静水速度 - 水流速度
⑦ 平均速度 = 总路程 ÷ 总时间

你选：_______
```

**任务名称**：结构化记忆法

**训练说明**：
综合性题目信息量大，需要结构化记忆。今天训练你快速提取和整理题目关键信息的能力。

**训练内容**：
```

"某公司计划投资一个新项目，有两种方案可选：
方案A：一次性投资100万元，年收益率10%，复利计算
方案B：分期投资，第一年投入60万元，以后每年投入20万元，
       年收益率12%，复利计算
问：5年后哪种方案的收益更高？"

闭眼回忆，整理出结构化信息：

方案A：
- 初始投资：____万元

- 第一年投资：____万元

**记忆小技巧**：

### Session 4：解题思路记忆 ⏱️2分钟

**任务名称**：模型归档

**训练说明**：
每做完一道题，要把解题思路"归档"保存。今天学习建立综合题型的解题模型。

**训练内容**：
```
【综合应用题解题模型】

模型类型：投资决策问题

标准解题流程（请按顺序记忆）：
① 分析题意（两种方案的差异点是什么？）
② 建立模型（方案A是等比数列，方案B是等比+等差混合）
③ 分别计算（先算A，再算B）
④ 比较分析（哪个更大？）
⑤ 得出结论（选择哪种方案）

关键公式回顾：
- 复利公式：F = P(1+r)^n
- 等比数列求和：S = a₁(1-q^n)/(1-q)

易错提醒：
- 注意"复利"的含义（利滚利）
- 注意投资时间的一致性
- 考虑资金的时间价值
```

**小红挑战**：
```
题型：投资决策

题目：两种存款方式
方式一：一次性存10000元，年利率4%，存5年
方式二：每年初存2000元，年利率4%，存5年
问：哪种方式5年后本息和更多？多多少？

这道题的解题思路：
方式一：直接用复利公式 F = 10000 × (1+0.04)^5 = ____元

方式二：用等比数列求和公式
首项a₁ = 2000，公比q = 1.04，项数n = 5
F = 2000 × (1.04^5 - 1) / (1.04 - 1) × 1.04 = ____元

（乘1.04是因为每年初存款）

你的答案：_______
```

## 模块三：学习方法应用（4次）

### Session 5：真题思维四步法（综合版）⏱️2分钟

**任务名称**：审题→破题→建模→验证

**训练说明**：
建立完整的解题思维流程，每道题都按这个流程走。

**训练内容**：
```
【综合题真题思维四步法】

第一步 审题（找什么？）
- 题目问什么？（最终目标）
- 有哪些已知条件？（数据信息）
- 有没有隐含条件？（需要推导）
- 两种方案的差异点在哪？

第二步 破题（用什么？）
- 这道题考什么核心知识点？
- 需要用到什么数学模型？
- 和之前做过的哪类题类似？
- 能否识别出题目的"原型"？

第三步 建模（怎么列？）
- 建立数学模型（方程/函数/不等式）
- 明确变量和常量
- 列出计算公式

第四步 求解验证（怎么算？）
- 按步骤计算
- 检查计算过程
- 验证结果合理性
```

**小红练习**：
```
用四步法解这道题：

"某工厂生产两种产品A和B，需要使用甲、乙两种原料。

生产1件B需要甲原料1kg、乙原料2kg，可获利40元
已知工厂每天有甲原料120kg、乙原料100kg的供应限制
问：每天生产A、B各多少件，能获得最大利润？"

第一步审题：
- 问：____________
- 已知：____________

第二步破题：
- 这是____________问题
- 用____________方法求解

第三步建模：
- 设生产A产品x件，B产品y件
- 约束条件：____________
- 目标函数：____________

第四步求解验证：
- 画出可行域
- 求出最优解
- 答案：x = ____，y = ____，最大利润 = ____元
```

### Session 6：压轴题攻略 ⏱️2分钟

**任务名称**：压轴题拆解

**训练说明**：
压轴题通常综合性强、难度大。学会把大题拆成小题，一步步攻克。

**训练内容**：
```
【压轴题拆解策略】

遇到压轴题，按这个顺序拆解：

第1步 读三遍（2分钟）
- 第一遍：了解题目大概讲什么
- 第二遍：标记关键数据和条件
- 第三遍：明确最终要解决什么问题

第2步 分小题（3分钟）
- 压轴题通常有3问，由易到难
- 先做第(1)问（通常是送分题）
- 第(1)问的结论往往是第(2)(3)问的台阶

第3步 找联系（3分钟）
- 第(2)(3)问和第(1)问有什么关系？
- 条件能重复利用吗？
- 有没有通用方法可以解所有小问？

第4步 写过程（5分钟）
- 第(1)问：详细写出过程
- 第(2)问：借鉴第(1)问的方法
- 第(3)问：总结归纳，一般化
```

**小红自测**：
```

### Session 8：易错题防范（高阶）⏱️2分钟

**任务名称**：高阶避坑训练

**训练说明**：
进阶题目有更多隐藏陷阱，需要更仔细的识别能力。

**训练内容**：
```
【高阶易错陷阱】

陷阱1：忽略定义域
例：求log₂(x² - 4)的定义域
易错：x² - 4 > 0 → x > 2或x < -2
正确：定义域是x > 2或x < -2（两部分都要写）

陷阱2：端点取值遗漏
例：函数f(x)在[0,1]上单调递增
易错：只考虑开区间
正确：端点0和1也要考虑

陷阱3：分类讨论不完整
例：解不等式|x - 1| > a
易错：只讨论a > 0的情况
正确：需要讨论a > 0、a = 0、a < 0三种情况

陷阱4：充分必要条件混淆
例：p是q的充分条件
易错：p → q写反
正确：p → q（p能推出q）

陷阱5：运算律使用错误
例：log₂(a + b) = log₂a + log₂b
易错：以为可以拆开
正确：只有log₂(ab) = log₂a + log₂b
```
