## 模块一：注意力训练（2次）

### Session 1：审题专注力训练 ⏱️2分钟

**任务名称**：关键词猎人

**训练说明**：
做数学题时，第一步不是计算，而是找到题目的"眼睛"——关键词。今天训练你快速识别关键词的能力。

**训练内容**：
```
题目：用方程解应用题
"某商店销售一种商品，进价为30元/件，售价为50元/件，
每天可卖出100件。如果每件涨价2元，每天销量减少20件。
问：每天定价多少元时，利润最大？"

找出这句话中的关键词：
□ 进价为30元    □ 售价为50元    □ 每天100件
□ 涨价2元    □ 销量减少20件    □ 求利润最大

正确关键词：进价30元、售价50元、每天100件、涨价2元销量减20件
```

**判断标准**：
- 找到全部6个关键词 = ★★★
- 找到4-5个关键词 = ★★
- 找到2-3个关键词 = ★

**小刚挑战**：
```
题目：一辆汽车从A地出发，速度为60千米/小时，
行驶3小时后距离B地还有45千米。
求A地到B地的总距离。

这道题的关键词是：
① 速度60千米/小时   ② 行驶3小时   ③ 距离B地45千米
④ 求总距离   ⑤ 汽车从A地出发

你选：_______
```

### Session 2：解题过程专注训练 ⏱️2分钟

**任务名称**：步骤追踪

**训练说明**：
解题就像走迷宫，每一步都要清楚。今天训练你在解题过程中保持专注，不跳步。

**训练内容**：
```
例题：解方程 3x - 7 = 20

你的解题步骤：
步骤1：3x - 7 + 7 = 20 + 7    ← 先做什么？
步骤2：3x = 27                 ← 得到什么？
步骤3：x = 27 ÷ 3             ← 最后做什么？
步骤4：x = 9                  ← 答案是什么？

问题：上面步骤1中，"先做什么"应该填什么？
□ 加7      □ 减7      □ 乘3      □ 除3
```

**小刚练习**：
```
计算：24 + 36 ÷ 4 - 3 × 2

步骤1：先算除法        → 36 ÷ 4 = ___
步骤2：再算乘法        → 3 × 2 = ___
步骤3：然后算加法      → 24 + ___ = ___
步骤4：最后算减法      → ___ - ___ = ___

你的最终答案：_______
```

## 模块二：记忆力训练（2次）

### Session 3：审题要点记忆 ⏱️2分钟

**任务名称**：五秒记忆法

**训练说明**：
考试时没时间反复看题，要在最短时间内记住关键信息。今天训练你的"五秒记忆"能力。

**训练内容**：
```
现在有15秒钟记忆这道题：

"某工厂要生产一批零件，原计划每天生产100个，
5天完成。现在要求提前1天完成，
每天需要多生产多少个零件？"

闭眼回忆，在心里说出：
1. 原计划每天生产：____个
2. 原计划完成时间：____天
3. 实际完成时间：____天
4. 需要解决的问题：________________

记忆完成后再睁开眼睛！
```

**记忆小技巧**：
- 把数字和公式联系起来（效率×时间=工作总量）
- 用"原计划→现在→差多少"三个关键词记住变化

### Session 4：解题思路记忆 ⏱️2分钟

**任务名称**：思路归档

**训练说明**：
每做完一道题，要把解题思路"归档"保存，下次遇到类似的题就能快速调用。

**训练内容**：
```
例题类型：一元一次方程应用题（工程问题）

标准解题流程（请按顺序记忆）：
① 找等量关系（工作总量 = 效率 × 时间）
② 设未知数（通常问什么设什么）
③ 列方程（根据等量关系列式）
④ 解方程（按步骤计算）
⑤ 检验答案（代入原题验证）
⑥ 写答句（完整回答问题）

工程问题特征：
- "合作完成" → 效率和 = 各效率之和
- "提前/推迟" → 时间变化引起效率变化
```

**小刚挑战**：
```
题型：工程问题

题目："一项工程，甲单独做需要10天完成，乙单独做需要15天完成。
如果两人合作，需要多少天完成？"

这道的等量关系是：
□ 工作总量 = 甲效率 × 10    ← 对吗？
□ 1/10 + 1/15 = 1/x
□ 需要多少天 = 10 + 15

你的答案：_______
```

## 模块三：学习方法应用（4次）

### Session 5：真题思维四步法 ⏱️2分钟

**任务名称**：审题→破题→解题→验证

**训练说明**：
建立完整的解题思维流程，每道题都按这个流程走。

**训练内容**：
```
【真题思维四步法】

第一步 审题（找什么？）
- 题目问什么？_____________
- 有哪些已知条件？_____________
- 有没有隐藏条件？_____________

第二步 破题（用什么？）
- 这道题考什么知识点？（方程/几何/函数...）
- 需要用到什么公式或方法？
- 和之前做过的哪道题类似？

第三步 解题（怎么算？）
- 写出解题步骤
- 每一步都要写清楚

第四步 验证（对不对？）
- 答案合理吗？
- 能代入原题检验吗？
```

**小刚练习**：
```
用四步法解这道题：

"某商品进价为80元/件，定价为120元/件，
可卖出100件。如果每件降价10元，可多卖出30件。
问：每天定价多少元时，利润最大？"

第一步审题：问____________，已知____________

第二步破题：考的是____________问题

第三步解题：
设：每天利润为y元，定价为x元
列式：____________
解：____________

第四步验证：____________

答案：定价______元时利润最大
```

### Session 6：数学物理综合思维 ⏱️2分钟

**任务名称**：数理联动

**训练说明**：
数学是物理的语言，物理是数学的应用。学会用数学解决物理问题。

**训练内容**：
```
【数学物理综合训练】

物理问题：一辆汽车以10 m/s²的加速度从静止开始加速，
求4秒后汽车的速度和位移。

数学解决步骤：
1. 速度公式：v = at
   v = 10 × 4 = 40 m/s

2. 位移公式：s = ½at²
   s = ½ × 10 × 16 = 80 m

关键点：物理列式 → 数学计算 → 物理解释
```

**小刚挑战**：
```
物理问题：一个物体从高度20米处自由落下，
忽略空气阻力，求落地时的速度。

数学部分：
1. 物理公式：v² = 2gh
   （速度的平方 = 2 × 重力加速度 × 高度）

2. 代入数据：v² = 2 × 10 × 20

3. 数学计算：v² = ___

4. 求速度：v = √___ = ____ m/s
```

### Session 7：难题攻略 ⏱️2分钟

**任务名称**：难题四步走

**训练说明**：
遇到不会的题不要慌，按这个顺序尝试。

**训练内容**：
```
【难题攻略】

遇到不会的题，按这个顺序尝试：

第1步 跳过（2分钟）
先做会做的题，难题留到最后
不要在一道题上卡太久

第2步 回头看（3分钟）
- 再读一遍题目
- 标记关键词
- 回忆相关知识点
- 想想老师上课讲过类似的吗

第3步 联想（3分钟）
- 这道题像之前做过的哪道题？
- 能否把大问题拆成小问题？
- 有没有类似的公式可以用？
- 已知和未知之间有什么联系？

第4步 求助（标记）
- 问老师或同学
- 查看参考答案
- 弄懂后一定要自己再算一遍
- 总结这类题的解法
```

**小刚自测**：
```
一道数学题看了5分钟还是不会做，你怎么做？
□ 继续死磕，直到想出来
□ 先跳过，最后再回来做
□ 直接看答案
□ 问同学或老师

正确做法：先跳过，最后再回来

如果最后还是不会，应该：
□ 算了，不做了
□ 看答案，然后自己再做一遍
□ 问老师，但不用搞懂

正确做法：看答案弄懂后，自己再算一遍，并总结方法
```

### Session 8：易错题防范 ⏱️2分钟

**任务名称**：避坑训练

**训练说明**：
很多同学不是不会做，而是掉进了"陷阱"。今天训练你识别陷阱的能力。

**训练内容**：
```
【常见易错陷阱】

陷阱1：单位不同
例：1.5小时 = ____分钟
易错：直接写1.5
正确：1.5 × 60 = 90分钟

陷阱2：运算顺序
例：3 + 5 × 2² = ____
易错：(3+5) × 2² = 32
正确：3 + 5 × 4 = 3 + 20 = 23

陷阱3：丢括号
例：解方程 2(x+3) = 16
易错：2x+3=16
正确：2x+6=16

陷阱4：移项不变号
例：3x + 5 = 20
易错：3x = 20 + 5
正确：3x = 20 - 5
```

**小刚挑战**：
```

   陷阱：____________
   答案：____________

3. 解方程 5(x-2) = 25
   第一步：____________
   陷阱：____________
   正确答案：____________

4. 解方程 4x - 7 = 21
   x = ?
   陷阱：____________
   答案：____________
```
