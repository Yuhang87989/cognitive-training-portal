## Session 1：注意力训练 - 复习目标识别

### 训练内容

在开始复习之前，先问自己三个问题：

| 问题 | 作用 | 自我回答 |
|------|------|----------|
| ① 我这次期末要达到什么分数？ | 明确目标 | __________分 |

| 学科 | 目标分数 | 现状估计 | 差距 | 需要重点突破的知识点 |
|------|----------|----------|------|----------------------|
| 语文 | | | | |
| 数学 | | | | |
| 英语 | | | | |
| 物理 | | | | |

### 游戏化元素
- **成就徽章**：获得"目标清晰"徽章（完成目标分析表）
- **今日差距总分**：算出所有学科的总差距

## Session 2：记忆力训练 - 目标拆解记忆

### 训练目标
掌握"大目标拆成小任务"的记忆方法，让复习不再迷茫

### 训练内容
**目标拆解三步法**

把"期末考好"这个大目标，拆成可以执行的小任务：

```
大目标：期末考好
    ↓
拆解：各科达到目标分数
    ↓
行动：每天复习什么内容
    ↓
执行：每次专注25分钟
```

### 记忆技巧
使用**数字口诀法**记忆拆解步骤：
> **"1大目标 → 2各科拆解 → 3每日行动 → 4专注执行"**
> 
> 谐音："一大二拆三行动四执行"

### 训练任务
**拆解你的数学复习目标**：

如果数学要提升____分，你需要：
1. 第一步：复习哪些知识点？（________、________、________）
2. 第二步：每天分配多少时间？（________分钟）
3. 第三步：用什么方式检验效果？（________）

### 记忆挑战
在90秒内记忆以下复习拆解关键词：
| 步骤 | 关键词 |
|------|--------|
| 1 | 目标 |
| 2 | 拆解 |
| 3 | 计划 |
| 4 | 执行 |
| 5 | 检验 |

## Session 3：学霸方法 - 复习优先级矩阵

### 训练目标
学会用"重要紧急矩阵"确定复习优先级，把时间用在刀刃上

### 训练内容
**复习优先级四象限**

把你的复习任务放进四个象限：

| 象限 | 特点 | 复习策略 | 你的复习任务示例 |
|------|------|----------|------------------|
| 🟥 第一象限<br/>重要且紧急 | 考试必考+掌握不好 | **优先复习，重点突破** | |
| 🟧 第二象限<br/>重要不紧急 | 考试必考+已掌握 | **定期巩固，保持状态** | |
| 🟨 第三象限<br/>紧急不重要 | 可能考+不太重要 | **快速浏览，留有余地** | |
| 🟩 第四象限<br/>不紧急不重要 | 考试不考/已精通 | **战略性放弃** | |

### 训练任务
**把你的期末复习任务放进象限**：

第一象限（重要且紧急，先做）：
1. _______________
2. _______________

第二象限（重要不紧急，保持）：
1. _______________
2. _______________

第三象限（紧急不重要，快速过）：
1. _______________

第四象限（不紧急不重要，少做）：
1. _______________

### 学霸秘诀
> **"第一象限Tasks不要超过3个，否则会焦虑；第二象限Tasks每天都要有，保持状态最重要！"**

## Session 6：学霸方法 - 听课习惯复习联动

### 训练目标
把"听课习惯"技能迁移到复习中，找到复习重点

### 训练内容
**复习中的"费曼技巧"**

用"讲给别人听"的方式检验复习效果：

| 步骤 | 操作 | 问自己 |
|------|------|--------|
| 1 | 选择一个知识点 | 我能完整讲出来吗？ |
| 2 | 尝试讲解（默讲或出声讲） | 哪里卡住了？ |
| 3 | 卡住的地方重新学习 | 这次理解了吗？ |
| 4 | 再次讲解 | 能讲清楚了吗？ |

### 复习联动技巧
回想课堂上的"这节课老师强调了什么"，这些就是复习重点：
- 老师反复讲的知识点 → **考试必考**
- 老师举例说明的地方 → **理解要深入**
- 老师让做练习的地方 → **需要多练习**

### 训练任务
**用费曼技巧复习一个知识点**：

我选择复习的知识点：________________

| 步骤 | 我的表现 |
|------|----------|
| 第一次讲解 | 流畅□ / 卡住□ → 卡在：____________ |
| 重新学习后 | 理解□ / 不理解□ |
| 第二次讲解 | 流畅□ / 卡住□ |

## Session 7：注意力训练 - 期末复习专注挑战

### 训练目标
模拟真实的期末复习场景，训练长时间专注能力

### 训练内容
**期末复习专注挑战**

现在，请完成以下专注挑战：

**挑战一：5分钟专注阅读**
- 选择一段需要复习的内容
- 全神贯注阅读5分钟
- 不能走神、不能看手机

**挑战二：3分钟记忆挑战**
- 看一段需要记忆的内容（如公式、单词）
- 闭眼回忆3分钟
- 写下你记住的内容
