## 🧠 第2次训练：物理公式记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
记忆基础物理公式，理解每个符号的含义。

### 今日挑战

**基础公式网络**
```
┌─────────────────────────────────────────────────┐
│                                                 │
│  【速度公式】                                    │
│     v = s ÷ t                                   │
│     速度 = 路程 ÷ 时间                          │
│                                                 │
│  【密度公式】                                    │
│     ρ = m ÷ V                                   │
│     密度 = 质量 ÷ 体积                           │
│                                                 │
│  【力的计算】                                    │
│     F = m × g                                   │
│     力 = 质量 × 重力加速度（g=10N/kg）           │
│                                                 │
│  【功的计算】                                    │
│     W = F × s                                   │
│     功 = 力 × 距离                               │
│                                                 │
└─────────────────────────────────────────────────┘
```

**记忆任务**（记忆60秒后完成）：
1. 速度公式中，v、s、t分别代表什么？
2. 如果知道力和距离，怎么求功？
3. 质量为5kg的物体，受到的重力是多少？

### 记忆技巧
💡 **单位记忆法**：记住单位就能记住公式
- 速度的单位是 m/s → 对应 s/t
- 功的单位是 J → N·m → F·s

## 🔍 第4次训练：数学解物理问题

**模块**：学习方法 - 学霸专题  
**时长**：约2分钟

### 🎯 今日技能：用数学方法解决物理问题

### 问题分析

**问题：小明跑步**
```
🏃‍♂️ 小明从家跑到学校：
   - 路程：600米
   - 时间：2分钟（120秒）

   问：小明跑步的速度是多少？
```

**分析过程**：

**Step 1 - 理解物理情境**
- 已知量：s = 600米，t = 120秒
- 待求量：速度 v

**Step 2 - 选择公式**
- 速度公式：v = s ÷ t

**Step 3 - 数学计算**
```
v = 600 ÷ 120
v = 5 （单位：米/秒）
```

**Step 4 - 答案**
- 小明跑步的速度是 5 m/s

### 练习任务

**问题1：小汽车行驶**
