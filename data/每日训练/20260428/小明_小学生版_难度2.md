## 🧠 元认知三步法（今日重点）

### ① 快速扫描（5秒搞定）
学习中快速"照镜子"，看看自己：
- 眼睛在看吗？
- 脑子在想什么？
- 心静不静？

### ② 立刻判断
状态好不好？
- 😊 好：继续加油
- 😐 一般：稍微调整
- 😫 不好：需要换方法

### ③ 马上调整
选一个方法试试：
- 深呼吸3次
- 换个角度想
- 先休息一下
- 设定小目标

## 🎮 Session 1 - 思维健身房

### 🏃 训练任务：注意力扫描挑战

**游戏规则**：
1. 看一段有趣的故事（约150字），限时1分钟
2. 读的时候注意：你的小眼睛在看哪里？脑子里在想什么？
3. 时间到，马上回答：

**扫描小测试**：
| 问题 | 回答 |
|------|------|
| 刚才我在认真看吗？ | |
| 中间有没有走神？ | |
| 走神时在想什么？ | |
| 走神几次？ | |

**思维健身评分**（1-5级）：
- 👶 1级：不知道自己在走神
- 🧒 2级：感觉走神了但说不清
- 👦 3级：能说出走神几次
- 🧑 4级：能说出走神时想什么
- 🦸 5级：走神后能立刻拉回来

**教练点评**：
能"看见"自己的大脑在干什么，这是思维健身的第一步！像照镜子一样，先看清楚自己。

## 🎮 Session 2 - 状态仪表盘

### 📊 训练任务：学习状态小仪表

**仪表盘介绍**：
你的大脑有三个小仪表：

```
[🔴注意力]  [🟡理解力]  [🟢动力]
```

**训练挑战**：
阅读一段有趣的科普短文（150字），边读边用"瞄一眼"检查你的仪表盘：

**状态记录卡**：
| 检查时间 | 注意力 | 理解力 | 动力 | 需要调整吗？ |
|---------|--------|--------|------|-------------|
| 开始时 | | | | |
| 30秒后 | | | |
| 1分钟后 | | | |
| 结束时 | | | |

**小发现**：
- 哪个仪表最先"掉电"？
- 为什么会这样？

**教练点评**：
发现自己的"弱点信号"很重要！下次看到这个信号，就能提前调整。

## 🎮 Session 3 - 3秒重启术

### ⚡ 训练任务：快速调节练习

**情境故事**：
小明正在做数学作业，做着做着觉得很无聊，脑子开始想别的事情了...

**挑战任务**：
立刻启动"3秒重启"：

| 秒数 | 做什么 | 默念 |
|------|--------|------|
| 第1秒 | 深呼吸一下 | "呼~" |
| 第2秒 | 想想做完了能干嘛 | "做完了可以..." |
| 第3秒 | 给自己定个小目标 | "先做完这1题！" |

**实战练习**：
用上面的方法，重新集中注意力完成2道数学题。

**自我检查**：
- 小目标完成了吗？
- 动力恢复了吗？
- 这个方法有用吗？

**调节小锦囊**（选一个试试）：
| 情况 | 好方法 |
|------|--------|
| 太难了 | 先做最简单的 |
| 太无聊 | 想象做完能玩 |
| 太累了 | 休息5分钟 |
| 不明白 | 换个例子想想 |

## 🎮 Session 4 - 思维追踪练习

### 🎯 训练任务：数学题思维记录

**题目**：
"水果店有苹果30个，橘子比苹果少10个，梨比橘子多15个。梨有多少个？"

**挑战任务**：
1. 读题时想一想：这道题难不难？
2. 做的时候注意：我的思路清晰吗？
3. 遇到困难了吗？用什么方法解决？
4. 做完了检查：我做对了吗？

**思维记录表**：
```
读题：这道题看起来____（简单/中等/难）

我的想法：____

遇到困难了吗？____

我怎么解决的：____

做完了，感觉：____
```

**教练点评**：
做完题后想一想"我刚才怎么做的"，这比知道答案更重要！

---

## 🎮 Session 5 - 综合调节挑战

### 🔄 训练任务：学习困难应对

**情境故事**：
语文课上要读一篇很难的古诗，感觉读不懂，越来越烦躁...

**挑战任务**：
这是"看不明白+心情烦躁"的双重困难！

**Step 1：先判断哪个更紧急**
- 是读不懂更难受？
- 还是烦躁更烦？

**Step 2：组合调节**（选2个方法）
- 🔄 换个方式读（小声/大声/指着读）
- 💺 站起来活动一下
- 🎯 只背一句
- 💪 给自己打气："我能行！"

**Step 3：检查效果**
- 好点了吗？
- 还需要什么帮助？

## 🎮 Session 7 - 错题想想看

### 📝 训练任务：当时我在想什么？

**重要观点**：
知道"正确答案"不是最重要的，知道"我为什么会做错"才重要！

**挑战任务**：
回忆一道最近做错的题：

1. 那是做什么作业时错的？
2. 当时第一眼看到题，脑子在想什么？
3. 后来发现哪里想错了？
4. 下次遇到类似的，我要提醒自己什么？

**错题思维卡**：
