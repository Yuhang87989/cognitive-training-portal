### 【Session 1】守恒定律入门 · 概念辨析

**训练名称**：守恒概念建立

**训练内容**：
**守恒定律核心要素**

守恒定律成立的基本条件：**孤立系统**（与外界无物质、能量交换）

| 守恒类型 | 守恒量 | 成立条件 |
|---------|-------|---------|
| 质量守恒 | 总质量 | 孤立系统 |
| 能量守恒 | 总能量 | 孤立系统 |
| 电荷守恒 | 总电荷 | 孤立系统 |
| 动量守恒 | 总动量 | 合外力为零 |

**思考题**：
1. "将一杯水倒入另一个杯子，水的质量变了吗？" - 适用哪种守恒？
2. "运动员起跳时，地面受到的力消失了？" - 对吗？为什么？
3. "化学反应后溶液变少了，说明质量不守恒" - 对吗？为什么？

**答案分析**：
1. 质量守恒（总质量不变，只是位置变化）
2. 不对（力的作用是相互的，大小相等方向相反）
3. 不对（如果考虑气体，质量仍守恒）

**✅ 完成标准**：理解守恒定律的核心要素，正确辨析守恒条件

### 【Session 2】质量守恒 · 化学反应分析

**训练名称**：质量守恒定律应用

**训练内容**：
**化学反应中的质量守恒**

实验验证：2H₂ + O₂ → 2H₂O

| 反应物 | 反应前质量 | 生成物 | 反应后质量 |
|-------|-----------|-------|-----------|
| H₂ | 4g | H₂O | ? |
| O₂ | 32g | | |

**计算分析**：
- 根据质量守恒：反应物总质量 = 生成物总质量
- 4g + 32g = 36g
- 生成物H₂O总质量 = 36g

**进阶练习**：
"12g碳与32g氧气恰好完全反应，生成CO₂多少克？若氧气过量4g，生成CO₂多少克？"

**解答步骤**：
```
Step 1：计算反应物总量
  碳：12g
  氧气：32g - 4g = 28g
  总质量：40g

Step 2：应用质量守恒
  生成物总质量 = 反应物总质量 = 40g

Step 3：验证
  CO₂中碳来自碳，氧来自氧气
  碳：12g，氧：28g ✓
```

**✅ 完成标准**：正确应用质量守恒定律解决化学问题

### 【Session 3】能量守恒 · 物理转换分析

**训练名称**：能量守恒定律应用

**训练内容**：
**能量转换与守恒**

**情景分析**：小球从高度h处自由下落

| 阶段 | 高度 | 速度 | 动能 | 势能 | 机械能 |
|------|------|------|------|------|-------|
| 初始 | h | 0 | 0 | mgh | mgh |
| 中间某点 | h/2 | v | ? | ? | ? |
| 落地前 | 0 | v_max | ? | 0 | ? |

**计算分析**：
- 高度h/2处：
  - 势能：Ep = mg(h/2) = mgh/2
  - 动能：Ek = mgh - mgh/2 = mgh/2
  - 机械能：E = Ek + Ep = mgh（守恒）

**能量守恒表达式**：
```
E_total = Ek + Ep = 常数

任意时刻：1/2 mv² + mgh = mgh_初始
```

**生活应用题**：
"某运动员从10m跳台跳下，重力势能最大时是哪个位置？动能最大时呢？"

**✅ 完成标准**：正确分析能量转换，理解机械能守恒条件

### 【Session 5】守恒思维 · 物理解题应用

**训练名称**：守恒法解题

**训练内容**：
**守恒定律解题策略**

**题目**：一个物体从斜面顶端滑下，到达底部时的速度为4m/s。已知斜面高2m，g=10m/s²。求物体与斜面间的摩擦产生的热量。

**解题策略**：

| 步骤 | 方法 |
|------|------|
| 1. 判断守恒条件 | 有摩擦力，非机械能守恒 |
| 2. 选择守恒定律 | 能量守恒（机械能不守恒，但总能量守恒） |
| 3. 列方程 | mgh = 1/2 mv² + Q |
| 4. 求Q | Q = mgh - 1/2 mv² |

**解答**：
```
mgh = 1/2 mv² + Q
Q = mgh - 1/2 mv²
  = m×10×2 - 1/2 m×16
  = 20m - 8m
  = 12m（J）

答：摩擦产生热量12m焦耳（与质量成正比）
```

**✅ 完成标准**：掌握守恒法解题流程，正确应用能量守恒

### 【Session 6】守恒思维 · 化学解题应用

**训练名称**：守恒法解化学题

**训练内容**：
**质量守恒在化学计算中的应用**

**题目**：
"将10g碳酸钙高温分解，完全反应后得到4.4g二氧化碳和生石灰。求生成生石灰的质量。"

**解题策略**：

| 步骤 | 方法 |
|------|------|
| 1. 写化学方程式 | CaCO₃ → CaO + CO₂↑ |
| 2. 标出已知量 | 10g → ? + 4.4g |
| 3. 应用质量守恒 | 反应物总质量 = 生成物总质量 |
| 4. 求未知量 | 生石灰质量 = 10g - 4.4g |

**解答**：
```
根据质量守恒定律：
m(CaCO₃) = m(CaO) + m(CO₂)
10g = m(CaO) + 4.4g
m(CaO) = 10g - 4.4g = 5.6g
```

**练习题**：
"6g碳在氧气中完全燃烧，生成22g CO₂，问消耗氧气多少克？"

**✅ 完成标准**：掌握质量守恒在化学计算中的应用

### 【Session 7】守恒思维 · 数学应用

**训练名称**：守恒思想在数学中的应用

**训练内容**：
**等量守恒思想**

**题目**：
"一个数先加上5，再乘以2，等于24。这个数是多少？"

**守恒思维分析**：

| 分析角度 | 内容 |
|---------|------|
| 等式守恒 | 等式两边同时变化，等量关系不变 |
| 逆向操作 | 减法是加法的逆运算，除法是乘法的逆运算 |
| 方程思想 | 设未知数，列方程，保持等式平衡 |

**解答方法**：
```
方法1：逆向运算
(□ + 5) × 2 = 24
□ + 5 = 24 ÷ 2 = 12
□ = 12 - 5 = 7

方法2：方程
设这个数为x
2(x + 5) = 24
x + 5 = 12
x = 7
```

**守恒思想总结**：
- 代数运算中，等式两边同时进行相同运算，等量关系保持
- 这是数学中的"守恒"思想

**✅ 完成标准**：理解等量守恒思想，掌握逆向运算方法

| 物理 | 机械能守恒 | 1/2mv² + mgh = 常数 | 只有重力/弹力做功 |
| 物理 | 动量守恒 | m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂' | 合外力为零 |
| 物理 | 电荷守恒 | Q总不变 | 孤立系统 |
| 化学 | 质量守恒 | m反应物 = m生成物 | 孤立系统 |
| 数学 | 等式守恒 | 两边同时运算，等式成立 | 无条件 |

**综合应用题**：
"一个带电小球（m=0.1kg，q=0.01C）从10m高处落下，忽略空气阻力，求落地时的速度。"

**解答**：
```
机械能守恒：
mgh = 1/2 mv²
v = √(2gh) = √(2×10×10) = √200 ≈ 14.1 m/s
```

**守恒思维核心**：
寻找变化中的不变量，建立等量关系
