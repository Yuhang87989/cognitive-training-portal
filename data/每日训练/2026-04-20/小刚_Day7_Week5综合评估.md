### 📊 任务1：守恒概念辨析

**📋 任务描述：**
请判断以下情境中，哪些量是守恒的（总量不变），哪些不是。

**情境组：**

| 情境 | 是否守恒？ | 理由 |
|-----|-----------|-----|
| 1. 把水从A杯倒到B杯 | ✅ 守恒 | 水量不变 |
| 2. 5+3=8，变成8=5+3 | ✅ 守恒 | 等式两边仍然相等 |
| 3. 把苹果分成两堆 | ✅ 守恒 | 苹果总数不变 |
| 4. 冰融化成水 | ⚠️ 需分析 | 质量守恒，但形态变化 |
| 5. 能量在转化（如动能→势能） | ✅ 守恒 | 机械能守恒（无摩擦时） |

**📝 你的分析：**

```
【守恒量识别训练】

✅ 明确守恒的量：
1. 质量：物体质量不随形态变化而改变
2. 电荷：电荷总量不变（摩擦起电中）
3. 能量：能量既不会凭空产生，也不会凭空消失
4. 数量：物品总数不变（只转移，不消失）
5. 等式关系：等式两边始终相等

⚠️ 需要条件的守恒：
1. 机械能：需要"无摩擦、无空气阻力"的条件
2. 动量：需要"系统不受外力"的条件

❌ 不守恒的情况：
1. 有摩擦时的机械能（转化为热能）
2. 有阻力时的运动（能量损耗）

【守恒思维的价值】
✓ 把复杂问题简化成"找不变量"
✓ 提供验证标准
✓ 帮助理解事物本质
```

### 📊 任务2：守恒定律应用练习

**📋 任务描述：**
应用守恒思维解决以下问题。

**问题1：能量守恒应用**
> 小明把一个质量为2kg的球举高5m，然后松手。
> 忽略空气阻力，问：
> （1）球在最高点的能量是什么？各是多少？
> （2）球落到地面时的速度是多少？

**📝 你的解答：**

```
【解答过程】

已知：m=2kg, g=10N/kg, h=5m

（1）最高点能量分析：
- 重力势能：Ep = mgh = 2×10×5 = 100J
- 动能：Ek = 0J（速度为0）

总机械能：E = Ep + Ek = 100J + 0J = 100J

【守恒检验】因为忽略空气阻力，机械能守恒

（2）落地时能量分析：
- 高度h=0，重力势能Ep = 0
- 根据机械能守恒：E落地 = E最高 = 100J
- 所以动能Ek = 100J
- Ek = ½mv² → 100 = ½×2×v²
- v² = 100 → v = 10m/s

【答案】
（1）最高点：重力势能100J，动能0J
（2）落地速度：10m/s
```
