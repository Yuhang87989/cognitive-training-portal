## Session 2 - 物理过程逆向分析

### 🔄 从结果倒推原因

**任务名称**：逆向工程师

**逆向分析原理**：
物理过程是可逆的吗？有些过程可逆，有些不可逆。学会逆向分析，能帮助我们更好地理解物理现象！

**可逆过程**：
- 简单机械（杠杆、滑轮）
- 光的反射
- 匀速运动

**不可逆过程**：
- 摩擦生热
- 扩散
- 能量转化（热能散失）

**练习内容**：

**练习1：运动逆向分析**
现象：一个物体从斜面滚下，越滚越慢，最后停在底部

逆向思考：
```
现在：静止
之前：在运动
为什么运动状态改变？→ 受到摩擦力
摩擦力从哪来？→ 物体与斜面接触
结论：摩擦力使物体停下来
```

**练习2：热传递逆向分析**
现象：冬天摸金属比摸木头更冷

逆向思考：
```
感觉：金属更冷
为什么会冷？→ 热量从手传到金属
为什么金属传热快？→ 金属导热性好
结论：金属导热快，热量散失多
```

**练习3：电学逆向分析**
现象：闭合开关，灯泡亮了

逆向思考：
```
灯泡亮 → 有电流通过 → 有电压 → 有电源
结论：电源提供电压，电流使灯泡发光
```

## Session 3 - 因果关系逆向推理

### 🔗 物理因果关系

**任务名称**：因果关系分析师

**因果关系类型**：

**类型1：一因一果**
- 原因 → 结果
- 例：用力推物体 → 物体运动

**类型2：多因一果**
- 多个原因 → 共同结果
- 例：速度和方向 → 运动状态

**类型3：一因多果**
- 一个原因 → 多个结果
- 例：加热 → 体积增大、温度升高、物态变化

## Session 4 - 实验设计逆向思维

### 🧪 从目标倒推实验方案

**任务名称**：逆向实验设计师

**逆向实验设计原理**：
先确定要验证的结论或要测量的量，然后倒推需要什么条件、什么步骤。

**逆向实验设计步骤**：

**第一步：明确目标**
- 要验证什么？
- 要测量什么？

**第二步：倒推需要的条件**
- 需要什么材料？
- 需要什么环境？

**第三步：设计实验步骤**
- 先做什么？
- 再做什么？
- 怎么比较？

**练习内容**：

