### 【Session 1】守恒的本质 · 概念建构 ⭐

**挑战名称**：守恒定律的科学定义

**核心概念**：
守恒定律指出：在封闭系统中，某些物理量 **不随时间变化**。

**任务内容**：
📖 **守恒定律体系梳理**

| 守恒类型 | 守恒量 | 适用条件 | 数学表达 |
|----------|--------|----------|----------|
| 能量守恒 | 能量E | 孤立系统 | ΔE₁ + ΔE₂ = 0 |
| 动量守恒 | 动量p | 合外力为零 | Σp = 常数 |
| 电荷守恒 | 电荷量Q | 孤立系统 | ΣQ = 常数 |
| 质量守恒 | 质量m | 化学反应 | Σm反应物 = Σm生成物 |

**思考问题**：
1. 为什么守恒定律这么重要？
2. 守恒与"不变量"有什么区别？

**✅ 完成标准**：完成表格填写，理解四大守恒定律的基本框架

### 【Session 2】能量守恒 · 定律推导 ⭐⭐

**挑战名称**：能量转换分析

**任务描述**：
从能量角度分析小球从斜面滚下的过程。

**任务内容**：
🔋 **能量转换计算**

**情境**：质量m=2kg的小球从h=5m高处静止滚下（忽略摩擦）

**已知条件**：
- 初始状态：高度h₁=5m，速度v₁=0
- 机械能守恒：Eₚ + Eₖ = 常数

**计算分析**：

| 状态 | 高度(m) | 速度(m/s) | 势能(J) | 动能(J) | 机械能(J) |
|------|---------|-----------|---------|---------|-----------|
| 初始 | 5 | 0 | ? | 0 | ? |
| 中间 | 2.5 | ? | ? | ? | ? |
| 底部 | 0 | ? | 0 | ? | ? |

**计算过程**：
```
重力势能：Ep = mgh
初始势能：Ep₁ = 2×10×5 = 100J

底部动能：Ek₂ = Ep₁ = 100J
速度计算：Ek = ½mv² → v = √(2Ek/m) = √(100/2) = √50 ≈ 7.07m/s
```

**✅ 完成标准**：完成表格计算，理解能量守恒的应用

### 【Session 3】动量守恒 · 碰撞分析 ⭐⭐

**挑战名称**：碰撞中的守恒

**任务描述**：
分析两球碰撞前后的动量变化。

**任务内容**：
⚡ **动量守恒计算**

**情境**：质量m₁=3kg的小球以v₁=4m/s向右运动，撞上静止的m₂=1kg的小球，碰撞后一起运动。

**动量守恒方程**：
```
m₁v₁ + m₂v₂ = (m₁ + m₂)v共

代入：3×4 + 1×0 = (3+1)v共
    12 = 4v共
    v共 = 3m/s
```

**扩展分析**：

| 分析维度 | 碰撞前 | 碰撞后 | 是否守恒 |
|----------|--------|--------|----------|
| 总动量 | ? kg·m/s | ? kg·m/s | ✓ |
| 总动能 | ? J | ? J | ? |

**思考**：碰撞后动能是否守恒？为什么？

**✅ 完成标准**：完成动量计算，分析动能变化原因

### 【Session 5】守恒法解题 · 策略训练 ⭐⭐⭐

**挑战名称**：守恒法vs过程法的抉择

**任务描述**：
同一道题，用两种方法求解，比较优劣。

**任务内容**：
📝 **一题多解训练**

**题目**：小球从h=20m高处自由下落，求落地速度。

**方法一：过程法（运动学）**
```
v² = 2gh
v = √(2×10×20) = √400 = 20m/s
```

**方法二：能量守恒法**
```
初始机械能 = 底部机械能
mgh = ½mv²
v = √(2gh) = √400 = 20m/s
```

**对比分析**：

| 分析维度 | 过程法 | 守恒法 |
|----------|--------|--------|
| 适用场景 | | |
| 计算复杂度 | | |
| 优势 | | |
| 劣势 | | |

**何时优先用守恒法？**
1. 过程复杂、变量多时
2. 只需要初末状态时
3. 某些中间量难以计算时

**✅ 完成标准**：完成对比分析，总结守恒法适用条件

### 【Session 6】守恒边界 · 条件与限制 ⭐⭐⭐

**挑战名称**：守恒的"边界条件"

**任务描述**：
理解守恒定律的适用条件，学会判断守恒是否成立。

**任务内容**：
🚧 **守恒条件分析**

**能量守恒的边界条件**：
```
适用条件：系统与外界无能量交换（孤立系统）

不适用场景：
- 有摩擦力做功（机械能→内能）
- 有空气阻力
- 有外力做功

例外处理：
- 若考虑内能，总能量仍守恒
- 机械能不守恒，但总能量守恒
```

**守恒思维的精髓**：
> 不是"能量不变"，而是"总能量不变"
> 不是"机械能守恒"，而是"在保守力系统中机械能守恒"

**应用训练**：

| 问题情境 | 守恒类型 | 守恒是否成立 | 说明 |
|----------|----------|--------------|------|
| 弹簧振子（忽略阻力） | 机械能 | ✓ | 保守力系统 |
| 弹簧振子（有阻力） | 机械能 | ❌ | 阻力耗散能量 |
| 弹簧振子（有阻力） | 总能量 | ✓ | 考虑内能 |
| 完全非弹性碰撞 | 动量 | ✓ | 内力不改变总动量 |

**✅ 完成标准**：完成表格分析，理解守恒边界
