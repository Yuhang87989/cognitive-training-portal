### 【Session 1】守恒入门 · 什么是守恒 ⭐

**挑战名称**：守恒的基本概念

**核心概念**：
守恒就是"某个量在变化过程中始终保持不变"。

**生活中的守恒例子**：
- 💰 零花钱：花了5元又攒了5元，总数不变
- 🍎 苹果：吃掉2个、买了3个，还是那么多
- 📦 积木：无论怎么拼，积木数量不变

**科学中的守恒**：
| 守恒类型 | 守恒的"量" | 例子 |
|----------|------------|------|
| 质量守恒 | 物质总量 | 冰熔化成水，质量不变 |
| 能量守恒 | 能量总量 | 动能↔势能，总和不变 |
| 动量守恒 | 运动总量 | 碰撞前后动量总和不变 |

**✅ 完成标准**：用自己的话解释什么是守恒，举出1个生活例子

### 【Session 2】能量守恒 · 基础计算 ⭐⭐

**挑战名称**：能量转换大师

**任务描述**：
理解能量如何在动能和势能之间转换。

**任务内容**：
⚡ **能量守恒基础计算**

**情境**：一个物体从高处落下（忽略空气阻力）

**能量守恒公式**：
```
机械能守恒：E总 = E动能 + E势能 = 常数

即：½mv² + mgh = 常数
```

**计算示例**：
质量m=2kg的物体从h=5m高处静止下落

| 状态 | 高度h | 速度v | 动能½mv² | 势能mgh | 总能量 |
|------|-------|-------|----------|--------|--------|
| 开始 | 5m | 0 | 0J | 2×10×5=100J | 100J |
| 中间 | 2m | ? | ? | ? | ? |
| 落地 | 0 | ? | ? | 0 | 100J |

**计算过程**：
```
底部势能为0，所以全部动能：
½mv² = mgh = 100J
v² = 100/2×2 = 100
v = 10m/s
```

**✅ 完成标准**：完成表格计算，理解能量守恒

### 【Session 3】动量守恒 · 碰撞基础 ⭐⭐

**挑战名称**：碰撞中的动量

**任务描述**：
理解碰撞前后动量如何守恒。

**任务内容**：
🔄 **动量守恒基础**

**动量公式**：
```
动量 p = mv（质量×速度）

动量守恒条件：系统不受外力（或合外力为零）

守恒方程：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
（碰撞前总动量 = 碰撞后总动量）
```

**简单计算**：

**情境A**：一个静止的小球被另一个球撞上
- 撞前：m₁=3kg以v₁=4m/s运动，m₂=2kg静止
- 撞后：两球粘在一起运动

```
守恒方程：m₁v₁ + m₂v₂ = (m₁+m₂)v
3×4 + 2×0 = (3+2)v
12 = 5v
v = 2.4m/s
```

**情境B**（你来算）：
- 撞前：m₁=5kg以v₁=6m/s运动，m₂=1kg以v₂=-2m/s（向左）运动
- 撞后：一起运动

```
守恒方程：5×6 + 1×(-2) = (5+1)v
? = 6v
v = ? m/s
```

### 【Session 4】守恒识别 · 判断技巧 ⭐⭐

**任务描述**：
学会快速判断哪些情况下守恒成立。

**任务内容**：
📋 **守恒判断法则**

**能量守恒判断**：
```
✓ 成立条件：只有重力/弹力做功，没有摩擦和阻力

✓ 适用：自由落体、竖直上抛、摆动、弹簧振子（理想）
✗ 不适用：有摩擦、空气阻力、有外力做功
```

**动量守恒判断**：
```
✓ 成立条件：系统所受合外力为零

✓ 适用：碰撞、爆炸、相互作用
✗ 不适用：有持续外力作用（如地面上滑动）
```

**练习判断**：

| 情境 | 能量守恒？ | 动量守恒？ | 原因 |
|------|------------|------------|------|
| 物体自由下落 | ✓ | ✗ | 有重力（外力） |
| 两球弹性碰撞 | ✓ | ✓ | 无外力、无能量损失 |
| 滑块在冰面滑动 | ✗ | ✓ | 有摩擦 |
| 子弹射入木块 | ✗ | ✓ | 机械能损失 |

### 【Session 5】守恒法解题 · 简单应用 ⭐⭐⭐

**挑战名称**：用守恒法解简单问题

**任务描述**：
用守恒思维解决物理问题。

**任务内容**：
📝 **守恒法解题训练**

**题目1**：一个质量1kg的物体从10m高处自由下落，求：
① 落地时的速度
② 下落一半高度时的速度

**解法（能量守恒法）**：
```
① 落地时：mgh₁ = ½mv²
   1×10×10 = ½×1×v²
   v² = 200
   v = 10√2 ≈ 14.1 m/s

② 一半高度时：h = 5m
   mgh₁ = mgh₂ + ½mv²
   100 = 50 + ½v²
   v² = 100
   v = 10 m/s
```

**题目2**（你来做）：
一个质量0.5kg的小球从h=3.2m高处滑下，滑到水平面时的速度是多少？

```
解：能量守恒
mgh = ½mv²
v = √(2gh) = √(2×10×3.2) = ? m/s
```

### 【Session 6】质量守恒 · 化学基础 ⭐⭐

**挑战名称**：化学反应中的守恒

**任务描述**：
理解化学反应中的质量守恒。

**任务内容**：
⚗️ **质量守恒定律**

**定义**：化学反应中，参加反应的物质质量总和等于反应后生成的物质质量总和。

**经典实验**：镁条燃烧
```
反应前：镁条 + 氧气 = ？
反应后：氧化镁

注意：氧化镁的质量 = 镁条质量 + 参与反应的氧气质量
```

**数学表达**：
```
m反应物 = m生成物

例如：2H₂ + O₂ → 2H₂O
      4g  + 32g = 36g
```

**计算练习**：

| 碳燃烧 | 6g C + ?g O₂ | 22g CO₂ | ? |

### 【Session 7】守恒综合 · 简单综合问题 ⭐⭐⭐

**挑战名称**：综合守恒应用

**任务描述**：
在一个简单情境中综合应用守恒思维。

**任务内容**：
🌟 **综合守恒分析**

**情境**：一个质量2kg的物体从高h=5m的斜面顶端滑下，到达底部时速度为8m/s。

**分析问题**：
1. **机械能守恒吗？**
   - 判断：不守恒（因为有摩擦）
   
2. **如果不守恒，能量去哪了？**
   - 答案：转化为内能（摩擦生热）

3. **计算内能损失**：
```
初始机械能：E₁ = mgh = 2×10×5 = 100J
底部机械能：E₂ = ½mv² = ½×2×8² = 64J
内能损失：ΔE = E₁ - E₂ = 100 - 64 = 36J
```

**关键理解**：
- 机械能不守恒，但总能量守恒
- 损失的36J转化为了内能（热）

**练习**（你来算）：
一个质量1kg的物体从h=8m处滑下，底部速度为10m/s，求：
① 初始机械能
② 底部机械能
③ 内能损失

```
① E₁ = mgh = ? J
② E₂ = ½mv² = ? J
③ ΔE = ? J
```
