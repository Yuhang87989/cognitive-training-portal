## Session 2：注意力训练 - 错误分类与归因分析

**方法名称**：错误四象限分析法

**分析框架**：
| 错误类型 | 特征 | 典型案例 | 解决策略 |
|---------|------|----------|----------|
| **知识漏洞型** | 概念不清、公式记错 | 误用平方差公式 | 概念重构+基础练习 |
| **思维偏差型** | 思路错误、方法不当 | 应用题建模错误 | 思维导图+方法对比 |
| **计算失误型** | 粗心、步骤漏算 | 进位错误、符号看错 | 计算规范+检查流程 |
| **理解偏差型** | 题意误解、条件遗漏 | 将“增加到”理解为“增加了” | 关键词标注+题意复述 |

**分析任务**：
分析以下错题，判断错误类型并说明理由：
> 题目：若x²-5x+6=0，求x的值。
> 错误解法：x²-5x+6=(x-2)(x-3)=0 → x=2或x=3（正确）
> 但学生写成了x=5或x=1

**错误归因**：
- 属于哪一类型？为什么？
- 反映出什么思维问题？
- 如何针对性强化？

## Session 6：学霸方法专题 - 反证法初步应用

**思维工具**：反证法（逆向证明策略）

**核心逻辑**：
要证明命题P为真，先假设P为假，推导出矛盾，从而证明P必须为真。

**应用场景**：
1. 唯一性证明
2. 无穷性证明
3. 不可能性证明
4. 存在性证明

**经典案例**：
证明：√2是无理数
1. 假设√2是有理数，可表示为既约分数a/b
2. 则2 = a²/b² → a² = 2b²
3. 故a²是偶数，a也是偶数，设a=2c
4. 代入得4c² = 2b² → b² = 2c²
5. 故b²也是偶数，b也是偶数
6. a和b都是偶数，与既约分数矛盾
7. 所以假设错误，√2是无理数

**思维训练**：
用反证法证明：素数有无穷多个
（提示：假设素数有限，构造新数N=所有素数乘积+1）

**逆向思维拓展**：
- 反证法体现了什么样的思维品质？
- 在哪些学科中反证法特别有用？

## Session 8：逆向思维进阶 - 复杂问题反向分析

**训练项目**：多步骤逆向推理挑战

**问题原型**：
> 一个三位数，百位数字是十位数字的2倍，十位数字比个位数字大1。如果将这个三位数的数字顺序颠倒，得到的新数比原数小396。求原数。

**正向分析难点**：
- 涉及多个变量关系
- 需要建立方程组
- 计算过程复杂

**逆向分析策略**：
1. **从结果反推**：
   - 差值为396，考虑数字反转的数学规律
   - 设原数为100a+10b+c，新数为100c+10b+a
   - 则(100a+10b+c) - (100c+10b+a) = 99(a-c) = 396
   - 得a-c = 4

2. **结合条件限制**：
   - a = 2b（百位是十位的2倍）
   - b = c+1（十位比个位大1）
   - a-c = 4

3. **系统求解**：
   - 由a-c=4，a=2b，b=c+1
   - 得2b - c = 4，代入b=c+1 → 2(c+1)-c=4 → c=2
   - 则b=3，a=6
   - 原数为632

**思维训练**：
设计一个需要至少三步逆向推理的数学问题，并给出解答思路。

**能力评估**：
- 逆向分析相比正向分析的优势是什么？
- 如何培养系统性逆向推理能力？
