## Session 1：注意力训练 - 解题步骤闯关第一关

**游戏名称**：数学冒险岛·读题侦查关

欢迎来到数学冒险岛！今天你要扮演一名小探险家，通过四个关卡掌握解题的完整步骤。

**第一关：读题侦查**
- **任务描述**：阅读以下问题，找出所有数字和关键词
  > 小明有15本故事书，小华比小明多8本。小华有多少本故事书？

- **侦查任务**：
  1. 圈出所有数字（15、8）
  2. 标出关键词（“比...多”）
  3. 判断问题类型：比较型加法问题

- **时间限制**：1分钟完成侦查
- **成功标准**：正确找出所有关键信息

**思维提示**：
- “比...多”表示要使用加法
- 问题问的是小华的数量，需要从小明的数量出发

## Session 2：注意力训练 - 解题步骤闯关第二关

**第二关：条件分析**
- **任务描述**：分析问题的已知条件和未知数

- **分析任务**：
  1. 已知条件：
     - 小明有15本书
     - 小华比小明多8本
  2. 未知数：小华的书本数量
  3. 关系：小华 = 小明 + 8

- **可视化工具**：画出小明和小华的书本数量关系图
  ```
  小明：15本
  小华：15本 + 8本 = ？
  ```

- **时间限制**：1分钟完成分析
- **成功标准**：清晰列出所有条件并理解关系

**逆向思维练习**：
- 如果知道小华有23本书，小华比小明多8本，小明有多少本书？

## Session 4：记忆力训练 - 解题步骤应用练习

**应用场景**：商店购物计算

**问题**：
> 一瓶果汁5元，小明买了3瓶，付给收银员20元，应该找回多少钱？

**解题步骤应用**：
1. **读题侦查**：圈出数字（5、3、20），标出关键词（“找回”）
2. **条件分析**：
   - 已知：单价5元，数量3瓶，付款20元
   - 未知：找回金额
   - 关系：总价 = 单价 × 数量，找回 = 付款 - 总价
3. **方法选择**：先乘法后减法
4. **验证检查**：用找回金额 + 总价 = 付款验证

**计算过程**：
- 总价 = 5 × 3 = 15元
- 找回 = 20 - 15 = 5元

**逆向验证**：
- 如果找回5元，总价15元，付款应该是20元，正确！

## Session 5：学霸方法专题 - 错题本初体验

**专题名称**：我的第一本错题本

**学习目标**：
- 了解错题本的作用和价值
- 学习记录一道错题的基本方法

**错题本是什么**：
错题本就像是你的“学习病历本”，记录你做错的题目、错误原因和正确解法，帮助你避免重复犯错。

**记录示范**：
```
题目：15 + 8 = ?
我的错误答案：22
正确答案：23
错误原因：把加法算成了减法（15-8=7的错误思维干扰）
正确解法：15 + 8 = 23
经验教训：仔细看运算符号，不要想当然
```

**你的任务**：
假设你刚才把“15+8”算成了22，按照示范格式记录这道错题。

**思维拓展**：
- 为什么记录错误原因很重要？
- 错题本多长时间复习一次比较好？

## Session 6：学霸方法专题 - 解题策略深化

**策略名称**：问题类型识别技巧

**学习目标**：
- 掌握3种常见问题类型的特征
- 学会快速识别问题类型的方法

**三种常见类型**：
1. **合并型**：“一共”“总共”“合起来”→加法
2. **比较型**：“比...多”“比...少”“相差”→加法或减法
3. **分配型**：“平均分”“每份”“平分”→除法

**识别练习**：
判断以下问题属于哪种类型：
1. 小红有12支铅笔，小刚有9支，两人一共有多少支铅笔？（合并型）
2. 小明身高140厘米，小华比小明高15厘米，小华多高？（比较型）
3. 老师把24颗糖平均分给6个小朋友，每人几颗？（分配型）

**应用任务**：
遇到新问题时，先判断类型，再选择方法。

**逆向思维应用**：
- 如果是合并型问题，已知“一共”和其中一个部分，如何求另一个部分？

## Session 7：听课习惯培养 - 课中专注技巧

**技巧名称**：主动倾听三步法

**学习目标**：
- 掌握课堂听讲时的专注技巧
- 提高听课效率和学习效果

**三步法详解**：
1. **课前准备**：提前预习，了解今天要学什么，带着问题听课
2. **课中互动**：
   - 眼睛看着老师或黑板
   - 跟着老师的思路走，心里默默回答老师的问题
   - 及时记录重点和疑问
3. **课后回顾**：下课后花1分钟回忆本节课的主要内容

**课堂模拟练习**：
假设现在正在上数学课，老师正在讲解“两位数加法”。
- 第一步：调整坐姿，眼睛看向“虚拟黑板”
- 第二步：跟着以下思路走：
  “今天学习两位数加法，重点是数位对齐...”
- 第三步：记录一个重点：“数位要对齐，从个位加起”

**专注挑战**：
在接下来的一天中，尝试在至少一节课上使用这个方法。

## Session 8：逆向思维进阶 - 反向设问训练

**训练名称**：侦探思维反转游戏

**学习目标**：
- 学会从结果倒推条件的思维方式
- 培养多角度思考问题的能力

**游戏规则**：
给你一个答案，请你设计一个能得到这个答案的问题。

**示例**：
- 答案：23
- 可能的问题：
  1. 小明有15本书，小华比小明多8本，小华有多少本书？
  2. 20减去多少等于-3？（注意：这是负数，适合高年级）
  3. 一个数加上8等于31，这个数是多少？

**你的任务**：
为答案“18”设计3个不同的问题。

**挑战升级**：
设计一个需要两步计算才能得到答案“18”的问题。

**思维拓展**：
- 为什么反向设问能帮助我们更好地理解问题？
- 在检查答案时，反向思考有什么用处？
