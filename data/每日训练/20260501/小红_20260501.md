### 【Session 2】能力组合应用 · 解决复杂问题

**训练名称**：综合问题解决

**训练内容**：
**多能力协同解决问题**

**综合问题**：

某工厂生产一种产品，成本为每个20元。原材料价格上涨10%后，工厂通过改进工艺使单个产品人工成本降低15%。若产品售价保持不变，请分析：
1. 产品利润率如何变化？
2. 如果要保持原利润率，售价应调整多少？

**能力组合分析**：

| 解决问题需要的 | 具体应用 |
|---------------|---------|
| 数学思维 | 建立数学模型、计算变化率 |
| 守恒思维 | 成本总量分析、利润守恒 |
| 逻辑推理 | 分析各因素关系 |
| 元认知 | 监控解题过程、检查漏洞 |

**解题过程**：

**Step 1：问题建模**
```
原成本 = 20元
材料上涨10% → 新材料成本 = 20 × 10% = 2元
人工降低15% → 设原人工成本为x，新人工成本 = 0.85x
```

**Step 2：方程构建**
```
新总成本 = 20 + 2 + 0.85x = 22 + 0.85x
原利润率 = (售价 - 20) / 20 × 100%
新利润率 = (售价 - 新成本) / 新成本 × 100%
```

**Step 3：求解分析**
需要假设原人工成本才能具体计算，这是一个典型的"条件不足需要假设"的问题

**元认知监控**：
- 解答完成后检查：模型是否正确？计算是否准确？
- 反思：这道题体现了哪些能力的综合应用？

**✅ 完成标准**：完成综合问题解决，记录能力组合分析

### 【Session 3】跨学科挑战 · 物理+数学+逻辑

**训练名称**：跨学科整合

**训练内容**：
**综合学科问题解决**

**问题情境**：

小球从倾角为30°的斜面顶端由静止开始下滑，已知斜面长度为L，摩擦系数为μ。求：
1. 小球滑到底端的速度
2. 如果斜面倾角变为45°，速度如何变化
3. 讨论速度与倾角的关系

**能力组合应用**：

| 能力 | 应用 |
|------|------|
| 物理思维 | 受力分析、能量守恒 |
| 数学思维 | 三角函数、几何关系 |
| 守恒思维 | 能量守恒、机械能守恒 |
| 逻辑推理 | 因果分析、归纳总结 |

**解题过程**：

**Step 1：物理分析**
- 受力分析：重力mg、支持力N、摩擦力f
- 沿斜面方向：mgsinθ - μmgcosθ = ma

**Step 2：能量分析**
- 初始机械能：E₁ = mgLsinθ
- 末态动能：E₂ = 1/2 mv²
- 热量损失：Q = fL = μmgcosθ × L
- 能量守恒：mgLsinθ = 1/2 mv² + μmgLcosθ

**Step 3：数学求解**
```
v² = 2gL(sinθ - μcosθ)
v = √[2gL(sinθ - μcosθ)]
```

**Step 4：倾角变化分析**
- θ=30°：v₁ = √[2gL(sin30° - μcos30°)]
- θ=45°：v₂ = √[2gL(sin45° - μcos45°)]
- 比较：需要知道μ的具体值才能判断

**✅ 完成标准**：完成跨学科问题解决
