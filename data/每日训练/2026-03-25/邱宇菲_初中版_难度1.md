## Session 2 - 高级听觉任务训练

### 👂 复杂听觉处理：多层次倾听

**任务名称**：声音多面手

**练习内容**：

**练习1：双层信息处理**
听一段话，同时关注两个层面：
内容："今天数学课我们学习分数的加法"
情绪：老师的语气是开心的还是严肃的？

答案：
- 内容：_____________
- 情绪：_____________

**练习2：信息分类**
听一组词语，按类别分类：
词语：苹果、香蕉、桌子、椅子、汽车、梨、火车、橙子

- 水果类：_____________
- 家具类：_____________
- 交通工具：_____________

**练习3：关键信息捕捉**
听一段话（家长朗读或自己阅读）：
"明天的科学实验需要准备以下材料：一个烧杯、50毫升水、两勺盐、一个搅拌棒。请大家提前准备好，如果没有烧杯可以用玻璃杯代替。"

回答：
- 实验需要什么材料？
- 如果没有烧杯怎么办？
- 你有没有这些材料？

**听觉技巧**：
- 带着问题去听
- 边听边在脑中构建画面
- 主动分类整理信息

## Session 3 - 听课习惯培养：课前预习

### 📚 预习系统化：让课堂更高效

**任务名称**：结构化预习法

**超能力介绍**：
高效的预习能让课堂效率提升50%以上！结构化预习法让你带着问题和准备进入课堂。

**预习三步法详解**：

**第一步：概览（1-2分钟）**
- 看标题猜内容
- 看小标题了解结构
- 看图片图表获取直观印象

**第二步：标记（2-3分钟）**
- 用"√"标已懂的内容
- 用"？"标不懂的内容
- 用"△"标重要内容

**第三步：提问（1-2分钟）**
- 根据标题提问
- 根据不懂的地方提问
- 写下2-3个最想知道的问题

**练习内容**：

**练习1：预习模拟练习**
假设明天要学习"分数的意义"，进行预习：
```
预习内容：分数的意义

第一步（概览）：
- 标题暗示：_____________
- 可能包含：_____________

第二步（标记）：
- 已懂：_____________
- 不懂：_____________
- 重点：_____________

第三步（提问）：
- 问题1：_____________
- 问题2：_____________
```

**练习2：真实课文预习**
找一篇语文课文或科学章节，实际练习预习：
课文名称：_____________
```
概览收获：_____________
标记情况：_____________
我的问题：_____________
```

## Session 4 - 听课习惯培养：课中专注

### 🎯 课堂专注：抓住每一分钟

**任务名称**：超级专注听课法

**专注技巧学习**：

**技巧1：眼睛锁定法**
- 选择老师的眼睛或嘴巴作为焦点
- 跟随老师的思路移动视线
- 避免看窗外、看手机等

**技巧2：耳朵追踪法**
- 注意老师的"重点词"：
  - "同学们注意了..."
  - "这个很重要..."
  - "考试会考..."
  - "因为...所以..."
  - "也就是说..."
- 听到这些词，立刻提高注意力

**技巧3：笔记锚点法**
- 老师讲新内容时，在笔记本上画⏺
- 老师讲例题时，在笔记本上画⏺
- 老师重复强调时，在笔记本上画⏺

**练习内容**：

**练习1：专注力自测**
假设这是一段20分钟的课堂片段，你会：
```
0-5分钟（开始）：_____________
6-10分钟（中期）：_____________
11-15分钟（疲劳期）：_____________
16-20分钟（结尾）：_____________
```
分析：_____________

**练习2：专注力提升计划**
针对不同时段设计应对策略：
```
如果开小差（0-5分钟）：
- 应对方法：_____________

如果太累（11-15分钟）：
- 应对方法：_____________
```

**练习3：课堂笔记示范**
以下面这段话为例，记录重点：
"光合作用需要三个条件：阳光、二氧化碳和水。阳光是能量来源，二氧化碳是原料，水是参与反应的物质。"

你的笔记：
```
光合作用三条件：
1. _______ - 能量来源
2. _______ - 原料
3. _______ - 参与反应
```

## Session 5 - 课后复习习惯培养

### 📝 课后复习：趁热打铁

**任务名称**：黄金复习时间

**超能力介绍**：
根据遗忘曲线，下课后10分钟内复习，能记住80%的内容！这就是"黄金复习时间"。

**课后复习四步法**：

**第一步：快速回顾（2分钟）**
- 闭上眼睛，回想今天学了什么
- 不要翻书，凭记忆回忆

**第二步：查看笔记（1分钟）**
- 检查笔记是否完整
- 补充遗漏的内容

**第三步：解决疑问（2分钟）**
- 当时没听懂的地方，现在懂了吗？
- 还不懂的，记录下来，准备问老师

**第四步：简单练习（5分钟）**
- 做1-2道相关练习
- 检验是否真的理解

**练习内容**：

**练习1：复习时间表设计**
设计你的课后复习计划：
```
今天最后一节课：______
下课时间：______
黄金复习时间：______-______（10分钟内）
复习内容：______
复习地点：______
```

**练习2：知识回顾练习**
回忆今天学到的最重要的3个知识点：
1. _____________
2. _____________
3. _____________

如果能回忆起来，说明你已经掌握了！

**练习3：疑问记录表**
记录还没解决的问题：
```
疑问1：_____________
疑问2：_____________
计划解决方式：_____________
```
