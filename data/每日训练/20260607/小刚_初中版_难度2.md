## 🔬 第2次训练：环境系统分析

**模块**：环境系统分析  
**时长**：约5分钟

### 系统动力学视角

运用系统思维分析环境问题的深层结构。

### 环境系统的组成要素

| 要素类型 | 构成要素 | 相互作用 |
|---------|---------|---------|
| 经济系统 | 产业、资源、消费 | 生产-消耗 |
| 社会系统 | 人口、文化、制度 | 需求-供给 |
| 生态系统 | 气候、生物、土壤 | 物质-能量循环 |
| 技术系统 | 能源、交通、信息 | 效率-创新 |

### 系统基模分析

**公地悲剧模型**：

```
┌─────────────────────────────────────┐
│                                     │
│    个人理性 → 集体非理性            │
│         ↓                           │
│    过度使用 → 资源枯竭              │
│         ↑                           │
│    利益驱动 ← 竞争机制              │
│                                     │
└─────────────────────────────────────┘
```

**增强回路与调节回路**：

| 回路类型 | 环境问题表现 | 解决思路 |
|---------|-------------|---------|
| 增强回路 | 消费增长→生产扩大→污染增加→环境恶化 | 引入调节机制 |
| 调节回路 | 环境恶化→健康损失→意识觉醒→行为改变 | 强化正向激励 |

### 反馈机制分析

**正反馈**：加速环境恶化
- 冰川融化 → 反射率降低 → 进一步升温
- 森林减少 → 生物多样性下降 → 生态系统崩溃

**负反馈**：自我调节机制
- 污染加剧 → 治理投入增加 → 环境改善
- 资源短缺 → 价格上升 → 节约意识增强

### 可持续发展深度解读

**经典定义**（Brundtland报告，1987）：
> 既满足当代人的需求，又不损害后代人满足其需求能力的发展。

**三维可持续发展模型**：

```
           经济可持续
              ▲
             ╱ ╲
            ╱   ╲
           ╱     ╲
    社会←────┼────→环境
   可持续   ╲     ╱   可持续
            ╲   ╱
             ╲ ╱
              ▼
          协调发展
```

### 行星边界理论

| 边界 | 安全边界 | 当前状态 | 风险等级 |
|-----|---------|---------|---------|
| 气候变化 | < 350 ppm CO₂ | > 420 ppm | 🔴 高 |
| 生物多样性 | < 10% 损失 | > 30% 损失 | 🔴 高 |
| 氮磷循环 | < 边界内 | 超出2倍 | 🔴 高 |
| 土地利用 | < 15% 变化 | ~20% 变化 | 🟡 中 |
| 淡水使用 | < 4000 km³ | ~4000 km³ | 🟡 中 |
| 臭氧层 | > 276 DU | ~300 DU | 🟢 低 |

## 🔄 第5次训练：生命周期评估

**模块**：生命周期评估  
**时长**：约4分钟

### 生命周期评估（LCA）方法

评估产品或系统全生命周期的环境影响。

### LCA框架

```
       ┌─────────────┐
       │ 原材料获取  │←─ 资源开采
       └──────┬──────┘
              ↓
       ┌──────┴──────┐
       │   生产制造  │←─ 能源消耗
       └──────┬──────┘
              ↓
       ┌──────┴──────┐
       │   运输分销  │←─ 物流排放
       └──────┬──────┘
              ↓
       ┌──────┴──────┐
       │   使用阶段  │←─ 能耗与维护
       └──────┬──────┘
              ↓
       ┌──────┴──────┐
       │   回收处理  │←─ 废弃物处理
       └─────────────┘
```

### 产品碳足迹计算

| 阶段 | 碳排放占比 | 降低策略 |
|-----|----------|---------|
| 原材料获取 | 30% | 使用再生材料 |
| 生产制造 | 35% | 清洁能源替代 |
| 运输分销 | 10% | 本地化采购 |
| 使用阶段 | 20% | 能效提升 |
| 回收处理 | 5% | 循环利用 |

### 对比分析案例

**一次性筷子 vs 可重复使用筷子**：

| 指标 | 一次性筷子 | 可重复使用筷子 | 结论 |
|-----|----------|--------------|------|
| 生产碳足迹 | 8g CO₂/双 | 50g CO₂/双 | 一次性更低 |
| 使用碳足迹 | 0 | 5g CO₂/次 | 可重复更低 |
| 10次使用 | 80g CO₂ | 100g CO₂ | 一次性更低 |
| 50次使用 | 400g CO₂ | 300g CO₂ | 可重复更低 |
| 临界点 | 约15次 | 约15次 | - |
