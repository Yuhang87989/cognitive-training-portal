## 🧠 第2次训练：习惯固化三步法记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
记忆习惯固化的三个步骤，按顺序复述并理解每个步骤的含义。

### 今日记忆内容

```
┌─────────────────────────────────────────────────┐
│  🧠 习惯固化三步法                               │
│                                                 │
│  第一步：固定时间启动                             │
│  - 每天固定时间开始学习，建立条件反射              │
│  - 比如：每天18:00准时开始训练                   │
│                                                 │
│  第二步：反复运用练习                             │
│  - 用同一方法解决不同问题，形成自动化运用          │
│  - 比如：每次学习都使用费曼技巧                   │
│                                                 │
│  第三步：标准化操作流程                           │
│  - 将复杂方法拆解为简单步骤，形成SOP              │
│  - 比如：预习→听课→复习→作业的标准流程            │
│                                                 │
│  ⏱️ 记忆时间：45秒                                │
└─────────────────────────────────────────────────┘
```

**回忆测试**：
1. 习惯固化的第一步是什么？_____
2. 第二步强调什么？_____
3. SOP是什么的缩写？_____

**答案**：固定时间启动、反复运用练习、标准化操作流程

### 挑战升级
- 🌟 基础：记住2个步骤
- ⭐⭐ 进阶：记住全部3个步骤和核心内容
- ⭐⭐⭐ 大师：能用自己的话解释每个步骤的意义

## 👀 第3次训练：体系稳定性观察游戏

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
观察以下场景，判断学习体系在哪些情况下表现稳定，哪些情况下可能出现波动。

### 今日观察题目

**场景A**：在安静的书房学习，没有手机干扰，按照计划完成了所有任务。
**场景B**：在嘈杂的客厅学习，电视开着，忍不住频繁走神。
**场景C**：身体有点不舒服，但还是坚持完成了基础训练任务。
**场景D**：遇到很难的题目，一开始就放弃了，没有尝试不同方法。

**稳定性判断**：
1. 哪个场景体现了学习体系的高度稳定？_____
2. 哪个场景说明注意力系统需要加强？_____
3. 哪个场景体现了执行系统的韧性？_____

**答案**：场景A、场景B、场景C

### 挑战升级
- 🌟 基础：正确判断2个场景
- ⭐⭐ 进阶：正确判断全部3个问题
- ⭐⭐⭐ 大师：为场景D提出3条体系优化建议

## 💡 第7次训练：习惯固化强化练习

**模块**：学习方法  
**时长**：约1分钟

### 今日练习
选择一个你想固化的学习习惯，设计一个7天强化练习计划。

### 习惯选择
**我要固化的习惯**：________

### 7天强化计划

| 天数 | 练习内容 | 完成标记 |
|-----|---------|---------|
| 第1天 | 明确习惯的具体操作步骤 | ☐ |
| 第2天 | 在固定时间刻意练习这个习惯 | ☐ |
| 第3天 | 记录练习过程中的感受和困难 | ☐ |
| 第4天 | 根据反馈调整练习方法 | ☐ |
| 第5天 | 在不同场景下练习这个习惯 | ☐ |
| 第6天 | 自动化尝试：不刻意提醒也能做到 | ☐ |
| 第7天 | 评估习惯固化程度，制定后续计划 | ☐ |

### 关键提示
习惯固化的关键是**重复+反馈+坚持**，通常需要21天才能真正形成稳定习惯。
