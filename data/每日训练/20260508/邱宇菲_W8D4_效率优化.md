## Session 1：效率认知启动（3分钟）
### 🎯 注意力追踪游戏 - 效率扫描仪
**训练目标**：建立效率意识，识别学习中的时间浪费点

**游戏规则**：
1. 屏幕上会快速显示多个图标组合
2. 每组图标代表一种学习活动（📚阅读、✍️做题、📱手机、🍵休息、🤔思考）
3. 用"扫描仪"找出效率最高的那一组
4. 回答：这个组合中高效活动多还是低效活动多？

**题目示例**：
- 组合A: 📚📚📱✍️✍️ → 高效活动(4) vs 低效活动(1) → ✅高效
- 组合B: 📱🍵📱🤔📱 → 高效活动(1) vs 低效活动(4) → ❌低效

**训练任务**：
完成10组效率扫描，找出最高效和最低效的组合，并思考为什么会这样。

---

## Session 2：时间日志分析（2分钟）
### 🧠 记忆力挑战 - 时间旅行箱
**训练目标**：学会记录和分析时间使用情况

**方法介绍**：效率优化四步法
1. **记录**：记录学习时间的使用情况（时间日志）
2. **分析**：找出时间浪费的环节和原因
3. **优化**：设计更高效的学习流程和策略
4. **验证**：执行并评估优化效果

**记忆任务**：
用"时间旅行箱"记住以下今天的时间分配情况：
- 起床到上学：1小时
- 学校上课：6小时
- 放学后复习：2小时
- 晚餐休息：1小时
- 作业时间：1.5小时
- 自由时间：2.5小时
- 睡眠准备：0.5小时

**回忆测试**：
快速说出：放学后复习+作业时间=？小时

## Session 3：效率瓶颈识别（2分钟）
### 🎯 注意力追踪游戏 - 瓶颈探测器
**训练目标**：识别自己的学习效率瓶颈

**训练任务**：
根据以下场景，判断哪个环节最可能是你的效率瓶颈：

| 场景 | 可能瓶颈 | 检测方法 |
|------|----------|----------|
| 开始学习时 | 注意力启动慢 | 计时：多久才能专注？ |
| 学习过程中 | 容易分心 | 记录：多久看一次手机？ |
| 做作业时 | 题目理解慢 | 标记：哪类题花时间最长？ |
| 复习时 | 抓不住重点 | 自测：复习后记住了多少？ |

**自我检测**：
回想今天的学习过程，圈出你觉得效率最低的环节：
- A. 开始学习时（很难进入状态）
- B. 学习中途（容易走神）
- C. 遇到难题时（卡住很久）
- D. 复习时（不知道重点）

## Session 4：工具升级学习（3分钟）
### 🧠 记忆力挑战 - 工具升级站
**训练目标**：了解学习工具的升级方法

**工具升级清单**：

1. **笔记工具升级**
   - 原来：随便找张纸写
   - 现在：用康奈尔笔记法（重点-笔记-总结三区域）

2. **记忆工具升级**
   - 原来：死记硬背
   - 现在：用记忆宫殿或联想记忆

3. **计划工具升级**
   - 原来：脑子记要做的事
   - 现在：用手机备忘录+闹钟提醒

4. **复盘工具升级**
   - 原来：想一下就完了
   - 现在：用三问复盘法（今天学了什么？哪里做得好？哪里要改进？）

**记忆任务**：

## Session 5：时间块管理（3分钟）
### 🎯 注意力追踪游戏 - 时间块拼图
**训练目标**：学会将时间整合为高效时间块

**时间块类型**：
| 类型 | 特点 | 适合做什么 |
|------|------|----------|
| 🌟黄金时间块 | 精力最旺 | 数学难题、新知识学习 |
| 🌿碎片时间块 | 时间短但可利用 | 背单词、记公式 |
| ☕缓冲时间块 | 休息调整用 | 喝水、活动、放松 |
| 🌙收尾时间块 | 精力下降 | 整理笔记、复习已学 |

**拼图任务**：
将以下碎片时间整合成时间块：
- 课间10分钟 + 课间10分钟 + 课间5分钟 = ?
- 放学后30分钟 + 晚餐前20分钟 = ?
- 睡前15分钟 + 早起10分钟 = ?

**效率计算**：
如果每天有3个黄金时间块（各30分钟），你能完成多少学习任务？
- 3个黄金时间块 = 90分钟 = 1.5小时高效学习

## Session 6：元认知效率监控（2分钟）
### 🧠 记忆力挑战 - 效率监控表
**训练目标**：学会用元认知监控自己的学习效率

**效率监控三问**：
1. **我现在效率高吗？**
   - 信号：高效率→思路清晰、做题顺利；低效率→脑子卡壳、总想放弃

2. **需要换一种方法吗？**
   - 信号：卡住超过5分钟→换个角度；一直错→回头复习基础

3. **这个任务还需要多少时间？**
   - 信号：预估时间→实际时间对比；超出太多→考虑求助或调整

**记忆任务**：
用顺口溜记住效率监控三问：
"效率高不高？方法要换吗？还要多久啊？"
