## 模块一：注意力训练（2次）

### Session 1：效率认知启动 ⏱️2分钟

**任务名称**：效率扫描仪

**训练说明**：
今天学习怎样优化学习流程、提升效率。首先建立效率意识！

**训练内容**：
```
【什么是学习效率？】

效率 = 有效产出 ÷ 时间投入

高效学习例子：
- 1小时完成3科作业 → 效率高
- 同样的作业做了3小时 → 效率低

效率分析：
❌ 低效表现：边刷手机边写作业
✅ 高效表现：关掉手机，专注做完再休息

训练任务：
判断以下学习方式的效率高低：

A. 先做数学30分钟，再做英语30分钟
B. 数学做5分钟，英语做5分钟，交替进行

答案：____（哪种效率更高？）
```

**小刚练习**：
```
分析自己的学习效率：

今天写作业用了多长时间？
□ 1小时内   □ 1-2小时   □ 2-3小时   □ 3小时以上

你觉得效率高吗？
□ 很高   □ 还可以   □ 有点低   □ 太低了

效率低的原因可能是：
____________
```

## 模块三：学习方法应用（4次）

### Session 5：效率优化四步法 ⏱️2分钟

**任务名称**：流程优化师

**训练说明**：
用四步法优化你的学习流程！

**训练内容**：
```
【效率优化四步法】

第1步 记录
- 用时间日志记录一周的学习时间分配
- 记录每个任务实际用时

第2步 分析
- 找出时间浪费的环节
- 常见浪费：手机干扰、犹豫不决、反复修改

第3步 优化
- 消除干扰源（手机静音、关掉电视）
- 设定时间限制（番茄钟法）
- 批量处理（同类任务一起做）

第4步 验证
- 对比优化前后的效率
- 调整不适用的方法
```

**小刚练习**：
```
用四步法分析一次写作业经历：

第1步 记录：写作业用了多长时间？____分钟
第2步 分析：哪里浪费时间了？____
第3步 优化：下次怎样改进？____
第4步 验证：（明天执行后填写）
```

### Session 6：番茄钟与专注力 ⏱️2分钟

**任务名称**：25分钟专注挑战

**训练说明**：
用番茄工作法提升专注力！

**训练内容**：
```
【番茄工作法详解】

基础规则：
- 设定25分钟闹钟
- 专注完成一项任务
- 闹钟响后休息5分钟
- 每4个番茄钟后休息15-30分钟

进阶技巧：
1. 任务拆分：把大任务分成25分钟内能完成的小任务
2. 记录中断：记录每次中断的原因，减少中断
3. 预估时间：先预估需要几个番茄钟，对比实际

今日挑战：
用番茄钟完成一项作业！

任务：____
开始时间：____
预计番茄钟数：____
完成情况：____
```

**小刚练习**：
```
今天用番茄钟做作业时，遇到的干扰是：
□ 手机消息
□ 有人打扰
□ 走神发呆
□ 其他：____

怎样减少这个干扰？
____
```
