## 模块一：注意力训练（2次）

### Session 1：效率认知启动 ⏱️2分钟

**任务名称**：效率小扫描

**训练说明**：
今天学习怎样让学习更快更好！先来认识什么是效率。

**训练内容**：
```
【什么是效率？】

效率就是：用更少的时间，做更多的事情！

例子：
❌ 慢吞吞：1小时只写了3个字
✅ 效率高：10分钟写了3页字

问题：哪个效率更高？
□ 10分钟做1道题
□ 10分钟做5道题

答案：_____ 做5道题效率更高！
```

**小明练习**：
```
下面哪组学习方式效率更高？

A. 做作业时，一会吃东西、一会玩玩具
B. 做作业时，专心做完再休息

你选：____

答案：B！专心做完效率高！
```

## 模块三：学习方法应用（4次）

### Session 5：效率四步法 ⏱️2分钟

**任务名称**：快快快四步

**训练说明**：
记住这四个步骤，让学习更快！

**训练内容**：
```
【效率快快快四步法】

第1步 记一记（今天要做什么？）
把任务写下来，不忘记

第2步 排一排（先做什么？）
重要的先做，不重要的后做

第3步 做一做（专心完成）
一次只做一件事，不分心

第4步 查一查（做完了吗？）
检查一遍，没做完继续做

小口诀："记、排、做、查，学习快快快！"
```

**小明练习**：
```
小明要做：语文作业、数学作业、收拾书包

第1步 记一记：要做____、____、____

第2步 排一排：先做____，再做____，最后____

你排的顺序：____ → ____ → ____
```

### Session 6：番茄钟学习 ⏱️2分钟

**任务名称**：25分钟挑战

**训练说明**：
用番茄钟，让学习更专注！

**训练内容**：
```
【番茄钟学习法】

什么是番茄钟？
就是设定25分钟，在这段时间里专心学习！

步骤：
1. 设定25分钟闹钟
2. 专心做一项作业
3. 闹钟响了就休息5分钟
4. 再开始下一个25分钟

小明的挑战：
今天用番茄钟做一项作业！
□ 我愿意试试！

约定：25分钟内不能做别的事！
```

**小明练习**：
```
用番茄钟做作业时，闹钟响了要：
□ 继续做
□ 休息5分钟
□ 跑出去玩

答案：____
```
