#### 🔍 任务一：计算问题（难度：⭐⭐）

> 小明去文具店买了3支铅笔，每支2元，又买了1个书包40元。小明付了50元，应该找回多少钱？

**解题策略回顾**（审题-找条件-选方法-验证）：

```
✅ 第一步 - 审题：找出问题问什么
   问题：应该找回多少钱？

✅ 第二步 - 找条件：
   • 铅笔：3支 × 2元 = 6元
   • 书包：40元
   • 付出：50元

✅ 第三步 - 选方法：
   先算总花费，再算找回的钱
   总花费 = 6 + 40 = 46（元）
   找回 = 50 - 46 = 4（元）

✅ 第四步 - 验证：
   46 + 4 = 50 ✓ 计算正确！
```

**练习任务**：

1. 小红有15颗糖，给了小刚5颗，又买了8颗，现在有多少颗？
2. 水果店上午卖了20个苹果，下午卖了15个，还剩30个。原来有多少个苹果？
3. 班级图书角有45本书，借出12本，又还回8本，现在有多少本？

#### 🔍 任务二：图形问题（难度：⭐⭐）

> 一个长方形的长是8厘米，宽比长少3厘米，这个长方形的周长是多少？

**解题策略回顾**：

```
✅ 第一步 - 审题：求什么？
   问题：长方形周长

✅ 第二步 - 找条件：
   • 长 = 8厘米
   • 宽 = 8 - 3 = 5厘米
   • 周长公式：（长+宽）× 2

✅ 第三步 - 选方法：
   周长 = (8 + 5) × 2 = 26（厘米）

✅ 第四步 - 验证：
   8 > 5，宽比长短 ✓
   26是合理答案 ✓
```

**练习任务**：

1. 正方形的边长是6厘米，周长是多少？
2. 一个长方形的长是10厘米，宽是6厘米，周长是多少？
3. 长方形的周长是30厘米，长是9厘米，宽是多少？

#### 🔍 任务三：逆向思维挑战（难度：⭐⭐⭐）

> 小明有一些糖果，给了一半给小红，还剩6颗。原来有多少颗？

**逆向思维技巧**：

```
🎯 正向思考（错误）：不知道总数，无法直接算
🎯 逆向思考（正确）：从结果反推！

思考过程：
• 给出一半后剩6颗
• 那说明另一半也是6颗
• 所以总数 = 6 + 6 = 12颗

验证：12的一半是6，给出6颗后剩6颗 ✓
```

**练习任务**：

### 🏆 训练3-4：错题分析与改进

#### 📋 错题本科学用法

**错误分类**：

| 错误类型 | 表现 | 解决方法 |
|----------|------|----------|
| 粗心型 | 看错数字、算错 | 慢读题、验算 |
| 理解型 | 不理解题意 | 画图理解 |
| 方法型 | 不会解题方法 | 学习例题 |
| 逆向型 | 不会逆向思考 | 多练习反推 |

**错题分析练习**：

```
❌ 错误题目：
"小明有8个球，小红有5个球，小明给小红几个球后，两人就一样多？"
小明的错误答案：给3个

✅ 正确答案与分析：
两人差：8 - 5 = 3（个）
要一样多，只需给一半的差：3 ÷ 2 = 1.5（个）

💡 错因分析：
小明误以为直接给差值，其实应该给差值的一半

✨ 改进方法：
用画图法：先画两人球的数量，从多的一边划一半给少的一边
```

**练习**：分析并改正以下错误

1. "小明有10元，买了3本笔记本，每本2元，还剩多少？" 小明算成10 - 3 = 7元
2. "正方形边长5厘米，周长是多少？" 小刚写成了 5 × 5 = 25厘米

## 🔬 第二部分：实验设计小擂台（4次训练）

### 🧪 训练5-6：趣味实验综合挑战

#### 🏅 挑战一：水杯琴音高实验（Day 2内容回顾）

**实验目标**：通过加水多少改变音高

**实验设计回顾**：

```
📝 实验步骤：
1. 准备5个相同的玻璃杯
2. 第1个杯子不加水，第2个加一点水...第5个加最多水
3. 用筷子敲击，记录音高高低

🔍 观察记录表：
| 杯子 | 水量 | 音高（高/中/低） |
|------|------|------------------|
| 1 | 无 | 高 |
| 2 | 少 | 较高 |
| 3 | 中 | 中 |
| 4 | 较多 | 较低 |
| 5 | 多 | 低 |

💡 原理：水量越多，振动越慢，音高越低
```

**拓展挑战**：如果用不同材质的杯子，音高会一样吗？

---

#### 🏅 挑战二：浮力实验（Day 2内容回顾）

**实验目标**：探索物体沉浮的秘密

**实验设计回顾**：

```
📝 实验准备：
• 大小不同的物体：鸡蛋、橡皮、塑料玩具、石头
• 透明容器、水、盐

📊 实验记录：
| 物体 | 水中状态 | 盐水中状态 |
|------|----------|------------|
| 鸡蛋 | 下沉 | 上浮 |
| 塑料玩具 | 上浮 | 上浮 |
| 石头 | 下沉 | 下沉 |

🔬 科学原理：
• 上浮物体受到的浮力 > 重力
• 下沉物体受到的浮力 < 重力
• 盐水密度大，浮力更大
```

**逆向思维挑战**：
```
❓ 如果物体在盐水中下沉，在普通水中会怎样？
🎯 答案：肯定也会下沉（更轻的盐水浮力更小）

❓ 如果物体在普通水中上浮，在盐水中会怎样？
🎯 答案：会上浮得更高（盐水浮力更大）
```

### 📊 训练7-8：数据记录与误差分析

#### 📋 简单数据记录练习

**测量任务**：用尺子测量5种物品长度

```
📏 测量记录表：
| 物品 | 估计长度 | 实际测量 | 误差 |
|------|----------|----------|------|
| 铅笔 | ___ cm | ___ cm | ___ cm |
| 橡皮 | ___ cm | ___ cm | ___ cm |
| 数学书 | ___ cm | ___ cm | ___ cm |
| 手指 | ___ cm | ___ cm | ___ cm |
| 钥匙 | ___ cm | ___ cm | ___ cm |

💡 误差分析：
• 误差 = |估计值 - 实际值|
• 误差越小，说明估测能力越强！
```

#### 🎯 误差来源大揭秘

**常见误差来源**：

| 误差类型 | 例子 | 减少方法 |
|----------|------|----------|
| 测量工具误差 | 尺子本身不准 | 用多个尺子测量 |
| 读数误差 | 看刻度不准确 | 平视读数 |
| 环境误差 | 温度影响长度 | 在同一环境测量 |

**逆向思考**：如果测量结果和标准值差异很大，可能是什么原因？

```
🎯 从结果反推原因：
误差很大 → 检查：测量方法？工具？读数？环境？
```
