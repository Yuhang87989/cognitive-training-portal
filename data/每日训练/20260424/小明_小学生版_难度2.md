## 🎮 第1次训练：思维追踪挑战

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
观察自己的思考过程，像追踪小动物一样追踪自己的想法。

### 今日挑战

**🌟 思维追踪任务**

闭上眼睛，花10秒钟思考这个问题："如果我有超能力，想有什么能力？"

在思考过程中，注意以下3个信号：

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  🔍 思维放大镜任务单                            │
│                                                 │
│  ☐ 你的大脑里先出现了什么？（画面/声音/感觉）    │
│                                                 │
│  ☐ 你的想法是一下子跳出来的，还是慢慢来的？       │
│                                                 │
│  ☐ 思考时你注意到自己在"想"吗？                │
│                                                 │
│  ⏱️ 用10秒钟，边想边记录                        │
│                                                 │
└─────────────────────────────────────────────────┘
```

**🎯 答案示例**（仅供参考）：

- 我先想到了一只好大好大的鸟，然后想到如果它能带我飞就好了
- 想法是一下子跳出来的
- 我好像在"看着"自己的想法

**💡 小贴士**
这个练习的目的是让你发现：大脑里有很多想法在跑来跑去，而你可以在旁边"观察"它们。

## 🔍 第4次训练：思维放大镜

**模块**：学习方法 - 元认知训练  
**时长**：约2分钟

### 训练目标
学会用"思维放大镜"观察自己做作业时的想法

### 今日学习内容

**🔬 思维放大镜使用指南**

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  🔬 思维放大镜使用步骤                          │
│                                                 │
│  第1步：暂停                                    │
│  → 做作业时，突然停下来                          │
│  → 问自己："我现在在想什么？"                   │
│                                                 │
│  第2步：观察                                    │
│  → 看看脑子里在想什么                           │
│  → 是题目相关，还是在想别的？                    │
│                                                 │
│  第3步：记录                                    │
│  → 简单记下来                                   │
│  → 比如："在想动画片"、"在想怎么算"              │
│                                                 │
│  第4步：调整                                    │
│  → 如果走神了，把注意力拉回来                    │
│  → 如果在想题目，继续加油！                     │
│                                                 │
└─────────────────────────────────────────────────┘
```

**💡 练习时刻**

当你在做这道题时，注意观察自己的思维：

题目：12 + 8 = ?

- 你先想到的是什么？
- 你的脑子是怎么"走"到答案的？
- 你是直接说出答案，还是"算"出来的？
