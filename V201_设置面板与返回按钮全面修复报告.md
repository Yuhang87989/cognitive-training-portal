## 🔧 修复内容明细

### 1. 🚨 严重语法错误修复（ui.js）

**问题**：`switchPlanDay` 函数被错误地打断，window 导出代码错误地插入到函数体中间，导致 JavaScript 执行错误。

**修复前（错误代码结构）**：
```javascript
function switchPlanDay(day) {
    window._planDay = day;
window.closeChangePasswordModal = ...;  // ← 错误地在函数体内
... (很多导出代码)
    const el = document.getElementById('module-content');
    if (el) renderPlan(el);
}
```

**修复后**：
```javascript
function switchPlanDay(day) {
    window._planDay = day;
    const el = document.getElementById('module-content');
    if (el) renderPlan(el);
}

// ============================================================
// 暴露全局函数到 window 对象
// ============================================================
window.closeChangePasswordModal = closeChangePasswordModal;
... (所有导出代码)
```

| `closeUserSwitchModal()` | 关闭用户切换弹窗 |
