## 🎮 第1次训练：学习体系六维稳定性扫描

**模块**：注意力训练  
**时长**：约2.5分钟

### 游戏规则
在45秒内完成学习体系六维度的快速扫描，理解每个维度的核心指标和稳定标准。

### 今日扫描任务

```

│  🎯 学习体系六维稳定性扫描                      │
│                                                 │
│  1️⃣ 注意力稳定性：干扰环境下保持专注时长         │
│     - 标准：≥30分钟不分心                       │
│     - 指标：抗干扰能力、持续专注时间             │

│     - 标准：当天复习后保留率≥85%                │
│     - 指标：记忆速度、保持时长、提取准确率       │
│                                                 │
│  3️⃣ 思维稳定性：复杂问题下的思考连贯性          │
│     - 标准：连续思考≥20分钟不混乱               │
│     - 指标：逻辑清晰度、思维连贯性、问题拆解能力  │
│                                                 │
│  4️⃣ 方法稳定性：不同学科方法运用的一致性        │
│     - 标准：方法运用准确率≥90%                  │
│     - 指标：方法选择、执行质量、迁移能力         │
│                                                 │
│  5️⃣ 元认知稳定性：状态监控和调节的及时性        │
│     - 标准：问题发现到调节完成≤5分钟            │
│     - 指标：问题识别速度、调节策略有效性         │
│                                                 │
│  6️⃣ 执行稳定性：计划完成度和时间管理能力        │
│     - 标准：日计划完成率≥80%                    │
│     - 指标：计划完成率、时间利用率、任务按时率    │
```

## 👀 第3次训练：多场景稳定性对比观察

**模块**：注意力训练  
**时长**：约2.5分钟

### 游戏规则
对比分析不同场景下学习体系的表现差异，识别影响稳定性的关键变量。

### 今日对比观察

**场景1：理想环境**
- 安静书房、书桌整洁、手机静音、精力充沛
- 表现：专注60分钟、记忆效率90%、计划完成100%

**场景2：轻度干扰**
- 客厅学习、家人看电视、偶尔说话、精力一般
- 表现：专注30分钟、记忆效率75%、计划完成80%

**场景3：中度压力**
- 考试前一天、时间紧张、有点焦虑、精力尚可
- 表现：专注45分钟、记忆效率80%、计划完成70%

**场景4：疲劳状态**
- 晚上10点后、很困、注意力涣散、精力不足
- 表现：专注15分钟、记忆效率50%、计划完成40%

**对比分析**：
1. 哪个场景下学习体系最稳定？_____
2. 影响稳定性的两大关键因素是什么？_____和_____
3. 从场景1到场景4，专注时间下降了多少分钟？_____分钟

**答案**：场景1（理想环境）、环境干扰、精力状态；45分钟

### 挑战升级
- 🌟 基础：正确回答2个问题
- ⭐⭐ 进阶：全部答对，分析每个场景的主要影响因素
- ⭐⭐⭐ 大师：为场景2、3、4分别设计具体的稳定性优化方案

### 21天三阶段练习计划

| 阶段 | 天数 | 练习目标 | 具体练习内容 | 验收标准 |
|-----|-----|---------|------------|---------|
| 启动期 | 1-7天 | 建立行为模式 | 固定时间+固定流程+环境准备 | 每天都能完成，不遗漏 |
