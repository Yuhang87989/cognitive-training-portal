## 训练模块分配（8次Session）

### Session 1 - 元认知启动（07:30）
**模块名称**：学习状态仪表盘
**任务类型**：元认知自检
**任务内容**：
快速评估今日学习状态（5秒内完成）：

**状态扫描**：
1. **注意力**：🔴低 / 🟡中 / 🟢高
2. **理解力**：🔴低 / 🟡中 / 🟢高
3. **动力值**：🔴低 / 🟡中 / 🟢高

**自我判断**：
如果有任何🔴或🟡，选择今日的调节策略：
- 注意力不足：深呼吸3次→重新聚焦→设定小目标
- 理解困难：换个角度→求助工具→暂时跳过
- 动力不足：想想目标→奖励自己→改变环境

**时长**：约3分钟

---

### Session 2 - 注意力训练（07:45）
**模块名称**：专注追踪挑战
**任务类型**：深度注意力练习
**任务内容**：
1. 设定10分钟计时器
2. 想象自己在解一道数学题，思维过程用不同颜色表示：
   - 🟢 专注状态：明亮的黄色
   - 🟡 走神状态：灰色
   - 🔴 完全分心：黑色
3. 记录颜色变化次数
4. 分析：什么让我的注意力最容易被分散？

**目标**：10分钟内走神不超过3次

**时长**：约3分钟

---

### Session 3 - 记忆力训练（08:00）
**模块名称**：习惯回路记忆术
**任务类型**：结构化记忆
**任务内容**：
用"习惯回路"框架记忆今日学习计划：

**习惯回路三要素**：
| 环节 | 内容 | 记忆技巧 |
|------|------|----------|
| 提示(Cue) | 学习时间到了 | 想象闹钟响起 |
| 常规(Routine) | 打开→计划→执行→复盘 | 链条记忆法 |
| 奖励(Reward) | 成就感+积分 | 想象完成任务后的笑脸 |

**记忆挑战**：用1分钟背下来，然后默写！

**时长**：约3分钟
