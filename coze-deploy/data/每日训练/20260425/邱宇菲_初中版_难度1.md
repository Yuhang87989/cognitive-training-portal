## Session 1 - 注意力热身

### 🎮 注意力游戏：计划排序

**任务名称**：任务排序挑战

**游戏规则**：
1. 看到一串任务
2. 按重要性和紧急性排序
3. 说明排序理由

**任务清单**：
A. 完成数学作业（明天要交）
B. 复习英语单词（下周考试）
C. 整理错题本（不紧急但重要）
D. 看动画片（娱乐）
E. 预习物理（新内容）
F. 回复朋友消息（社交）

**排序思考**：
- 第一重要：_______
- 第二重要：_______
- 第三重要：_______
- 第四重要：_______
- 第五重要：_______
- 第六重要：_______

**排序原则**：
- 重要又紧急 → 立即做
- 重要不紧急 → 计划做
- 紧急不重要 → 尽快做
- 不重要不紧急 → 空闲做

**时间管理四象限**：
```
        紧急
    ┌─────────┬─────────┐
    │  重要   │ 不重要  │
非  │   A     │   F     │
紧  │ 立即做  │ 尽快做  │
急  ├─────────┼─────────┤
    │  重要   │ 不重要  │
    │   C     │   D     │
    │  计划做 │  空时做  │
    └─────────┴─────────┘
        紧急
```

**训练要点**：学会根据重要性和紧急性排序任务。

## Session 5 - 自主学习：进度评估

### 📊 评估学习进度

**任务名称**：进度检查

**进度评估方法**：

**方法一：对比计划与实际**
- 计划完成率 = 实际完成数 ÷ 计划总数 × 100%

**方法二：任务质量评估**
- 优秀（90%+）：完全掌握
- 良好（80%+）：基本掌握
- 一般（60%+）：部分掌握
- 待改进（<60%）：需要加强

**方法三：时间效率评估**
- 计划用时 vs 实际用时
- 效率 = 任务量 ÷ 实际用时

**进度评估练习**：

**周计划**：
- 总任务数：10项
- 已完成：7项
- 计划完成率：_______%

**任务完成质量**：

- 待改进：0项

**时间效率**：
