## 💧 第2次训练：探索表面张力

**模块**：表面张力实验  
**时长**：约1.5分钟

### 水的表面为什么能撑住回形针

一个有趣的现象：回形针明明比水重，为什么能浮在水面上？

### 实验材料

- 玻璃杯
- 回形针（多个）
- 面巾纸
- 水
- 滴管

### 实验步骤

**实验A：直接放置（观察失败）**
1. 将回形针直接轻轻放在水面上
2. 观察回形针的状态
3. 记录：回形针是否浮在水面上？

**实验B：借助纸片（观察成功）**
1. 将回形针放在一小块面巾纸上
2. 轻轻将纸片放到水面上
3. 等待纸片浸湿下沉
4. 观察回形针的状态
5. 记录：这次回形针浮住了吗？

### 原理分析

**表面张力的本质**
- 水的表面分子受到内部水分子的拉力
- 这种拉力使水的表面像一层"弹性膜"
- 当物体足够轻、接触面积足够大时，表面张力可以支撑物体

**影响因素**
- 水温：温度越高，表面张力越小
- 杂质：洗洁精等表面活性剂会大大降低表面张力
- 接触角度：物体与水的接触角影响浮力

厨房里的小苏打和醋混合会产生什么？

**实验目的**：测量不同配比下CO₂的产量

- 白醋（CH₃COOH）
- 量勺
- 气球
- 塑料瓶

### 实验步骤

**步骤1**：称量不同质量的小苏打（2g、4g、6g）

**步骤2**：将小苏打装入气球

**步骤3**：在瓶中倒入等量白醋（50mL）

**步骤4**：将气球套在瓶口

**步骤5**：将气球中的小苏打倒入醋中

**步骤6**：观察并记录气球膨胀程度

## ⚖️ 第4次训练：密度梯度实验

**模块**：密度梯度实验  
**时长**：约1.5分钟

### 分层液体的奥秘

为什么油会浮在水上？不同液体为什么不会混在一起？

### 实验设计

**实验目的**：观察不同液体的密度差异

**实验材料**
- 透明玻璃杯
- 蜂蜜
- 牛奶
- 水
- 食用油
- 酒精（医用）

### 实验步骤

**步骤1**：按密度从大到小依次缓慢倒入：
1. 蜂蜜（最底层）
2. 牛奶
3. 水（加几滴食用色素便于观察）
4. 食用油
5. 酒精（最上层）

**步骤2**：观察液体分层情况

**步骤3**：轻轻放入一个小物体（如葡萄），观察它在哪一层停住

### 数据记录

| 液体 | 密度（g/mL） | 在哪一层 |
|-----|-------------|---------|
| 蜂蜜 | ~1.4 | 最底层 |
| 牛奶 | ~1.03 | 第2层 |
| 水 | 1.0 | 第3层 |
| 油 | ~0.92 | 第4层 |
| 酒精 | ~0.79 | 最上层 |

### 原理应用

**死海为什么能让人浮起来？**
- 死海盐浓度极高（约为海水的8-10倍）
- 湖水密度高达1.25 g/mL
- 人体密度约1.0 g/mL，小于湖水密度
- 因此人可以浮在水面上

## 🌈 第5次训练：光的折射观察

**模块**：光的折射观察  
**时长**：约1.5分钟

### 为什么筷子"折断"了

把筷子斜着插入水中，会发现筷子好像"折断"了。这是为什么？

### 实验材料

- 透明玻璃杯
- 水
- 筷子（白色以便观察）
- 激光笔（可选）
- 硬币

### 实验步骤

**实验A：筷子折射**
1. 玻璃杯装半杯水
2. 将筷子斜插入水中
3. 从侧面观察筷子的弯曲程度
4. 改变筷子插入角度，记录变化

**实验B：硬币重现**
1. 将硬币放在杯底
2. 向后退直到刚好看不到硬币
3. 保持位置不变，向杯中倒水
4. 观察：能看到硬币了吗？

### 折射定律

**斯涅尔定律**：n₁ sinθ₁ = n₂ sinθ₂

- n₁：光从第一种介质进入第二种介质
- θ₁：入射角（入射光线与法线的夹角）
- θ₂：折射角（折射光线与法线的夹角）

**水的折射率约1.33**

## 🌱 第6次训练：植物蒸腾作用

**模块**：植物蒸腾作用  
**时长**：约1.5分钟

### 植物是怎么"喝水"的

植物的根系吸收水分，但水分是如何到达叶片的？

### 实验设计

**实验目的**：验证植物的蒸腾作用

**实验材料**
- 带叶的枝条（或芹菜）
- 红色食用色素
- 玻璃瓶
- 水
- 小刀

### 实验步骤

**步骤1**：在玻璃瓶中加满水，滴入大量红色色素

**步骤2**：用小刀斜切枝条底部（增加吸收面积）

**步骤3**：将枝条插入色素水中

**步骤4**：放置2-4小时（或过夜）

**步骤5**：观察并记录：
- 叶片颜色变化
- 茎的纵切面颜色
- 变色速度

### 原理分析

**蒸腾作用的过程**
1. 叶片表面的气孔散发水分
2. 产生向上的拉力（蒸腾拉力）
3. 水分子通过毛细作用在导管中上升
4. 根部吸收水分补充

**影响蒸腾作用的因素**
- 光照强度：光照越强，蒸腾越快
- 温度：温度越高，水分子运动越快
- 湿度：湿度越低，蒸腾越快
- 风速：适当的通风促进蒸腾

## 🔌 第7次训练：从开关到并联电路

**模块**：简易电路搭建  
**时长**：约1.5分钟

### 串联与并联的对比

两个灯怎么控制更方便？串联和并联各有什么特点？

### 实验材料

- 电池（2节）
- 电池盒
- 2个小灯泡（带灯座）
- 导线若干
- 开关（2个）

### 实验步骤

**实验A：串联电路**
1. 按下图搭建串联电路
2. 观察：两个灯泡亮度是否相同？
3. 拧松任意一个灯泡，两个灯都灭了吗？

**实验B：并联电路**
1. 按下图搭建并联电路
2. 观察：两个灯泡亮度是否相同？
3. 拧松任意一个灯泡，另一个还亮吗？

### 对比分析

| 特性 | 串联电路 | 并联电路 |
|-----|---------|---------|
| 电流路径 | 只有一条 | 多条路径 |
| 灯泡亮度 | 两个都暗（分电压） | 正常亮度 |
| 一个灯灭 | 两个都灭 | 另一个仍亮 |
| 开关控制 | 一个开关控制全部 | 可以单独控制 |
