## Session 3 - 学科整合：数学→物理

### 📐 数学在物理中的应用

**任务名称**：数学公式的物理意义

**数学公式的物理表达**：

**公式1：v = s/t**
- 数学角度：速度等于路程除以时间
- 物理角度：描述物体运动的快慢
- 单位：m/s（米/秒）

**公式2：ρ = m/V**
- 数学角度：密度等于质量除以体积
- 物理角度：描述物质的紧密程度
- 单位：kg/m³（千克/立方米）

**公式3：F = ma**
- 数学角度：力等于质量乘以加速度
- 物理角度：描述力的大小
- 单位：N（牛顿）

## Session 4 - 学科整合：物理→数学

### 🔬 物理问题中的数学方法

**任务名称**：用数学工具解物理题

**物理问题中的数学方法**：

**方法一：列方程**
物理问题 → 数学方程 → 求解

**示例**：
问题：小明以5m/s的速度走了10秒，走了多远？
- 物理公式：s = v × t
- 数学计算：s = 5 × 10 = 50m

**方法二：画图像**
物理过程 → 数学图像 → 分析

**示例**：
用s-t图（路程-时间图）描述匀速运动：
- 斜率 = 速度
- 纵坐标 = 路程
- 横坐标 = 时间

**方法三：几何分析**
物理图形 → 几何关系 → 求解

**示例**：
力的分解：用平行四边形法则
- 物理：力的分解
- 数学：几何作图

**练习：用数学方法解物理题**

## Session 5 - 学科整合：图像与函数

### 📊 函数图像在物理中的应用

**任务名称**：s-t图和v-t图的秘密

**s-t图（路程-时间图）**：

**匀速运动**：
```
路程
  ↑
  │    ●
  │   ╱ 
  │  ╱  
  │ ╱   
  │╱    
  └────────→ 时间
```
- 特点：直线
- 斜率 = 速度

**静止**：
```
路程
  ↑
  │─────●
  │
  └────────→ 时间
```
- 特点：水平线
- 速度 = 0

**v-t图（速度-时间图）**：

**匀速运动**：
```
速度
  ↑
  │──●──
  │       
  └────────→ 时间
```
- 特点：水平线
- 面积 = 路程

**匀加速运动**：
```
速度
  ↑
  │    ●
  │   ╱
  │  ╱
  │ ╱
  └────────→ 时间
```
- 特点：倾斜直线
- 斜率 = 加速度

## Session 6 - 学科整合：坐标系中的物理

### 📍 坐标系的物理应用

**任务名称**：定位与运动

**一维坐标系**：

**位置表示**：
- 原点O设置
- 正方向规定
- 单位长度确定

示例：规定向右为正方向
- A点在原点：x = 0
- B点在原点右边3米：x = +3m
- C点在原点左边2米：x = -2m

**二维坐标系**：

**平面直角坐标系**：
- x轴：水平方向
- y轴：竖直方向
- 坐标(x,y)：确定平面上的点

**物理应用**：
- 物体的位置：用坐标表示
- 物体的位移：用坐标差表示
- 物体的运动：用坐标变化描述

**练习：坐标表示**
