## 🧠 第2次训练：速度公式记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 核心公式：速度公式三角

```
           速度 v
          /      \
       路程 s   时间 t

公式家族：
┌─────────────────────────────┐
│  v = s ÷ t                 │
│  s = v × t                 │
│  t = s ÷ v                 │
└─────────────────────────────┘
```

### 记忆挑战

**填空练习**
```
1. 速度 = 路程 _____ 时间
2. 路程 = 速度 _____ 时间
3. 时间 = 路程 _____ 速度
```

**单位对应**
```
┌──────────┬──────────┬──────────┐
│   速度   │   路程   │   时间   │
├──────────┼──────────┼──────────┤
│  m/s     │    m     │    s     │
│  km/h    │   km     │    h     │
└──────────┴──────────┴──────────┘

填空：
• 速度单位是 m/s 时，路程单位是_____，时间单位是_____
• 速度单位是 km/h 时，路程单位是_____，时间单位是_____
```

### 💡 记忆技巧
🎯 **三角记忆法**：画一个三角形
- v在顶部，s和t在底部
- 顶部÷底部 = 另一个底部

```
┌─────────────────────────────────────┐

│  长度：1 km = 1000 m                │
│  时间：1 h = 60 min = 3600 s        │
│  速度：1 m/s = 3.6 km/h            │
│  质量：1 kg = 1000 g               │
└─────────────────────────────────────┘
```

### 换算挑战

**填空**
```
1. 2 km = _____ m

2. 3000 g = _____ kg

3. 1 h = _____ s

4. 72 km/h = _____ m/s（提示：÷3.6）

```

**应用计算**
```

问题1：他走路的速度是 1 m/s，需要多少秒？
s = 2000 m，v = 1 m/s
t = s ÷ v = _____ ÷ _____ = _____ s

问题2：他骑自行车的速度是 5 m/s，需要多少秒？
t = _____ ÷ _____ = _____ s

问题3：如果骑车上学用了 8 分钟（480秒），速度是多少？
v = s ÷ t = _____ ÷ _____ = _____ m/s
```
