### 🧠 训练1-2：题型分类与解题方法

#### 📚 专题一：代数基础问题综合

**题型识别要点**：

```
【常见题型分类】

1. 计算题
   特征：给出算式，求值
   方法：按运算顺序计算，注意符号

2. 解方程
   特征：含有未知数x，求x的值
   方法：移项、合并同类项、系数化为1

3. 应用题
   特征：文字描述实际问题
   方法：找等量关系、列方程、解方程

4. 证明题
   特征：求证某个结论
   方法：分析已知与结论的关系
```

**基础综合练习（难度2）**：

---

**问题1**：解方程 `3x + 7 = 22`

**解题步骤**：

```
【第一步】审题
• 方程类型：一元一次方程
• 目标：求x的值

【第二步】移项
3x + 7 = 22
3x = 22 - 7    （移项要变号）

【第三步】合并同类项
3x = 15

【第四步】系数化为1
x = 15 ÷ 3
x = 5

【第五步】检验
左边 = 3×5 + 7 = 22
右边 = 22
22 = 22 ✓ 正确！
```

**练习题**：

1. 解方程：2x - 5 = 13
2. 解方程：4x + 8 = 3x - 2
3. 解方程：5(2x + 1) = 3(x + 4)

---

**问题2**：应用题

> 小刚买了3本笔记本和2支钢笔，共花了46元。已知笔记本每本8元，钢笔每支多少元？

**解题流程**：

```
【审题】
• 已知：笔记本3本×8元，钢笔2支
• 未知：钢笔单价
• 等量关系：笔记本总价 + 钢笔总价 = 46

【设未知数】
设钢笔每支x元

【列方程】
3×8 + 2x = 46
24 + 2x = 46

【解方程】
2x = 46 - 24
2x = 22
x = 11

【答】钢笔每支11元

【检验】
3×8 + 2×11 = 24 + 22 = 46 ✓
```

**练习题**：

1. 小红买了5支铅笔和3块橡皮，共花25元。铅笔每支2元，橡皮每块多少元？
2. 班级买了一批练习本，花了60元。如果每本6元，一共买了多少本？
3. 小刚的体重是35kg，比小红轻5kg。小红的体重是多少kg？

#### 📚 专题二：几何基础问题

**几何证明入门**：

```
【几何问题常见类型】

1. 证明边相等：找对应三角形全等
2. 证明角相等：找对应三角形全等或平行线性质
3. 证明平行：同位角相等、内错角相等
4. 证明垂直：利用直角或角度关系
```

**基础证明练习**：

---

**问题**：如图，AB∥CD，∠1=∠2，求证：∠E=∠F

**分析思路**：

```
【审题】
• 已知：AB∥CD，∠1=∠2
• 目标：证明∠E=∠F

【逆向思考】
要证∠E=∠F
→ 需证什么？∠E和∠F分别与哪些角有关系？
→ ∠E与∠1是内错角，∠F与∠2是对顶角？
→ 需要添加辅助线...

【正向证明】
证明：
∵ AB∥CD（已知）
∴ ∠ABC = ∠BCD（两直线平行，内错角相等）

∵ ∠1 = ∠2（已知）
∴ ∠ABC - ∠1 = ∠BCD - ∠2

即 ∠E = ∠F

∴ ∠E = ∠F（得证）
```

**练习题**：

1. 已知AB⊥BC，BC⊥CD，求证：∠ABC = ∠BCD
2. 三角形ABC中，AB=AC，AD是BC边上的中线，求证：∠BAD = ∠CAD

### 📋 训练3-4：错题分析方法

#### 🔍 错题分析三步法

```
┌─────────────────────────────────────────────────────────┐
│               错题分析三步法                             │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  第一步：找错因                                          │
│  • 审题错误？条件看漏？                                 │
│  • 计算错误？符号错误？                                 │
│  • 方法错误？思路不对？                                 │
│                                                         │
│  第二步：学方法                                          │
│  • 这道题的正确方法是什么？                             │
│  • 有没有更简单的做法？                                 │
│  • 和哪类题型有关？                                     │
│                                                         │
│  第三步：做同类                                          │
│  • 找1-2道同类题练习                                    │
│  • 确保下次遇到同类型不再错                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**错题分析示例**：

```
❌ 错题：解方程 2(x+3) = 4x - 1

小刚的错误解法：
2(x+3) = 4x - 1
2x + 3 = 4x - 1
2x = 4
x = 2

✅ 错误分析：
小刚在去括号时出错！
2(x+3) = 2x + 6（不是2x + 3）

✅ 正确解法：
2(x+3) = 4x - 1
2x + 6 = 4x - 1
2x - 4x = -1 - 6
-2x = -7
x = 3.5

💡 错因总结：
去括号时，括号前面是负数或数字，要每一项都乘
本题中 2×3=6，不是3

✨ 改进练习：
解方程 3(x-2) = 2x + 3
```

**练习**：分析并改正

1. 错题：x² = 4，求x → 小刚答：x=2
2. 错题：计算 3² + 4² → 小刚答：7²=49

```
│  ⑤ 分析数据：整理观察到的现象                           │
│     ↓                                                   │

│                                                         │
└─────────────────────────────────────────────────────────┘
```

**基础实验案例**：

---

**实验：探究"影响水蒸发快慢的因素"**

**① 提出问题**
> 湿衣服为什么有时干得快，有时干得慢？

**② 提出猜想**

```
猜想1：可能与温度有关，温度越高，干得越快
猜想2：可能与空气流动有关，风越大，干得越快
猜想3：可能与接触面积有关，面积越大，干得越快
```

**③ 设计实验（控制变量法）**

```
【实验器材】
• 相同的水杯（或玻璃片）2个
• 相同的水量（10mL）
• 温度计、吹风机、计时器

【实验1：探究温度影响】（控制风速、接触面积不变）
┌─────────────────────────────────────────┐
│ 实验组   │ 条件设置           │ 蒸发时间 │
├─────────────────────────────────────────┤
│ 常温组   │ 室温(20℃)          │ ___分钟  │
│ 加热组   │ 用吹风机加热(40℃)  │ ___分钟  │
└─────────────────────────────────────────┘

【实验2：探究空气流动影响】（控制温度、接触面积不变）
┌─────────────────────────────────────────┐
│ 实验组   │ 条件设置           │ 蒸发时间 │
├─────────────────────────────────────────┤
│ 无风组   │ 静止空气           │ ___分钟  │
│ 有风组   │ 用吹风机吹         │ ___分钟  │
└─────────────────────────────────────────┘
```

**④ 进行实验并记录**

**⑤ 分析数据**

```
结论分析：
• 若加热组比常温组干得快 → 温度越高，蒸发越快 ✓
• 若有风组比无风组干得快 → 空气流动越快，蒸发越快 ✓
```

**⑥ 得出结论**

```
实验结论：
1. 温度越高，水蒸发越快
2. 空气流动越快，水蒸发越快
```

**误差分析**：

```
【可能存在的误差】
• 每次水量不完全相同 → 使用量筒准确量取
• 温度测量时机不同 → 固定测量时间点
• 实验环境有风差异 → 选择无风环境或固定通风条件
```

### 📊 训练7-8：逆向思维基础应用

#### 🔄 逆向思维方法回顾

**逆向思考三步法**：

```
┌─────────────────────────────────────────────────────────┐
│              逆向思考三步法                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  第一步：正向想                                         │
│  已知条件 → 结果                                        │
│                                                         │
│  第二步：倒着想                                         │
│  结果 ← 倒数第二步 ← 第一步                             │
│  （从结论往前推，看看需要什么条件）                      │
│                                                         │
│  第三步：搭桥梁                                         │
│  把正向和逆向的方法连接起来                             │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**逆向思维练习**：

---

**练习1：逆向求原数**

```
问题：小明想了一个数，把这个数乘2再加5，得到21。
     小明想的是什么数？

【逆向思考】
正向：x × 2 + 5 = 21
逆向：21 - 5 = 16 → 16 ÷ 2 = 8

验证：8 × 2 + 5 = 16 + 5 = 21 ✓
答案：8
```

**练习2：逆向推理路程**

```
问题：小刚从家到学校用了20分钟，
     返回时速度是去时的2倍，用了多少分钟？

【逆向思考】
• 去时速度：设为v
• 去时时间：20分钟
• 路程：20v

• 返回时速度：2v
• 返回时间：20v ÷ 2v = 10分钟

答案：10分钟
```

**练习3：实验逆向验证**

```
问题：某同学测得小石块密度为2.5g/cm³，
     但标准值为3g/cm³。如何用逆向思维找原因？

【逆向思考】
结果偏差 → 反推可能原因：
• 质量测量偏小？→ 检查天平
• 体积测量偏大？→ 检查排水法操作
• 计算错误？→ 重新计算ρ=m/V

逐一排查，找出真正原因
```
