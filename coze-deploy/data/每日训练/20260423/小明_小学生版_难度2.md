## 🎮 第1次训练：数字追踪挑战

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
快速扫视屏幕上的数字，找出指定条件的数字并记录数量。

### 今日题目

```
┌─────────────────────────────────────────────────┐
│  🎯 挑战1：找出所有大于5的数                    │
│                                                 │
│     3    8    2    7    4    9    1    6       │
│     5    7    3    6    8    2    9    4       │
│                                                 │
│  大于5的数有 _____ 个                          │
│                                                 │
│  ⏱️ 限时：20秒                                 │
└─────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────┐
│  🎯 挑战2：找出所有偶数                         │
│                                                 │
│     1    4    7    2    9    6    3    8       │
│     5    2    8    4    6    1    7    3       │
│                                                 │
│  偶数有 _____ 个                               │
│                                                 │
│  ⏱️ 限时：20秒                                 │
└─────────────────────────────────────────────────┘
```

**答案**：挑战1 - 大于5的数有8个；挑战2 - 偶数有8个

### 挑战升级

## 🧠 第2次训练：场景画面记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
仔细观察场景画面，记忆其中的数学元素，然后回答问题。

### 今日场景

**场景：操场上的数学**
```
┌─────────────────────────────────────────────────┐
│                                                 │
│    🏃‍♂️→→→→→→→→→→→→→→→→→→→→→→→              │
│         10米    15米    8米                     │
│                                                 │
│    👦站在起跑线，准备跑步！                      │
│                                                 │
│    🏁→→→→→→→→→→→→→→→→→→→→→→→→→→→→           │
│         终点线距离起跑线有多远？                  │
│                                                 │
│    操场边有4棵树，每棵树间距5米                  │
│                                                 │
└─────────────────────────────────────────────────┘
```

**记忆任务**（记忆30秒后回答）：
1. 起跑线和终点线之间有多少米？
2. 操场边有几棵树？
3. 如果每棵树间距5米，4棵树占多长？

### 记忆技巧
💡 **位置记忆法**：把数学信息和位置联系起来
- 起跑线在左边，终点线在右边
- 用"从左到右"的顺序记住距离

## 🔍 第4次训练：数学物理小侦探

**模块**：学习方法 - 学霸专题  
**时长**：约2分钟

### 🎯 今日技能：跨学科问题分析

数学和物理是好朋友！很多数学知识可以用来解决物理问题，很多物理现象可以用数学来描述。

### 侦探任务

**任务1：小汽车的速度问题**
```
🚗 小汽车从起点出发：
   - 第1秒走了3米
   - 第2秒走了3米
   - 第3秒走了3米

🔍 数学问题：
   - 每秒走的距离一样吗？
   - 3秒一共走了多少米？
   - 这叫什么运动？
```

**任务2：苹果落地的规律**
```
🍎 苹果从树上掉落：
   - 高度：9米
   - 下落时间：3秒

🔍 数学问题：
   - 如果平均分配，每秒下落多少米？
   - 用除法怎么表示？
```

### 学霸笔记
📖 **跨学科思维记录**：

我发现：
- 速度可以用 ____ ÷ ____ = ____ 来计算
- 时间和距离的关系是：距离 = ____ × ____
