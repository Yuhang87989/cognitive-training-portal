## 🎯 第1次训练：什么是元认知

**模块**：元认知基础  
**时长**：约2分钟

### 核心概念
元认知（Metacognition）就是"对自己思考的思考"。它包括三个方面：
- **元认知知识**：了解自己的学习特点和习惯
- **元认知体验**：觉察自己当下的学习感受
- **元认知监控**：主动调节自己的学习状态

### 训练任务
**任务A：理解元认知**

阅读以下场景，判断属于哪种元认知：

场景1：小明发现自己早上记忆英语单词效果最好，所以把英语背诵安排在早晨。
→ 这属于：□ 元认知知识 □ 元认知体验 □ 元认知监控

场景2：上课时，小红突然意识到自己走神了，于是把注意力拉回来。
→ 这属于：□ 元认知知识 □ 元认知体验 □ 元认知监控

场景3：考试时，小刚感觉很紧张，他深呼吸让自己平静下来。
→ 这属于：□ 元认知知识 □ 元认知体验 □ 元认知监控

**任务B：反思练习**
回想最近一次学习经历，思考：
1. 你当时的状态如何？（专心/走神）
2. 你是否意识到自己的状态？
3. 你做了什么调整？

### 答案参考
**任务A**：
- 场景1：元认知知识（了解自己的学习规律）
- 场景2：元认知监控（主动调节注意力）
- 场景3：元认知体验+监控（感受紧张并调节）

### 积分奖励：10分

## 👀 第3次训练：专注力诊断

**模块**：注意力训练  
**时长**：约2分钟

### 核心概念
专注力是学习效率的关键。常见的专注力问题包括：
- 注意力的稳定性（能坚持多久）
- 注意力的广度（同时关注多少信息）
- 注意力的分配（能否同时做多件事）

### 训练任务
**任务A：注意力稳定性测试**

阅读以下短文，回答问题（计时）：

```
在学习过程中，注意力的分配和转移非常重要。
研究表明，青少年的注意力持续时间通常在15-25分钟之间。
当我们在学习时遇到困难，往往容易走神；
而当我们对学习内容感兴趣时，注意力更容易集中。
因此，了解自己的注意力特点，
并据此安排学习计划，是提高学习效率的有效方法。
```

**问题**（在3分钟内完成）：
1. 青少年的注意力持续时间是多少？____
2. "分配"在文章中出现几次？____
3. 注意力集中的时候有什么特点？____
4. 提高学习效率的方法是什么？____

**任务B：自我评估**
- 你的阅读速度：□ 快 □ 中等 □ 慢
- 你是否需要重读某些句子？□ 是 □ 否
- 你的正确率：□ 100% □ 80%以上 □ 60%以上 □ 更低

### 答案参考
1. 15-25分钟
2. 2次
3. 对学习内容感兴趣时
4. 了解自己的注意力特点，并据此安排学习计划

### 积分奖励：10分

## 🔍 第4次训练：思维过程观察

**模块**：思维监控  
**时长**：约2分钟

### 核心概念
元认知的核心是观察和监控自己的思维过程。

### 训练任务
**任务A：思维过程记录**

当你面对一道数学难题时，你的思维过程是怎样的？

**示例思维过程**：
1. 阅读题目 → 理解题意
2. 分析已知条件 → 提取有用信息
3. 尝试方法1 → 行不通
4. 换方法2 → 有思路
5. 继续推导 → 得出答案
6. 检验答案 → 确认正确

**请填写你的思维过程**：
1. ____ → ____
2. ____ → ____
3. ____ → ____
4. ____ → ____

**任务B：思维障碍识别**

阅读以下问题，记录你的思维过程：

问题：小明从家走到学校，每分钟走60米，走了15分钟。学校离家多远？如果提前5分钟到，需要每分钟走多少米？

**思维记录**：
- 第一步：____
- 遇到困难：____
- 如何解决：____
- 最终答案：____

### 答案参考
**任务B**：
- 第一步：计算总路程（60×15=900米）
- 遇到困难：不知道现在用多少时间
- 解决思路：15分钟到学校，现在走15分钟，提前5分钟就是15-5=10分钟
- 答案：900÷10=90米/分钟
