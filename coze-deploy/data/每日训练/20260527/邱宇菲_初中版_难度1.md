## 🎭 第1次训练：我现在是什么状态

**模块**：自我觉察  
**时长**：约2分钟

### 游戏规则
闭上眼睛，深呼吸三次，然后想一想自己现在是什么样的状态？

### 今日题目

**问题1：身体感觉**
你现在感觉：
- 精神饱满（眼睛亮亮的、有力气）✓?
- 状态一般（不太精神也不累）✓?
- 有点疲惫（想休息）✓?
- 很累（需要马上休息）✓?

**问题2：心情感觉**
你现在心情：
- 很开心（想笑）✓?
- 还不错（平静）✓?
- 一般（没什么特别感觉）✓?
- 不太好（想安静一会儿）✓?

**问题3：专注程度**
你回想一下最近一次学习：
- 很专心（眼睛一直跟着老师/书本）✓?
- 比较专心（偶尔会走神）✓?
- 有点走神（需要提醒自己）✓?
- 经常走神（脑子里想别的事）✓?

**问题4：学习动力**
你现在的学习动力：
- 很强（很想学习新东西）✓?
- 还可以（按部就班学）✓?
- 一般（学不学都行）✓?
- 不想学（学习让我烦）✓?

### 思考问题
如果发现自己状态不够好，有什么简单的办法改善？

**参考答案**：
1. 深呼吸3-5次
2. 站起来活动一下
3. 喝点水
4. 休息5分钟
5. 换个科目学习

## 👀 第2次训练：专注力小测试

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
仔细阅读下面的短文，数一数指定字词出现的次数！

### 今日题目

```
阅读短文：
今天数学课上，我认真听老师讲加法的计算方法，
老师讲得很好，我学会了用加法解决简单的问题。
上数学课的时候，我发现数学很有意思，
所以我认真做了数学练习题。
```

**问题**：
1. "我"字出现了几次？____
2. "数学"出现了几次？____
3. "上"字出现了几次？____
4. "认真"出现了几次？____
5. "今天"和"上课"分别出现在第几句？

### 答案
1. 5次
2. 3次
3. 2次
4. 2次
5. "今天"在第1句，"上课"在第2句

### 专注小提示
- 阅读时用手指点着字
- 眼睛跟着手指移动
- 数过的字可以做个小记号

## 🧠 第4次训练：我记得牢吗

**模块**：记忆觉察  
**时长**：约2分钟

### 游戏规则
测试一下自己的短时记忆力！

### 今日题目

**记忆任务**（看40秒，然后盖住）：

词语：苹果、香蕉、铅笔、老师、太阳、学习、学校、朋友、妈妈、快乐、风景、运动、思考、安静、努力

**问题**：
1. 第5个词是什么？____
2. "朋友"是第几个？____
3. 哪个词和"老师"是同一类的？____
4. "妈妈"后面连续两个词是什么？____
5. 一共有几个"物"字开头的词？____

### 答案
1. 太阳
2. 第8个
3. 铅笔（都是学校相关）
4. 快乐、风景
5. 3个（苹果、香蕉、铅笔）

### 记忆小技巧
- 分类记忆法：水果、文具、人、抽象概念

- 位置记忆：按顺序记住每个位置
