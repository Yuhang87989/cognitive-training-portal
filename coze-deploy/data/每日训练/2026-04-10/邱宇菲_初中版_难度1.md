**第二步：正向思维 vs 逆向思维**
正向思维（顺向）：
条件A → 条件B → 结果C
从"因为"到"所以"

结果C → 条件B → 条件A
从"所以"到"因为"

什么情况适合逆向思维？
- 知道结果，需要反推条件

## Session 3 - 数学训练：逆向推理基础

### 🔄 逆向推理三步法

**任务名称**：从结果追溯原因

**逆向推理三步法**：

**第一步：明确目标（结果）**
"我想要得到什么？"或"答案是多少？"

**第二步：追溯上一步**
"要得到这个结果，上一步需要什么？"

**第三步：重复追溯**
继续向前追溯，直到找到起始条件

**练习：逆向计算**
目标：算出x的值
x + 5 = 12

逆向思考：
- 目标：x = _______
- 上一步：12 - 5 = x → x = 7
- 验证：7 + 5 = 12 ✓

**练习：逆向推理链**
目标：到达学校（8:00到校）
- 8:00到校 → 需要7:50出门
- 需要7:50出门 → 需要7:40起床
- 需要7:40起床 → 需要前一晚22:00睡觉

**训练要点**：掌握逆向推理的基本步骤。

## Session 4 - 数学训练：反证法入门

### ⚖️ 反证法初步 - 假设与反驳

**任务名称**：证明"不可能"的智慧

**反证法原理**：
反证法是一种间接证明方法：
1. 先假设结论不成立
2. 从假设出发进行推理
3. 推出矛盾或错误
4. 从而证明原结论成立

**反证法示例**：
证明：没有最大的自然数

证明过程：
1. 假设：存在最大的自然数N
2. 推理：那么N+1也是自然数，且N+1 > N
3. 矛盾：与"最大"矛盾
4. 结论：没有最大的自然数

**任务名称**：逆向思维解题技巧

**应用场景一：逆向工程**

示例：

逆向思考：什么数的平方是16？

示例：
