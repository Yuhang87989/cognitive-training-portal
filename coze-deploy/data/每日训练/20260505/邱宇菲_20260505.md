## Session 3：学霸方法 - 预防性调节三步法

### 训练目标
掌握"预判→预防→预备"的完整预防策略

### 训练内容
**预防性调节三步法**

这是从"事后补救"升级到"事前预防"的核心方法！

#### 第一步：预判
**问题预测**：在学习开始前，先想想"可能会出什么问题？"

| 学习情境 | 常见问题预判 |
|----------|--------------|
| 语文阅读 | 可能遇到不理解的词语、跑神 |
| 数学解题 | 可能遇到难题卡住、时间不够 |
| 英语背诵 | 可能记不住、容易分心 |
| 物理学习 | 可能概念混淆、公式记错 |

#### 第二步：预防
**提前行动**：针对预判的问题，提前做好预防措施

| 问题预判 | 预防措施 |
|----------|----------|
| 可能跑神 | 准备一杯水放在旁边，用喝水拉回注意力 |
| 可能卡住 | 先复习相关基础知识，准备好草稿纸 |
| 可能分心 | 提前把手机放到另一个房间 |
| 可能疲惫 | 设定30分钟闹钟，到点必须休息5分钟 |

#### 第三步：预备
**备用方案**：准备Plan B，以防预防措施失效

| 情境 | Plan A（预防） | Plan B（备用） |
|------|----------------|----------------|
| 注意力涣散 | 喝口水调整 | 站起来活动30秒 |
| 题目太难 | 先做简单题热身 | 跳过，先做下一题 |
| 时间紧张 | 提高速度 | 放弃难题，保基础分 |
| 情绪焦虑 | 深呼吸3次 | 写下焦虑原因再继续 |

## Session 4：注意力训练 - 学习状态预测表

### 训练目标
学会根据日期、时段、任务类型预测学习状态

### 训练内容
**学习状态预测表**

预测自己的学习状态，就像看天气预报一样！

#### 因素分析表

| 因素 | 选项 | 对注意力的影响 |
|------|------|----------------|
| **时段** | 早上（7-9点） | 🟢 高效期 |
| | 上午（9-12点） | 🟢 较好 |
| | 下午（2-5点） | 🟡 容易困 |
| | 晚上（6-9点） | 🟢 黄金期 |
| | 睡前（9点后） | 🔴 效率低 |
| **任务难度** | 简单（抄写、背诵） | 🟢 容易坚持 |
| | 中等（练习题） | 🟡 需要意志力 |
| | 困难（难题、新知识） | 🔴 容易放弃 |
| **任务长度** | 短（15分钟内） | 🟢 轻松完成 |
| | 中等（30分钟） | 🟡 需要规划 |
| | 长（1小时以上） | 🔴 需要拆分 |
| **前一天睡眠** | 充足（8小时+） | 🟢 精力充沛 |
| | 一般（6-7小时） | 🟡 有些疲惫 |
| | 不足（<6小时） | 🔴 难以专注 |

#### 预测公式
**状态预测 = 时段 + 任务难度 + 任务长度 + 睡眠质量**
