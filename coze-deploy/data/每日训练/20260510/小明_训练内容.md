## Session 1：我的成长故事书 - Week 8精彩回顾

**训练类型**：注意力训练 + 记忆力激活

**难度2 基础训练内容**：

### 🎯 任务一：成长相册回忆（注意力深度聚焦训练）
闭上眼睛，想象这一周的学习生活像一本相册，一页一页翻开：
- 周一：我记得那天学了____，感觉____
- 周二：那天我尝试了"预测问题"的方法，结果____
- 周三：做真题练习时，我学会了____
- 周四：我发现了一个让学习变快的方法：____
- 周五：我整理了自己的学习工具，最喜欢的是____
- 周六：我开始设计自己的学习系统，里面有____

**注意力训练**：每回忆一天，保持专注20秒，尽量想起细节

### 🎯 任务二：进步星星收集
这一周你收获了哪些进步星星？（至少收集3颗）
⭐ 注意力星星：我能专注更久了，从____分钟到____分钟
⭐ 记忆力星星：我记住了更多东西，比如____
⭐ 方法星星：我学会了____新方法
⭐ 自信星星：我现在更敢____了
⭐ 其他星星：____

### ⏱️ 训练时长：约2分钟
### 💡 难度2提示：
- 如果记不清具体内容，可以画一个笑脸代表那天很开心
- 每颗星星都值得骄傲，哪怕只是很小的进步

## Session 2：学习红绿灯 - 我的复习准备评估

**训练类型**：记忆力训练 + 学习方法

**难度2 基础训练内容**：

### 🎯 任务一：科目红绿灯评估（记忆检索训练）
用"红绿灯游戏"来看看你各科的准备情况：
- 🟢 绿灯（准备好啦）：这科我已经知道怎么复习了
- 🟡 黄灯（还需要准备）：我大概知道要学什么，但还需要帮助
- 🔴 红灯（需要加油）：我还不知道从哪里开始

| 科目 | 红绿灯 | 我的一句话 |
|------|--------|------------|
| 语文 |  | 我觉得____ |
| 数学 |  | 我觉得____ |
| 英语 |  | 我觉得____ |
| 科学 |  | 我觉得____ |

### 🎯 任务二：复习小计划设计
选一个你最想复习的科目，设计一个简单的复习小计划：
- 科目：____
- 我想先复习：____（比如：生字词、乘法口诀）
- 复习时间：____分钟
- 和谁一起复习：____（自己/爸爸/妈妈/同学）

### ⏱️ 训练时长：约2分钟
### 💡 难度2提示：
- 选择你最感兴趣的科目开始，会更有动力
- 复习时间不要太长，10-15分钟就很好

## Session 4：我的超能力清单 - Week 1-8成长盘点

**训练类型**：记忆力训练 + 自信建立

**难度2 基础训练内容**：

### 🎯 任务一：学习超能力大检阅（记忆巩固训练）

- [ ] 📚 方法超能力：我学会了好的学习方法
- [ ] 🔮 预测超能力：我能提前发现学习中的问题

## Session 5：我的小雷达 - 学习问题预警系统

**训练类型**：元认知训练 + 问题解决

**难度2 基础训练内容**：

### 🎯 任务一：预警雷达发现（元认知监控训练）
你的"学习小雷达"发现了哪些信号？
- 🔔 走神预警：当我____的时候，就是要走神了
- 🔔 不懂预警：当老师讲____的时候，我可能会听不懂
- 🔔 累了预警：当我____的时候，就想休息了
- 🔔 着急预警：当____的时候，我就会很着急

### 🎯 任务二：应对小妙招
针对你发现的一个预警信号，想一个应对的小妙招：
- 预警信号：____
- 应对妙招：____（比如：感觉要走神时，先伸个懒腰）
- 这一周用了几次：____次
- 效果怎么样：😊😐😕

### ⏱️ 训练时长：约2分钟
### 💡 难度2提示：
- 能发现信号就已经很厉害了，妙招可以慢慢想
- 和爸爸妈妈一起 brainstorm（头脑风暴）会更有趣

## Session 7：我的小目标 - Week 9 期待清单

**训练类型**：目标设定 + 规划能力

**难度2 基础训练内容**：
