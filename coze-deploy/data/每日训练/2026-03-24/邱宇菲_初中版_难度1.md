**概念图原理**：
```

```

**练习2：为"分数"构建概念图**

分支3（运算）：加法/减法/乘法/除法

## Session 4 - 深度理解策略

### 📚 深度理解：从知道到理解

**任务名称**：学霸的深度理解法

**超能力介绍**：
真正的理解不是背下来，而是能用不同方式表达、知道怎么用、能举例子！

**深度理解四层次**：
```
层次1：能说出来（复述）
层次2：能写出来（解释）
层次3：能举例（应用）
层次4：能教别人（传授）
```

**练习内容**：

**练习1：多方式表达**
选择一个数学概念（如"分数"），用不同方式表达：

- 用符号表示：1/2
- 用文字解释：把一个整体平均分成2份，取其中1份
- 用图形表示：画一个圆，涂一半
- 用生活例子：半个苹果

**练习2：追问法深入理解**
对于"光合作用"，不断追问：
- 是什么？（植物制造食物的过程）
- 为什么？（因为植物需要能量）
- 怎么做？（用阳光、水、二氧化碳）
- 用在哪里？（植物生长、氧气来源）

**练习3：知识追问练习**
对于"等腰三角形"，问自己这四个问题：
- 是什么：_____________
- 为什么：_____________
- 怎么做：_____________
- 用在哪里：_____________

**深度理解技巧**：
- 不满足于"知道"，要追问"为什么"
- 找生活中的例子
- 试着用自己的话解释
