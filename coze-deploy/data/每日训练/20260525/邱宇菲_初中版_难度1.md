## 🧠 第2次训练：五大能力维度记忆

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
记忆10周训练形成的五大核心能力维度，理解每个维度的具体含义。

### 今日记忆内容

```
🧠 我的五大核心学习能力：

1. 注意力能力
   表现：专注时间延长、抗干扰能力增强、
         注意力切换更灵活

2. 记忆力能力
   表现：记忆速度更快、保持时间更长、
         提取更准确

3. 思维力能力
   表现：思路更清晰、逻辑更严密、
         解决问题更有方法

4. 学习方法能力
   表现：有系统的学习方法、知道如何高效学习、
         能根据不同内容调整方法

5. 元认知能力
   表现：能监控自己的学习状态、
         能及时调整学习策略、
         能自主规划学习
```

### 记忆挑战（难度1）
1. 五大能力中，第一个能力是什么？
2. 能监控自己学习状态的是什么能力？

**记忆技巧**：把这五个能力想象成你学习时随身携带的"五件法宝"，每件都有特殊功能。
