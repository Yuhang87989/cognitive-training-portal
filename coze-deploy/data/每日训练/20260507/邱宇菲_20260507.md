## 模块一：注意力训练（2次）

### Session 1：审题专注力训练 ⏱️2分钟

**任务名称**：关键词猎人

**训练说明**：
做数学题时，第一步不是计算，而是找到题目的"眼睛"——关键词。今天训练你快速识别关键词的能力。

**训练内容**：
```
题目：用方程解应用题
"一个书包的原价是x元，打8折后是64元，求原价。"

找出这句话中的关键词：
□ 一个书包    □ 原价是x元    □ 打8折后    □ 64元
□ 求原价

正确关键词：原价是x元、打8折后、64元
```

**判断标准**：
- 找到全部3个关键词 = ★★★
- 找到2个关键词 = ★★
- 找到1个关键词 = ★

**邱宇菲挑战**：
```
题目：小明从家走到学校，速度是5千米/小时，走了0.8小时，
求小明家到学校的距离。

这道题的关键词是：
① 速度   ② 0.8小时   ③ 求距离   ④ 小明从家走到学校

你选：_______
```

### Session 2：解题过程专注训练 ⏱️2分钟

**任务名称**：步骤追踪

**训练说明**：
解题就像走迷宫，每一步都要清楚。今天训练你在解题过程中保持专注，不跳步。

**训练内容**：
```
例题：解方程 2x + 5 = 15

你的解题步骤：
步骤1：2x + 5 - 5 = 15 - 5    ← 先做什么？
步骤2：2x = 10                 ← 得到什么？
步骤3：x = 10 ÷ 2             ← 最后做什么？
步骤4：x = 5                  ← 答案是什么？

问题：上面步骤1中，"先做什么"应该填什么？
□ 加5      □ 减5      □ 乘2      □ 除2
```

**邱宇菲练习**：
```
计算：35 + 17 × 2 - 8 ÷ 4

步骤1：先算乘法        → 17 × 2 = ___
步骤2：再算除法        → 8 ÷ 4 = ___
步骤3：然后算加法      → 35 + ___ = ___
步骤4：最后算减法      → ___ - ___ = ___

你的最终答案：_______
```

## 模块二：记忆力训练（2次）

### Session 3：审题要点记忆 ⏱️2分钟

**任务名称**：三秒记忆法

**训练说明**：
考试时没时间反复看题，要在最短时间内记住关键信息。今天训练你的"三秒记忆"能力。

**训练内容**：
```
现在有10秒钟记忆这道题：

"某商店销售一种商品，进价为40元/件，
售价为60元/件，每天可卖出100件。
如果每件涨价1元，每天销量减少10件。
问：每天定价多少元时，利润最大？"

闭眼回忆，在心里说出：
1. 商品的进价：____元
2. 原来的售价：____元
3. 原来每天销量：____件
4. 涨价1元后销量减少：____件
5. 要解决的问题：________________

记忆完成后再睁开眼睛！
```

**记忆小技巧**：
- 把数字和生活联系起来（比如60元 = 6张10元）
- 用"进、原、涨、减、利润"五个字记住关键量

### Session 4：解题思路记忆 ⏱️2分钟

**任务名称**：思路归档

**训练说明**：
每做完一道题，要把解题思路"归档"保存，下次遇到类似的题就能快速调用。

```

④ 解方程（按步骤计算）

⑥ 写答句（完整回答问题）

现在记忆这道"配套问题"的特征：
```

## 模块三：学习方法应用（4次）

### Session 5：真题思维四步法 ⏱️2分钟

**任务名称**：审题→破题→解题→验证

**训练说明**：
建立完整的解题思维流程，每道题都按这个流程走。

**训练内容**：
```
【真题思维四步法】

第一步 审题（找什么？）
- 题目问什么？_____________
- 有哪些已知条件？_____________
- 有没有隐藏条件？_____________

第二步 破题（用什么？）
- 这道题考什么知识点？
- 需要用到什么公式或方法？
- 和之前做过的哪道题类似？

第三步 解题（怎么算？）
- 写出解题步骤
- 每一步都要写清楚

第四步 验证（对不对？）
- 答案合理吗？
- 能代入原题检验吗？
```

**邱宇菲练习**：
```
用四步法解这道题：

"小明买了3本笔记本，每本x元，又买了1支钢笔花了12元，
一共花了45元。求笔记本的单价。"

第一步审题：问____________，已知____________

第二步破题：考的是____________方程

第三步解题：
列式：____________
解：____________

第四步验证：____________

笔记本单价是______元
```

### Session 6：数学物理综合思维 ⏱️2分钟

**任务名称**：数物联动

**训练说明**：
数学是物理的语言，物理是数学的应用。学会用数学解决物理问题。

**训练内容**：
```
【数学物理综合训练】

物理问题：一个小球从5米高处自由落下，
忽略空气阻力，求落地时的速度。

数学解决步骤：
1. 物理公式：v² = 2gh
   （速度的平方 = 2 × 重力加速度 × 高度）

2. 代入数据：v² = 2 × 10 × 5

3. 数学计算：v² = 100

4. 求速度：v = √100 = 10 m/s

关键点：物理列式 → 数学计算 → 物理解释
```

**邱宇菲挑战**：
```
物理问题：一辆汽车以20 m/s的速度行驶，
司机看到前方有障碍物，反应0.5秒后开始刹车，
刹车加速度大小为8 m/s²，求刹车距离。

数学部分：
反应距离 = 速度 × 反应时间 = 20 × 0.5 = ____ m

刹车过程：v² = v₀² - 2as
0² = 20² - 2 × 8 × s
0 = 400 - 16s
s = ____ m

总刹车距离 = ____ + ____ = ____ m
```

### Session 7：难题攻略 ⏱️2分钟

**任务名称**：难题四步走

**训练说明**：
遇到不会的题不要慌，按这个顺序尝试。

**训练内容**：
```
【难题攻略】

遇到不会的题，按这个顺序尝试：

第1步 跳过（1分钟）
先做会做的题，难题留到最后
不要在一道题上卡太久

第2步 回头看（2分钟）
- 再读一遍题目
- 标记关键词
- 回忆相关知识点

第3步 联想（2分钟）
- 这道题像之前做过的哪道题？
- 能否把大问题拆成小问题？
- 有没有类似的公式可以用？

第4步 求助（标记）
- 问老师或同学
- 查看参考答案
- 弄懂后一定要自己再算一遍
```

**邱宇菲自测**：
```
一道数学题看了3分钟还是不会做，你怎么做？
□ 继续死磕，直到想出来
□ 先跳过，最后再回来做
□ 直接看答案
□ 问同学或老师

正确做法：先跳过，最后再回来

如果最后还是不会，应该：
□ 算了，不做了
□ 看答案，然后自己再做一遍
□ 问老师，但不用搞懂

正确做法：看答案弄懂后，自己再算一遍
```

### Session 8：易错题防范 ⏱️2分钟

**任务名称**：避坑训练

**训练说明**：
很多同学不是不会做，而是掉进了"陷阱"。今天训练你识别陷阱的能力。

**训练内容**：
```
【常见易错陷阱】

陷阱1：单位不同
例：0.5小时 = ____分钟
易错：直接写0.5
正确：0.5 × 60 = 30分钟

陷阱2：运算顺序
例：3 + 5 × 2 = ____
易错：(3+5) × 2 = 16
正确：3 + (5×2) = 13

陷阱3：丢符号
例：-3 + 5 = ____
易错：-3+5=-8（错）
正确：-3+5=2

陷阱4：审题不完整
例："增加了"vs"增加到"
易错：混淆两者的区别
正确：增加 = 增量；增加到 = 最终值
```

**邱宇菲挑战**：
```
找出下面每道题的"陷阱"，说出正确答案：

1. 2.5小时 = ____分钟
   陷阱：____________
   答案：____________

2. 20 - 4 × 3 = ____
   陷阱：____________
   答案：____________

3. 某商品先涨价20%，再降价20%，现价是原价的____%
   陷阱：____________
   答案：____________
```
