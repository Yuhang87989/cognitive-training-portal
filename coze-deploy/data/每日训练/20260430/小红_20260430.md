### 【Session 2】迁移四步法 · 识别-回忆-调整-应用

**训练名称**：迁移四步法训练

**训练内容**：
**迁移四步法详解**

| 步骤 | 问题 | 行动 |
|------|------|------|
| 1. 识别 | "这个问题像训练中的哪个类型？" | 找到相似性 |
| 2. 回忆 | "我当时是怎么解决的？" | 调取记忆 |
| 3. 调整 | "这次有什么不同？需要怎么调整？" | 适应性修改 |
| 4. 应用 | "执行并验证效果" | 落地实施 |

**案例演练**：

**场景**：数学作业 - 一元二次方程应用题

**迁移分析**：

| 步骤 | 分析内容 |
|------|---------|
| **识别** | 这道题和训练中的"问题建模"类型相似 |
| **回忆** | 训练中学习了：提取关键信息→建立数学关系→求解验证 |
| **调整** | 应用题需要先建立方程，训练中是直接计算 |
| **应用** | 按步骤执行：设未知数→找等量关系→列方程→求解 |

**练习题**：

用迁移四步法分析英语阅读理解题：

| 步骤 | 分析内容 |
|------|---------|
| **识别** | 这道题像训练中的"____"类型 |
| **回忆** | 当时用的方法是：____ |
| **调整** | 这次的不同是：____ |
| **应用** | 具体的解题步骤：____ |

**✅ 完成标准**：完成迁移四步法案例分析

### 【Session 3】数学迁移实战 · 思维方法应用

**训练名称**：数学思维迁移

**训练内容**：
**训练方法 → 数学学习**

**迁移映射**：

| 训练方法 | 迁移到 | 数学应用 |
|---------|-------|---------|
| 逻辑推理 | → | 数学证明 |
| 模型构建 | → | 应用题建模 |
| 逆向思维 | → | 反证法、逆向推理 |
| 归纳总结 | → | 公式推导、规律发现 |

**实战演练**：

**题目**：某商店销售一种商品，进价40元，售价60元。现在搞促销活动，买5件送1件。如果某公司要买30件，至少需要多少钱？

**迁移分析**：

| 维度 | 分析 |
|------|------|
| **训练来源** | 训练中的"经济问题建模" |
| **关键信息提取** | 进价40、售价60、买5送1、买30件 |
| **模型构建** | 买5送1 = 付5件的钱得到6件 |
| **计算策略** | 30 ÷ 6 = 5组（每组付5件的钱） |
| **求解** | 5 × 5 × 60 = 1500元 |

**迁移总结**：
- 从训练中迁移了"分组策略"
- 关键是从实际问题中抽象出"6件一组"的数学模型

**✅ 完成标准**：完成数学迁移案例分析

**训练名称**：错题元认知迁移

**训练内容**：
**训练方法 → 错题分析**

**迁移框架**：

| 训练方法 | 迁移到 | 错题分析 |
|---------|-------|---------|
| 逆向思维 | → | 错误原因追溯 |

| 元认知监控 | → | 思维过程复盘 |
| 复盘方法 | → | 错题分类总结 |

**错题分析四步法**：

**Step 1：问题识别**
- 这道题我哪里错了？
- 具体是什么类型的错误？（计算/概念/审题/方法）

**Step 2：原因追溯**（5Why分析）
```
为什么错？（第1层）
→ 为什么会有这个原因？（第2层）
→ ...（继续追问5层）
→ 根本原因是什么？
```

**Step 3：思维复盘**
- 当时我是怎么想的？
- 正确的思路是什么？
- 我的思路和正确思路差在哪里？

**Step 4：改进措施**
- 短期措施：____
- 长期措施：____
- 同类题训练：____
