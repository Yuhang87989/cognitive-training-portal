## 🧠 元认知三步法（今日重点）

### ① 快速观察（5秒内完成）
学习时快速"扫描"自己的状态：
- 注意力：专注还是走神？
- 理解：懂了还是迷糊？
- 动力：有劲还是没力？

### ② 即时判断
状态好不好？
- ✅ 好 → 继续保持
- ⚠️ 一般 → 小调整
- ❌ 差 → 必须调整

### ③ 快速调节
选个方法试试：
- 注意力不集中：深呼吸 → 重新聚焦 → 定小目标
- 理解困难：换个角度 → 找工具帮忙 → 先跳过
- 动力不足：想想目标 → 给自己奖励 → 换个环境

## 🎮 Session 1 - 思维健身房

### 🏃 训练任务：注意力扫描挑战

**背景**：
就像跑步需要先热身，思维训练也需要先"照镜子"——看清自己的大脑在干什么。

**训练任务**：
1. 阅读一段故事（约200字），限时1分钟
2. 阅读时注意：你的注意力在哪里？有没有走神？
3. 时间到，快速回答：

**扫描测试**：
| 问题 | 答案 |
|------|------|
| 刚才我一直在认真看吗？ | |
| 中间走神了吗？ | |
| 走神几次？ | |
| 走神时在想什么？ | |

**思维健身等级**（1-5级）：
- 1级：不知道自己在走神
- 2级：感觉走神但说不清
- 3级：能说出走神几次
- 4级：能说出走神时想什么
- 5级：走神后能立刻拉回来

**我的等级：____级**

**教练提示**：
"思维健身"第一步是意识到自己在想什么。像照镜子一样，先看清楚自己的思维状态。

## 🎮 Session 3 - 3秒重启练习

### ⚡ 训练任务：快速调节训练

**情境**：
做英语作业时越来越无聊，脑子开始想别的事情...

**立刻启动"3秒重启"**：

| 秒数 | 做什么 | 默念 |
|------|--------|------|
| 第1秒 | 深呼吸 | "呼~" |
| 第2秒 | 想想为什么做 | "做完了可以..." |
| 第3秒 | 定个小目标 | "先做完这篇！" |

**实战练习**：
用3秒重启法，专注完成3道英语题。

**检查**：
- 小目标完成了吗？
- 动力恢复了吗？
- 这个方法有用吗？

**调节策略库**（选一个试试）：
| 情况 | 方法 |
|------|------|
| 太难 | 先做简单的1题 |
| 无聊 | 想象做完能休息 |
| 太累 | 休息5分钟 |
| 不懂 | 换个例子想 |

## 🎮 Session 4 - 数学题思维追踪

### 🎯 训练任务：解题过程记录

**题目**：
"某商店销售一种商品，进价40元，售价60元。现在搞促销活动，买5件送1件。如果某公司要买30件，至少需要多少钱？"

**训练任务**：
1. 读题时判断：这道题难不难？____
2. 做的时候注意：思路清晰吗？____
3. 遇到困难了吗？怎么解决的？____
4. 做完后检查：做对了吗？____

**思维记录**：
```
我的解题思路：____

卡在哪里了：____

怎么解决的：____

这次元认知表现：____（好/一般/差）
```

**教练提示**：
做完题后想一想"我刚才怎么做的"，这比知道答案更重要！

## 🎮 Session 5 - 综合调节挑战

### 🔄 训练任务：复杂情况应对

**情境故事**：
物理课要做实验报告，感觉内容很多看不懂，而且有点烦躁...

**这是"内容难懂+心情烦躁"的双重困难！**

**Step 1：判断哪个更紧急**
- 是内容难懂更烦？
- 还是烦躁更烦？

**Step 2：选2个方法组合使用**
- 🔄 换个方式读（先看图/先看结论）
- 💺 站起来活动一下
- 🎯 先只搞懂1个点
- 💪 给自己打气："慢慢来！"

**Step 3：检查效果**
- 好点了吗？
- 还需要什么帮助？
