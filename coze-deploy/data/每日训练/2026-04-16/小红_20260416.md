## 🔋 能量守恒定律应用

### 核心公式回顾

**机械能守恒条件**：
$$E_{k1} + E_{p1} = E_{k2} + E_{p2}$$
仅当**只有重力/弹力做功**时，机械能守恒。

**能量守恒（普适）**：
$$E_{总初} = E_{总末}$$
各种形式的能量可以互相转换，但总量不变。

### 🎢 类型一：光滑轨道能量守恒

**例题1**：质量m=1kg的小球从光滑半圆形轨道最高点A由静止开始滑下，轨道半径R=0.8m。求小球滑到最低点B时的速度。

**解析**：
- 初始状态（A点）：$v_A = 0$，$h_A = 2R = 1.6m$
- 末状态（B点）：$h_B = 0$
- 机械能守恒：$mgh_A = \frac{1}{2}mv_B^2$
- $v_B = \sqrt{2gh_A} = \sqrt{2 \times 10 \times 1.6} = \sqrt{32} ≈ 5.66 m/s$

**关键点**：抓住初末状态，忽略中间过程的细节复杂性。

### 🌊 类型二：含摩擦的多过程分析

**例题2**：小物块从A点由静止滑下，经粗糙水平面BC后冲上光滑斜面CD。小物块质量m=2kg，A距地面h=1.25m，BC段摩擦系数μ=0.5，CD光滑。求物块在斜面上能上升的最大高度。

**解析**：
```
A → B：机械能守恒
B → C：动能定理（摩擦做负功）
C → D：机械能守恒
```

**步骤1**：A到B（机械能守恒）
$$mg h = \frac{1}{2}mv_B^2 \Rightarrow v_B = \sqrt{2gh} = 5 m/s$$

**步骤2**：B到C（动能定理）
$$-\mu mg \cdot s = \frac{1}{2}mv_C^2 - \frac{1}{2}mv_B^2$$

**步骤3**：C到D（机械能守恒）
$$\frac{1}{2}mv_C^2 = mgh_{max}$$

**思维方法**：分段分析，每段选择最合适的定理/定律。

## 💥 动量守恒定律应用

### 核心公式回顾

**动量守恒条件**：
$$\sum p_{初} = \sum p_{末}$$
系统**不受外力**或**外力远小于内力**时，动量守恒。

**碰撞分类**：

| 类型 | 特点 | 能量关系 |
|------|------|---------|
| **完全弹性碰撞** | 恢复系数e=1 | 动能守恒 |
| **完全非弹性碰撞** | 共速 | 动能损失最大 |
| **非弹性碰撞** | 0<e<1 | 动能部分损失 |

### 🎯 类型三：碰撞问题

**例题3**：质量$m_1 = 2kg$的小球以$v_1 = 10 m/s$的速度与静止的质量$m_2 = 3kg$的小球发生一维对心碰撞。求：
（1）若为完全弹性碰撞，碰撞后各自速度
（2）若为完全非弹性碰撞，碰撞后共同速度

**解析**：

（1）完全弹性碰撞：
$$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$$
$$\frac{1}{2}m_1v_1^2 = \frac{1}{2}m_1v_1'^2 + \frac{1}{2}m_2v_2'^2$$

联立解得：
$$v_1' = \frac{m_1 - m_2}{m_1 + m_2}v_1 = \frac{2-3}{5} \times 10 = -2 m/s$$
$$v_2' = \frac{2m_1}{m_1 + m_2}v_1 = \frac{4}{5} \times 10 = 8 m/s$$

（2）完全非弹性碰撞：
$$v' = \frac{m_1v_1 + m_2v_2}{m_1 + m_2} = \frac{20}{5} = 4 m/s$$

**关键技巧**：记住弹性碰撞速度公式可以直接使用！

### 🚀 类型四：反冲现象

**例题4**：总质量M=100kg的船静止在湖面上，质量m=20kg的人从船头走到船尾。船长L=5m。求人和船的运动情况。

**解析**：
- 系统动量守恒：$0 = m v_人 + (M-m) v_船$
- 相对位移关系：$v_人 t - v_船 t = L$（船相对岸的位移差等于船长）

设人相对船的速度为$v_{相对}$，则：
$$m(v_船 + v_{相对}) + (M-m)v_船 = 0$$
$$mv_{相对} + Mv_船 = 0 \Rightarrow v_船 = -\frac{m}{M}v_{相对}$$

**思维方法**：抓住动量守恒和相对运动的约束关系。

## ⚡ 电荷守恒与质量守恒

### 电荷守恒定律

**内容**：电荷既不会凭空产生，也不会凭空消失，只能从一个物体转移到另一个物体，或者从物体的一部分转移到另一部分。

**典型应用**：

**例题5**：两个相同的金属小球A、B分别带电$q_A = +6\mu C$和$q_B = -2\mu C$。让它们接触后分开，求分开后各带电量。

**解析**：
- 总电荷守恒：$Q_{总} = q_A + q_B = 6\mu C + (-2\mu C) = 4\mu C$
- 两个相同导体接触后电荷均分：$q_A' = q_B' = \frac{Q_{总}}{2} = 2\mu C$

### 质量守恒定律（化学）

**例题6**：将10g氢气在氧气中完全燃烧，问最多能生成多少克水？

**解析**：
- 化学方程式：$2H_2 + O_2 \xrightarrow点燃 2H_2O$
- 质量守恒：反应物总质量 = 生成物总质量
- 水的质量 = 氢气质量 + 消耗氧气质量

**计算**：由化学计量比，4g H₂对应36g H₂O
$$m_{H_2O} = 10 \times \frac{36}{4} = 90g$$

## 🎯 综合应用：多守恒量问题

### 🔥 高级挑战题

**例题7**（综合）：光滑水平面上，质量为M的滑块通过轻质弹簧与墙壁连接。一质量为m的子弹以速度$v_0$水平射入滑块并留在其中。

（1）求子弹射入瞬间系统的动能损失
（2）求弹簧被压缩最大时，子弹和滑块的共同速度
（3）求弹簧的最大压缩量

**解析**：

**（1）动量守恒求共同速度**：
$$mv_0 = (M+m)v_1 \Rightarrow v_1 = \frac{mv_0}{M+m}$$

**动能损失**：
$$\Delta E_k = \frac{1}{2}mv_0^2 - \frac{1}{2}(M+m)v_1^2 = \frac{1}{2}mv_0^2\left[1 - \frac{m}{M+m}\right] = \frac{1}{2}\frac{Mm}{M+m}v_0^2$$

这部分能量转化为弹簧的弹性势能（假设完全转化为内能等）。

**（2）弹簧压缩最大时**：
- 能量守恒（机械能守恒，因为弹力是保守力）
- 子弹和滑块再次共速时动能为 $\frac{1}{2}(M+m)v_1^2$
- 全部转化为弹性势能：$E_p = \frac{1}{2}k x_{max}^2 = \frac{1}{2}(M+m)v_1^2$

**（3）最大压缩量**：
$$x_{max} = v_1\sqrt{\frac{M+m}{k}}$$

**学霸方法总结**：
1. **第一阶段（碰撞）**：动量守恒，能量不守恒
2. **第二阶段（压缩）**：动量守恒（系统不受外力），机械能守恒
3. 关键是**分清不同阶段**，选择合适的守恒定律

## 📊 解题策略总结

### 守恒定律选择流程

```
                    开始分析问题
                          ↓
              ┌───────────────────────┐
              │ 是否有外力作用？      │
              └───────────────────────┘
                      ↓
         ┌────────────┴────────────┐
         ↓                        ↓
       无外力                    有外力
         ↓                        ↓
    动量守恒 ✓              ┌──────────────────┐
         ↓                  │ 外力做功情况？  │
    是否只有重力/弹力？     └──────────────────┘
         ↓                        ↓
    ┌────┴────┐            ┌──────┴──────┐
    ↓         ↓            ↓             ↓
   是        否       只有重力      有其他力
    ↓         ↓         ↓             ↓
 机械能守恒  能量守恒  机械能守恒    动能定理
```

### 选择守恒定律的优先顺序

1. **动量守恒** → 当外力为零或可忽略时（碰撞、爆炸、反冲）
2. **机械能守恒** → 当只有保守力做功时
3. **能量守恒** → 当涉及多种能量转换时（热能、内能等）

小物块从高h的光滑斜面顶端滑下，在水平面上冲上粗糙斜面。已知物块质量m=1kg，h=0.8m，粗糙斜面摩擦系数μ=0.2。求物块能上升的最大高度。

**答案**：0.6m（提示：动能→势能+摩擦热）

### 🌟🌟 第二关：动量守恒综合（必做）

质量m₁=3kg的物体A以v₁=4m/s向右运动，与静止的m₂=2kg的物体B发生弹性正碰。求碰后各自速度。

**答案**：v₁'=0.4m/s向右，v₂'=6.4m/s向右

一颗子弹质量m=0.02kg，以v=600m/s的速度射入静止在光滑水平面上的木块（M=2kg），射穿后子弹速度v₁=100m/s。求：
（1）木块获得的速度
（2）系统损失的机械能
（3）若木块与弹簧连接，求弹簧的最大压缩量（k=1000N/m）

**答案**：（1）v₂=5m/s （2）ΔE=3500J （3）x=0.1m
