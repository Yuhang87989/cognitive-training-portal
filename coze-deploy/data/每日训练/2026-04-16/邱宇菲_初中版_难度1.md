## Session 3 - 守恒思维：能量守恒应用

### ⚡ 机械能守恒应用

**任务名称**：机械能守恒实战

**机械能守恒条件**：
只有重力或弹力做功时，机械能守恒
机械能 = 动能 + 势能

**例题分析**：
一个小球从20m高处自由落下（忽略空气阻力）
求：小球落地时的速度

**解题步骤**：
1. 明确守恒条件：只有重力做功 → 机械能守恒
2. 确定初始状态：高度20m，速度0
   初始机械能 E₁ = ½mv² + mgh = 0 + m×10×20 = 200m
3. 确定结束状态：高度0，速度v
   结束机械能 E₂ = ½mv² + 0 = ½mv²
4. 建立守恒方程：E₁ = E₂
   200m = ½mv²
   v² = 400
   v = 20m/s

**练习题**：
一个物体从10m高处落下，不考虑空气阻力，落地时速度是多少？（g=10m/s²）

## Session 4 - 守恒思维：动量守恒入门

### 🚀 动量守恒初步

**任务名称**：运动物体的"量"

**动量概念**：
动量 = 质量 × 速度
符号：p = mv
单位：kg·m/s

**动量守恒条件**：
系统不受外力或外力之和为零时，动量守恒

**动量守恒示例：碰撞**

**碰撞前**：
- 物体A：质量2kg，速度3m/s → p₁ = 6 kg·m/s
- 物体B：质量1kg，速度0 → p₂ = 0

**碰撞后**：
如果两个物体粘在一起，速度为v
总质量 = 3kg
动量守恒：6 + 0 = 3v
v = 2m/s

**练习：动量守恒计算**
两个滑冰的人，质量分别是50kg和60kg：
- 小明（50kg）以3m/s滑向静止的小红（60kg）
- 撞在一起后一起运动
- 求共同速度

## Session 5 - 守恒思维：守恒法vs过程法

### ⚖️ 选择最优方法

**任务名称**：守恒法与过程法的对比

**方法对比**：

**过程法**：
- 从头开始一步步计算
- 优点：思路清晰
- 缺点：步骤多，容易出错

**守恒法**：
- 利用守恒关系直接建立方程
- 优点：简洁高效
- 缺点：需要判断守恒条件

**例题对比**：
问题：一个小球从5m高处自由落下，求落地时的速度。

**方法一（过程法）**：
v² = 2gh = 2×10×5 = 100
v = 10m/s

**方法二（守恒法）**：
初始机械能 = 结束机械能
mgh = ½mv²
v = √(2gh) = √100 = 10m/s

**哪种方法更好？**
这道题：两种方法差不多

**练习：选择合适的方法**
问题：从同一高度释放两个小球（一个轻一个重），哪个先落地？
你的选择：_______
选择理由：_______
（提示：这道题不需要计算，守恒法反而不好用）

**方法选择决策树**：
1. 问题复杂，中间步骤多？→ 用守恒法
2. 只需要开始和结束状态？→ 用守恒法
3. 需要理解过程？→ 用过程法
4. 不确定？→ 两种都试试

**训练要点**：学会根据问题特点选择最优方法。

## Session 6 - 守恒思维：多守恒量综合

### 🔄 综合守恒问题

**任务名称**：多个守恒量一起用

**综合问题分析**：

**问题：能量和动量都守恒的碰撞**
一个质量2kg的小球（速度4m/s）撞上一个静止的质量2kg的小球，碰撞后两个小球运动方向成90°角，求各自的速度。

**分析**：
1. 动量守恒（x方向）：
   2×4 = 2v₁cosθ
   8 = 2v₁cosθ
   4 = v₁cosθ ... (1)

2. 动量守恒（y方向）：
   0 = 2v₂ + 2v₁sinθ
   v₁sinθ = -v₂ ... (2)

3. 能量守恒（弹性碰撞）：
   ½×2×4² = ½×2×v₁² + ½×2×v₂²
   16 = v₁² + v₂² ... (3)

## Session 7 - 守恒思维：守恒的学科迁移

### 🌉 物理守恒到数学思维

**任务名称**：守恒思维的迁移

**守恒思想的数学应用**：

**数学中的守恒**：
等式两边守恒
如果 a = b，那么 a + c = b + c
如果 a = b，那么 a × c = b × c

**练习：等式守恒**
如果 x + 5 = 10
两边同时减去5：x = 5 ✓

如果 2x = 14
两边同时除以2：x = 7 ✓

**守恒思维解方程**：
方程 3x + 2 = 17
分析：等式两边守恒
步骤1：两边减2 → 3x = 15
步骤2：两边除以3 → x = 5
验证：3×5 + 2 = 17 ✓

**守恒在证明中的应用**：
证明：a(b+c) = ab + ac
这就是乘法对加法的分配律（也是一种守恒）
