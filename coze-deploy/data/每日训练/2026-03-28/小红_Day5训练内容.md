## Session 1/8

### 🎯 注意力训练模块：高级认知负荷管理游戏

**任务目标**：在2.5分钟内同时处理四重认知任务，优化注意资源分配

**模拟场景**：你是一名航空管制员，需要同时监控雷达、协调通话、处理紧急情况、更新航班信息

**四重任务整合**：
1. **视觉任务**：监控雷达屏幕上4架飞机的实时位置和航向
   - 目标：识别偏离航线的飞机（每15秒检查一次）
   - 干扰：其他6架正常飞机的信号
2. **听觉任务**：监听飞行员通话，提取关键信息
   - 目标：记录特定航班（CA123）的高度和速度变更
   - 干扰：同时3个频道的通话，背景无线电噪音
3. **记忆任务**：保持最新航班信息在工作记忆中
   - 目标：记忆4架重点航班的当前状态（延误/正常/紧急）
   - 干扰：信息每20秒更新一次
4. **计算任务**：简单时间/距离计算
   - 目标：计算两架相向飞行飞机的预计相遇时间
   - 干扰：需要在30秒内完成计算

**资源分配优化**：
- 系统提供实时表现反馈
- 根据任务重要性动态调整优先级
- 学习在不同认知负荷下保持效能

**积分规则**：
- 每项任务正确完成：+15分
- 四重任务同步处理良好：额外+30分负荷管理奖励
- 资源分配策略有效：+20分优化奖励

**中级难度调整**：
- 反应时间窗口：1.5秒（适度压力）
- 干扰程度：中等偏高
- 任务复杂度：四重整合，需策略应用
- 负荷水平：需主动认知控制

### 🧠 记忆力训练模块：高级记忆策略体系构建游戏

**任务目标**：针对复杂信息结构，设计多策略组合的记忆方案

**记忆材料**：细胞生物学知识体系
- 细胞结构：细胞膜、细胞核、线粒体、内质网、高尔基体、溶酶体、核糖体、细胞骨架
- 功能对应：各结构的主要功能和工作机制
- 相互关系：能量转换、蛋白质合成、物质运输等过程

**策略体系构建**：
1. **结构分析阶段**：识别信息的层级关系和连接模式
2. **策略匹配阶段**：为不同信息类型匹配合适的记忆策略
   - 结构名称→位置法（想象细胞内部布局）
   - 功能对应→图像联想（将功能转化为生动图像）
   - 相互关系→概念图法（构建逻辑关系网络）
3. **整合应用阶段**：将多种策略有机组合，形成统一记忆方案

**游戏流程**：
1. 分析信息结构复杂度（1分钟）
2. 设计个性化策略组合方案（2分钟）
3. 应用方案记忆完整知识体系（3分钟）
4. 测试回忆效果并优化策略（2分钟）

**积分规则**：
- 成功回忆每个知识点：+12分
- 策略组合设计合理：+40分体系构建奖励
- 记忆效果超出基准：+30分效能优化奖励
- 提出创新策略组合：额外+25分创新奖励

**中级难度调整**：
- 信息复杂度：中等复杂知识体系
- 策略要求：需要2-3种策略组合
- 时间压力：适中，需要效率优化

### 📚 学习方法应用模块：学习规划架构师游戏

**任务目标**：掌握高级考试策略，构建系统化长期学习规划

**学霸专题**：高级考试策略 + 系统化长期规划

**具体内容**：
1. **高级考试策略学习（3分钟）**
   - 不同题型应对技巧：选择题快速排除法、主观题结构得分法
   - 压力管理策略：考试中情绪调节、注意力恢复技巧
   - 时间优化技术：基于题目难度动态调整时间分配
   - 复盘优化方法：考后系统分析，针对性改进

2. **系统化长期规划实践（4分钟）**
   - 学习规划框架构建：
     * 学期目标设定（SMART原则深化应用）
     * 各科资源平衡分配（时间/精力/优先级）
     * 能力发展路线图（基础知识→综合应用→创新思维）
   - 动态调整机制：
     * 进度监控指标设计
     * 风险识别与应对策略
     * 灵活调整原则与方法

**模拟挑战**：
- **考试策略模拟**：在模拟高压考试环境中应用高级策略
- **规划架构挑战**：为下学期制定完整学习发展路线图

**积分规则**：
- 完成高级策略学习：+50分
- 制定系统化长期规划：+60分
- 规划架构合理可行：额外+40分架构师认证积分
- 通过模拟挑战：获得"学习规划架构师"徽章
