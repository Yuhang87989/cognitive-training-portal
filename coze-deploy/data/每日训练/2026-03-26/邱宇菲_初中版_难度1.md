## Session 4 - 题型分类与模式识别

### 🔍 题型识别：看清题目的真面目

**任务名称**：火眼金睛识别题型

**超能力介绍**：
很多题目虽然看起来不同，但其实是同一类型！学会识别题型，就能举一反三。

**常见题型分类**：

**加法类题目**
特征：合并、增加、共多少
例：小明有5本书，小红有3本书，他们一共有多少本？

**减法类题目**
特征：减少、相差、还剩多少
例：小明有8个苹果，吃了3个，还剩多少？

**乘法类题目**
特征：几个几、共多少、倍数
例：每排有6个座位，有4排，共有多少座位？

**除法类题目**
特征：平均分、每组多少、包含几个
例：20个苹果平均分给4个小朋友，每个小朋友几个？

**练习内容**：

**练习1：题型识别**
判断以下题目属于哪一类：
1. "3个同学每人折了5只纸鹤，一共折了多少只？" → 类型：______
2. "把12块饼干平均分给3个人，每人几块？" → 类型：______
3. "小明比小红多3本书，小红有7本，小明有几本？" → 类型：______

**练习2：模式匹配**
找出与例题相似的题目类型：
例题：小明有10元，买笔用了3元，还剩多少？
匹配题目：
A. 小明有5支笔，又买了3支，现在有几支？
B. 小明有8本书，小红有5本书，他们相差几本？

答案：与例题相似的是______，原因是______

## Session 6 - 分步解题训练

### 📝 分步解决：一步一步来

**任务名称**：慢以致远 - 分步解题

**练习内容**：

**练习1：简单分步**
15 + 27 = ？
```
分步计算：
第一步：15 + 20 = 35
第二步：35 + 7 = 42
答案：42
```

**练习2：复杂分步**
小明买了3本笔记本，每本8元，又买了一支笔3元，一共花了多少元？

```
分步解题：
第一步：笔记本多少钱？3 × 8 = 24元
第二步：一共花了多少？24 + 3 = 27元
答案：27元
```

**练习3：综合分步练习**
李老师买了20本故事书，分给5个班级，每个班级分4本，还剩多少本？

```
分步解题：
第一步：_____________
第二步：_____________
第三步：_____________
答案：_____________
```

**分步解题技巧**：
- 每一步都要写在纸上

- 遇到难题就分解成小步骤
