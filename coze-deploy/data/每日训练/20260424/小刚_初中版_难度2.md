## 🔍 第4次训练：思维放大镜训练

**模块**：学习方法 - 元认知训练  
**时长**：约2分钟

### 训练目标
学习使用"思维放大镜"观察自己做作业时的想法

### 今日学习内容

**🔬 思维放大镜使用四步法**

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   🔬 思维放大镜使用四步法                       │
│                                                 │
│   【第1步：暂停】                              │
│   ─────────────────────────────────────        │
│   → 做作业时，突然停下来                        │
│   → 问自己："我现在在想什么？"                 │
│   → 就像按下了暂停键                           │
│                                                 │
│   【第2步：观察】                              │
│   ─────────────────────────────────────        │
│   → 看看脑子里在想什么                         │
│   → 是题目相关，还是在想别的？                  │
│   → 脑子里是画面还是声音？                      │
│                                                 │
│   【第3步：记录】                              │
│   ─────────────────────────────────────        │
│   → 简单快速记下来                             │
│   → 比如：                                       │
│   → "在想动画片结局"                           │
│   → "在想等下去吃什么"                         │
│   → "在想这道题怎么做"                        │
│                                                 │
│   【第4步：调整】                              │
│   ─────────────────────────────────────        │
│   → 如果走神了，把注意力拉回来                  │
│   → 如果在想题目，继续加油！                   │
│   → 给自己打个分                               │
│                                                 │
└─────────────────────────────────────────────────┘
```

**💡 练习时刻**

用上面的四步法，观察自己思考这道题时的思维：

**题目**：计算 15 × 7 = ?

- 暂停时，你在想什么？
- 观察时，脑子里是什么画面？
- 记录：_____
- 调整后是否更专注了？
