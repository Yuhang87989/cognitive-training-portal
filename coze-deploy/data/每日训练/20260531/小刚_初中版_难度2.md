## 🎯 第1次训练：理解PBL的核心价值

**模块**：项目式学习原理  
**时长**：约2分钟

### 什么是项目式学习（PBL）？

项目式学习（Project-Based Learning）是一种**以项目为载体、以问题为导向**的学习方式。学生围绕一个真实问题，通过调研、实践、创造来获取知识和技能。

### 项目式学习 vs 传统学习

| 维度 | 传统学习 | 项目式学习 |
|-----|---------|-----------|
| 目标 | 掌握知识点 | 解决真实问题 |
| 动力 | 外部驱动（考试） | 内部驱动（好奇心） |
| 知识来源 | 教科书 | 多元来源 |
| 输出形式 | 试卷/作业 | 作品/方案/产品 |
| 能力培养 | 记忆/理解 | 分析/应用/创造 |

### 项目式学习的认知价值

1. **知识整合** - 需要调用多个学科的知识
2. **深度学习** - 解决真实问题比做题更有意义
3. **元能力培养** - 计划、执行、复盘等可迁移能力
4. **学习动机** - 成就感驱动，自主性更强

### 优秀项目的特征

| 特征 | 说明 | 判断标准 |
|-----|------|---------|
| 真实性 | 解决真实问题 | 问题来源于实际 |
| 复杂性 | 需要多步骤 | 不是一步能完成的 |
| 开放性 | 没有标准答案 | 可以有多种解决方案 |
| 意义性 | 有实际价值 | 能帮助自己或他人 |

### 项目选择矩阵

| 维度 | 我感兴趣 | 我能做到 | 有社会价值 |

- 分析和解决身边的一个实际问题

- 深入研究一个感兴趣的话题
- 例：探索数学在游戏中的应用

**类型C：创意制作型**
- 用所学知识创造一个作品
- 例：制作一个学习工具/APP原型

**类型D：调查研究型**
- 调查一个社会现象并分析
- 例：班级同学睡眠质量调查

## 🧩 第3次训练：WBS任务分解技术

**模块**：任务拆解方法  
**时长**：约2分钟

### 什么是WBS？

WBS（Work Breakdown Structure，工作分解结构）是一种将复杂项目分解为可管理单元的系统方法。

### 分解原则

| 原则 | 说明 | 示例 |
|-----|------|------|
| 100%原则 | 所有工作都要分解 | 不能有遗漏 |
| 可交付原则 | 每个单元都可交付成果 | 能看到具体产出 |
| 80小时原则 | 每个任务不超过80小时 | 建议更短（1-3天） |

### 分解示例：制作"校园植物地图"

```
Level 1：项目总目标
     ↓
Level 2：主要阶段
    ├── 阶段1：调研准备
    ├── 阶段2：实地勘察
    ├── 阶段3：资料整理
    └── 阶段4：地图制作
     ↓
Level 3：具体任务
    ├── 1.1 确定植物种类范围
    ├── 1.2 学习植物识别方法
    ├── 1.3 准备记录工具
    ├── 2.1 规划勘察路线
    ├── 2.2 拍摄植物照片
    ├── 2.3 记录位置和特征
    └── ...以此类推
```

## 🎤 第7次训练：中期成果展示练习

**模块**：阶段性展示  
**时长**：约2分钟

### 为什么要练习展示？

项目做到一半，向他人展示阶段性成果可以：
1. **获得反馈** - 别人能看到你没发现的问题
2. **增强动力** - 看到进展更有干劲
3. **锻炼表达** - 费曼输出能力的实践
4. **社交支持** - 获得他人的鼓励和建议

### 展示框架（1分钟版）

```
【开场】
"大家好，我的项目是______，目标是______"

【进展】
"目前完成了______，主要成果是______"

【挑战】
"目前遇到的问题是______"

【计划】
"下一步计划是______"

【请求】
"希望获得______方面的建议"
```

### 费曼技巧在展示中的应用

- **简化**：不用专业术语，用听众能理解的话
- **比喻**：用生活例子解释复杂概念
- **结构**：按"是什么-为什么-怎么做"组织

## 🌟 第8次训练：完整复盘方法论

**模块**：项目复盘  
**时长**：约3分钟

### 什么是复盘？

复盘来自围棋术语，指对局结束后重演一遍，审视每一步的得失。项目管理中的复盘是对整个项目过程的系统回顾。

### 四步复盘法

| 步骤 | 问题 | 回答 |
|-----|------|------|
| 回顾目标 | 当初想要什么？ | |
| 评估结果 | 实际做到了什么？ | |
| 分析原因 | 成功/失败的原因是什么？ | |
| 总结经验 | 下次怎么做不同？ | |
