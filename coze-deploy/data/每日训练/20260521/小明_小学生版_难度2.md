## 🎮 第1次训练：考试小妙招连连看

**模块**：注意力训练  
**时长**：约2分钟

### 游戏规则
快速认识考试前的4个小妙招，然后回答简单的问题。

### 今日题目

```
┌─────────────────────────────────────────────────┐
│  🎯 考试小妙招（小明版）                          │
│                                                 │
│  1️⃣ 看课本：把课本上的知识再看一遍               │
│  2️⃣ 改错题：把以前做错的题再做一遍               │
│  3️⃣ 做练习：做一点点练习，保持做题的感觉         │
│  4️⃣ 早点睡：考试前一天要早点睡觉               │
│                                                 │
│  ⏱️ 限时：20秒，记住这4个小妙招                  │
└─────────────────────────────────────────────────┘
```

**问题1**：考试前要把课本怎么样？再____一遍
**问题2**：以前做错的题要怎么样？再____一遍
**问题3**：考试前一天要早一点____？

**答案**：看、做、睡觉

### 挑战升级
- 🌟 基础：回答对1个问题
- ⭐⭐ 进阶：回答对2个问题
- ⭐⭐⭐ 小勇士：全部回答正确！

## 🧠 第2次训练：考前准备四步走

**模块**：记忆力训练  
**时长**：约2分钟

### 游戏规则
记住考试前准备的四个步骤，像走楼梯一样，一步一步来！

### 今日记忆内容

```
🚶 考试准备四步走：

第一步（考前3天）：
看课本 + 改错题
→ 目标：把基础知识记牢

第二步（考前2天）：
做练习 + 问老师
→ 目标：不会的题目搞懂

第三步（考前1天）：
收书包 + 早点睡
→ 目标：放松心情，准备好

第四步（考试当天）：
吃早餐 + 笑一笑
→ 目标：开开心心去考试
```

### 记忆挑战（难度2）
1. 考前3天要做哪两件事？
2. 考试当天早上要做什么？

**记忆小技巧**：把这四步想象成四级台阶，每天上一级，最后就到考试啦！😃
