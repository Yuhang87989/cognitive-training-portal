## Session 1 - 注意力热身（游戏化任务）

### 🎮 冲突抑制游戏：颜色词挑战

**任务名称**：专注力冲刺 - 挑战你的大脑

**游戏规则**：
1. 当看到颜色词时，说出字的书写颜色（不是字的意思！）
2. "红"字用蓝色写 → 说"蓝色"
3. "黄"字用绿色写 → 说"绿色"
4. 目标：克服本能反应，说出正确颜色！

**冲突大师等级**：
- 🎨🎨🎨 正确率85%以上 - 冲突抑制大师
- 🎨🎨 正确率70%以上 - 冲突抑制高手
- 🎨 正确率50%以上 - 冲突抑制学徒

**训练要点**：通过冲突任务训练大脑的抑制控制能力，提升抗干扰能力。

**任务名称**：大脑指挥官

**超能力原理**：
大脑有时候会"抢答"——看到一个词就自动理解它的意思。但学霸能控制这种本能，按要求完成任务。这就是"认知控制"能力！

**练习内容**：

**练习1：Stroop任务进阶**
快速说出字的颜色（不是字的意思）：
- "红"（蓝色）→ 说"蓝色"
- "蓝"（红色）→ 说"红色"
- "黄"（绿色）→ 说"绿色"
- "绿"（黄色）→ 说"黄色"

记录：正确数______个（共4个）

**练习2：数字-大小冲突**
规则：说"大"或"小"（不是说数字！）

- 看到"5"（字很大）→ 说"大"
- 看到"9"（字很小）→ 说"小"
- 看到"2"（字很大）→ 说"大"
- 看到"7"（字很小）→ 说"小"

记录：正确数______个（共4个）

**练习3：干扰抑制实战**

**抑制技巧**：

## Session 3 - 课后复习方法

### 📝 科学复习：让记忆更牢固

**任务名称**：复习方法大师

**超能力介绍**：
复习不是把学过的东西再看一遍，而是要主动回忆、检验、巩固！

**高效复习三步法**：

**第一步：主动回忆（不看书！）**
- 闭上眼睛
- 努力回忆今天学了什么
- 写下关键词
- 不要翻书！

**第二步：对照检查**
- 打开书，对照回忆的内容
- 找出遗漏的地方
- 标记没记住的内容

**第三步：针对性巩固**
- 重点复习没记住的内容
- 用自己的话解释
- 做相关练习

**练习内容**：

**练习1：主动回忆练习**

## Session 4 - 高效笔记整理

### 📓 笔记系统：康奈尔笔记法入门

**任务名称**：成为笔记整理达人

**超能力介绍**：
康奈尔笔记法是全世界最受欢迎的笔记方法！它把笔记本分成三部分，让笔记更有条理。

**康奈尔笔记法布局**：
```
+-----------+------------+
|   笔记区   |   线索区   |
|   (主栏)   |   (副栏)   |
+-----------+------------+
|          总结区         |
+-------------------------+
```

- **笔记区（左侧大区域）**：上课时记录主要内容
- **线索区（右侧小区域）**：课后写下提示词、问题
- **总结区（下方区域）**：写下这页的总结和思考

**练习内容**：

**练习1：康奈尔笔记模板**
为数学课"分数的意义"设计一页康奈尔笔记：
```
日期：3月27日
主题：分数的意义

+----------【线索区】---+
| ★ 什么是分数？       |
| ★ 分子分母的关系？   |
| ★ 如何比较大小？     |
+-----------+---------+
|                       |
|     【笔记区】         |
|                       |
|                       |
|                       |
|                       |
+-----------+-----------+
| 【总结区】            |
| 分数表示把整体       |
| 平均分后取其中几份   |
+----------------------+
```

**练习2：课堂笔记优化**
以下是一段课堂笔记，请优化整理：
原始笔记："老师讲了分数 分数的定义 分子分母 分数线 分数比较大小 同分母 异分母 例子 1/2 2/4"

优化后的笔记：
```
主题：分数的意义

笔记区：
- 分数定义：把整体平均分成若干份，取其中一份或几份
- 分数组成：分子/分母/分数线
- 分数比较：同分母看分子，分子大分数大

线索区：
□ 什么是"平均分"？
□ 1/2 = 2/4 吗？

总结区：
分数的本质是表示"部分与整体"的关系
```

**练习3：自己设计一页笔记**
选择今天学的一个内容，用康奈尔法整理：

```
日期：_____________

|                     |
+-----------+---------+
|                       |
|     【笔记区】         |
|                       |
|                       |
|                       |
|                       |
+-----------+-----------+
| 【总结区】            |
|                     |
+----------------------+
```

## Session 5 - 复习与笔记整合

### 🧠 笔记复习：让笔记发挥最大价值

**任务名称**：笔记复习循环

**超能力介绍**：
笔记整理完不是终点，而是学习的起点！定期复习和更新笔记，才能让知识真正变成自己的。

**笔记复习流程**：

**Step 1：当天复习（放学后）**
- 看线索区的问题，尝试回答
- 补充遗漏的内容
- 标记不熟悉的部分

**Step 2：3天后复习**
- 遮住笔记区，只看线索区
- 尝试完整复述内容
- 检查掌握程度

**Step 3：周末复习**
- 做一页笔记的"迷你测试"
- 找出薄弱环节
- 针对性加强

**练习内容**：

**练习1：当天复习练习**
使用你今天设计的康奈尔笔记进行复习：
- 看线索区问题，尝试回答：_____________
- 补充遗漏：_____________
- 标记不熟悉：_____________

**练习2：3天复习模拟**
假设今天是复习日，复习3天前的一页笔记：
- 遮住笔记区，尝试回答线索区问题
- 能回答______个（共3个）
- 不能回答的：_____________

**练习3：复习日历制定**
