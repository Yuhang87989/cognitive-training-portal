## Session 3 - 综合建模：真实问题建模流程

### 🏗️ 真实问题建模六步法

**任务名称**：用六步法解决真实问题

**六步建模法**：

**第一步：问题理解**
理解真实问题的背景和目标
示例：如何规划放学回家的时间？

**第二步：模型假设**
抓住本质，简化非关键因素
假设：忽略等红绿灯时间，假设匀速行走

**第三步：变量识别**
区分已知量和未知量
已知：距离3km，步行速度1.5m/s（约5.4km/h）
未知：需要时间

**第四步：建立关系**
运用公式或规律
公式：时间 = 距离 ÷ 速度

**第五步：求解计算**
进行计算
时间 = 3000m ÷ 1.5m/s = 2000s ≈ 33分钟

**第六步：验证与优化**
检验结果合理性
实际可能需要40-45分钟（考虑等红绿灯等）

**练习：用六步法建模**
问题：计算家里每月用电量

## Session 4 - 综合建模：运动问题建模

### 🏃 运动问题的综合建模

**任务名称**：分析日常运动问题

**问题**：小明每天骑自行车上学，家离学校2公里，他以10km/h的速度骑行，需要多少分钟到达？

**建模分析**：

**1. 物理模型**：
- 模型类型：匀速直线运动
- 已知量：距离s=2km，速度v=10km/h
- 未知量：时间t

**2. 数学模型**：
公式：v = s/t → t = s/v
代入：t = 2km ÷ 10km/h = 0.2h = 12分钟

**3. 验证优化**：
- 实际可能需要15分钟（考虑启动、停车）
- 可以加入加速度模型进行更精确计算

**练习：运动问题建模**
问题：小华从家出发去图书馆，距离1.5km，步行速度5km/h，骑自行车速度15km/h。如果走路需要多少时间？骑车呢？

## Session 5 - 综合建模：成本问题建模

### 💰 成本问题的综合建模

**任务名称**：做精明的消费者

**问题**：手机套餐选择
- 套餐A：月租30元，包含100分钟通话，超出0.2元/分钟
- 套餐B：月租50元，包含300分钟通话，超出0.1元/分钟
如果每月通话200分钟，选哪个套餐更划算？

**建模分析**：

**1. 建立成本模型**：
套餐A成本 = 30 + (200-100)×0.2 = 30 + 20 = 50元
套餐B成本 = 50元

**2. 守恒思维应用**：
总费用 = 基础费 + 超额费

**3. 比较分析**：
| 通话时间 | 套餐A费用 | 套餐B费用 | 更划算 |
|---------|----------|----------|-------|
| 100分钟 | | | |
| 200分钟 | | | |
| 400分钟 | | | |

**4. 结论**：
- 200分钟时，两个套餐费用相同
- 低于200分钟，套餐A更划算
- 高于200分钟，套餐B更划算

**练习：选择最优方案**
小明想买一个计算器
- 甲店：原价50元，打8折
