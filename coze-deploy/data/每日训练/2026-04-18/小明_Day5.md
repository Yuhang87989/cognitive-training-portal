### Session 2 - ⏱️ 3分钟（注意力进阶）
**任务名称**：双线追踪
**任务描述**：
听老师讲述一个有趣的校园故事，同时用手指在空中画"∞"字型。故事结束后回答3个问题。

**故事内容**：
小明今天要策划班级运动会。他需要在上午安排跑步比赛，下午安排跳绳比赛。他有10个同学参加跑步，8个同学参加跳绳，其中5个同学两个项目都参加。

**问题**：
1. 有多少同学只参加跑步？
2. 小明需要准备多少份奖品？
3. 如果跑步比赛每组4人，需要分成几组？

### Session 3 - ⏱️ 3分钟（记忆力训练）
**任务名称**：校园宝藏图
**任务描述**：
小明在学校里找到了一个"知识宝藏"，他把宝藏藏在学校的某个角落。现在给你学校的平面图，记住宝藏可能藏的位置（用序号标注），然后完成其他任务后再回忆。

**记忆内容**：
- 宝藏位置序号：3号建筑（图书馆）二楼第5个书架
- 线索1：宝藏不在一楼
- 线索2：宝藏离窗户不远
- 线索3：宝藏旁边有一盆绿色植物

**回忆测试**：
1. 宝藏藏在几号楼？
2. 在几楼？
3. 什么东西旁边？

### Session 4 - ⏱️ 3分钟（记忆力进阶）
**任务名称**：能量守恒小实验
**任务描述**：
阅读下面的"环保小卫士"故事，记住关键数据，然后回答问题。

**故事内容**：
小红家有一个太阳能热水器。每天早上，阳光给热水器补充能量，相当于给10杯水的温度加热。晚上家人洗澡使用能量，相当于5杯水的温度。如果当天下雨，没有阳光补充能量，那么到第二天早上，热水器里还剩相当于几杯水的能量？

**记忆挑战**：
1. 太阳能热水器每天补充多少能量？
2. 家人使用多少能量？
3. 雨天的能量变化是什么？

### Session 5 - ⏱️ 3分钟（学习方法 - 系统性思维）
**任务名称**：小小规划师 - 班级活动设计
**任务描述**：
你是班级活动的小小规划师！需要设计一个有趣的活动。

**任务要求**：
1. **确定目标**：你想让同学们学到什么？（如：学会合作、锻炼身体等）
2. **识别资源**：你有多少时间？有多少同学参加？需要什么道具？
3. **安排步骤**：把活动分成几个步骤，按顺序写下来
4. **检查完善**：想想有没有遗漏的环节？

**活动设计模板**：
```

⏰ 活动时间：___________分钟
👥 参加人数：___________人

📦 需要准备：
1. ___________
2. ___________

📝 活动步骤：
第1步：___________
第2步：___________
第3步：___________

⚠️ 需要注意：___________
```

### Session 6 - ⏱️ 3分钟（学习方法 - 守恒思维）
**任务名称**：环保小卫士 - 家庭节能方案
**任务描述**：
你发现家里的电费有点高！作为"环保小卫士"，设计一个节能方案。

**能量守恒视角思考**：
- 家庭能量来源：电、燃气
- 能量去向：灯光、电器加热、制冷等
- 守恒思想：减少不必要的能量"流失"，就是节约

**任务要求**：
1. 列出家里3个最费电的地方
2. 想出3个节能小妙招
3. 计算：如果每个妙招每天节约1度电，一周能节约多少度电？

**节能方案模板**：
```
🌍 家庭节能方案

⚡ 费电大户TOP3：
1. ___________
2. ___________
3. ___________

💡 节能小妙招：
妙招1：___________（预计每天节约_____度电）
妙招2：___________（预计每天节约_____度电）
妙招3：___________（预计每天节约_____度电）

📊 节能计算：
每天节约：_____度电
每周节约：_____度电（每天×7）
每月节约：_____度电（每周×4）

🌟 我家的节能口号：___________
```
