### Session 1 - ⏱️ 3分钟（注意力热身）
**任务名称**：复杂信息筛选
**任务描述**：
阅读以下工程问题摘要，提取关键变量和约束条件。

**问题背景**：
某城市计划建设一座小型水电站。水库容量为100万立方米，年均降雨量为1200毫米，其中60%的雨水可收集入库。水电站装有3台发电机组，每台额定功率为500千瓦，效率为85%。城市年用电需求为2000万千瓦时。

**提取任务**：
1. 水库有效容量（万立方米）
2. 年均可用水量（万立方米）
3. 发电机组总装机容量（千瓦）
4. 年均发电量（万千瓦时，考虑效率）
5. 是否能满足城市需求？

### Session 2 - ⏱️ 3分钟（注意力进阶）
**任务名称**：双约束条件追踪
**任务描述**：
在处理以下问题时，同时保持两个约束条件的追踪。

**问题**：
某物流公司需要配送货物。从仓库A到目的地B、C、D。仓库有货物100箱，其中甲类30箱，乙类40箱，丙类30箱。
- 甲类货物必须先送B
- 丙类货物必须最后送D
- 每辆车最多装20箱

**分析任务**：
1. 需要几辆车才能一次配送完？
2. 如果甲类有10箱改成了丙类，需要几辆车？
3. 写出一种可能的配送顺序

### Session 3 - ⏱️ 3分钟（记忆力训练）
**任务名称**：多学科知识网络激活
**任务描述**：
阅读以下综合问题，回忆相关知识点并建立关联。

**问题**：

2. 撞击过程中损失的机械能
3. 若该能量转化为热量，能加热多少克水上升1°C？（水的比热容c=4.2×10³ J/(kg·°C)）

**关联知识点**：
- 力学：自由落体公式、能量守恒
- 数学：一元二次方程、计算
- 热学：能量转化、比热容公式

**记忆挑战**：
完成问题后，回忆：
1. 自由落体速度公式
2. 机械能的计算方式
3. 能量守恒定律的表述

**记录初始条件**：

### Session 6 - ⏱️ 3分钟（学习方法 - 守恒思维）
**任务名称**：多守恒量综合应用
**任务描述**：
分析以下复杂问题中的多个守恒量。

**问题**：蹦床运动员的能量转化分析

质量为60kg的运动员从3m高处自由下落，落在蹦床上。蹦床最大形变量为1m，弹性系数k=15000N/m。忽略空气阻力。

**分析任务**：

**1. 机械能守恒分析**
- 初始状态（离蹦床1m高处）：
  - 重力势能：Ep = mgh = ____ J
  - 动能：Ek = ____ J
  - 机械能总量：E总 = ____ J

- 中间状态（刚接触蹦床时）：
  - 重力势能：Ep = ____ J
  - 动能：Ek = mv²/2 = ____ J
  - 机械能总量：E总 = ____ J

- 最低点状态：
  - 重力势能：Ep = ____ J
  - 弹性势能：E弹 = ____ J
  - 动能：Ek = ____ J
  - 验证守恒：E总 = ____ J

**2. 动量分析**
- 下落过程中的加速度：a = ____ m/s²
- 接触蹦床瞬间的动量：p = ____ kg·m/s

**3. 能量转化守恒**
- 最高点点能量 = 最低点点能量（忽略弹性损耗）
- 实际：运动员会反弹，说明____能守恒
