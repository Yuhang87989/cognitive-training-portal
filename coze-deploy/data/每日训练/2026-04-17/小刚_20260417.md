## 第3次训练（学习方法：创新思维）

### 💡 思维定式认知

**学习内容**：
```
🔑 什么是思维定式？
   思维定式是大脑形成的"自动化反应"，会让我们"想当然"

⚠️ 生活中常见的思维定式：

1. 功能固着
   只能想到物品的常规用途
   例：筷子只能夹菜？还能搅拌、压平、当尺子...

2. 经验主义
   觉得"以前这样做行，现在也行"
   例：上次这种题用这种方法，这次一定也行

3. 权威依赖
   觉得"老师/书上说的就是对的"
   实际上可能需要批判性思考

4. 框架限制
   在固定框架里思考，跳不出来
   例：数学只能用数学方法想

🎯 练习：生活中还有哪些思维定式？
```

**积分规则**：识别生活思维定式+200积分

## 第6次训练（学习方法：创新思维）

### 🧠 创新思考方法

**学习内容**：
```
🔧 基础创新思考方法

1️⃣ 逆向思考法
   正常：从A到B
   逆向：从B倒推回A
   
   例：证明"两直线平行，同位角相等"
       → 逆向：假设同位角不等，能推出什么矛盾？

2️⃣ 类比思考法
   把陌生问题转化为熟悉问题
   
   例：学习电流→想象成水流

3️⃣ 分解组合法
   把大问题拆成小问题
   
   例：一道复杂应用题→拆成3个小问题分别解决

📝 练习：
   用"逆向思考法"解决：如何证明 √2 是无理数？
   用"类比思考法"理解：平行线的性质
```

**积分规则**：完成逆向思考练习+200积分，类比练习+200积分
