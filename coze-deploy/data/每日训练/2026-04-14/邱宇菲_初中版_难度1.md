## Session 3 - 系统性思维：知识网络入门

### 🌐 什么是知识网络

**任务名称**：把知识串成网

**知识网络概念**：
知识网络 = 把相关的知识用线条连接起来，形成一张"网"

**为什么需要知识网络？**
1. 知识不再孤立
2. 更容易理解和记忆
3. 能发现知识间的联系
4. 学习效率更高

**知识网络三要素**：
- **节点**：知识点的概念（写在圈里）
- **连线**：知识间的联系（用线连接）
- **标签**：说明是什么关系（在线上写关系）

**示例：数学知识网络一角**
```
      运算
     ↙   ↘
   加法    乘法
    ↓     ↓
  整数   自然数
    ↘   ↙
     数
```

**我的知识网络尝试**：
构建一个"数"的迷你网络：
（把相关的数的关系画出来）

**训练要点**：理解知识网络的基本构成。

## Session 4 - 系统性思维：概念图绘制

### 📊 概念图绘制方法

**任务名称**：绘制你的第一张概念图

**概念图绘制步骤**：

**第一步：确定核心概念**
选择要构建网络的知识点
示例：围绕"三角形"构建

**第二步：列出相关概念**
三角形的相关概念：
- 内角和（180°）
- 分类（直角、锐角、钝角）
- 面积公式
- 周长公式
- 全等/相似

**第三步：建立联系**
用箭头和词语说明关系：
- 三角形 → 有 → 三个内角
- 内角和 → 等于 → 180°
- 三角形 → 分为 → 直角三角形

**第四步：整理布局**
- 核心概念放中间
- 相关概念放周围
- 用层级和位置表示重要性

**练习：绘制"四边形"概念图**
核心概念：_______
相关概念：_______
关系连线：_______
（尝试画出来）

**训练要点**：掌握概念图的绘制方法。

## Session 6 - 系统性思维：思维导图入门

### 🧠 思维导图绘制

**任务名称**：用思维导图整理笔记

**思维导图特点**：
- 中心主题在中间
- 分支向四周展开
- 每个分支一个关键词
- 可以用颜色和图画

**思维导图示例**：
主题：三角形
分支1：分类 → 直角三角形、锐角三角形、钝角三角形
分支2：性质 → 稳定性、内角和180°
分支3：公式 → 面积、周长
分支4：应用 → 建筑、测量

**练习：制作"物理"思维导图**

一级分支：_______
二级分支：_______
（尝试画一张简单的思维导图）

**思维导图优势**：
- 一目了然
- 便于记忆
- 激发创意
- 易于更新

**训练要点**：学习用思维导图整理知识。

## Session 7 - 系统性思维：关联分析

### 🔗 发现知识间的联系

**任务名称**：知识关联大发现

**关联类型**：

**类型一：包含关系**
示例：
- 平行四边形 → 四边形（包含）
- 正方形 → 矩形 → 平行四边形 → 四边形

**类型二：并列关系**
示例：
- 加法 ←→ 减法（互为逆运算）
- 乘法 ←→ 除法（互为逆运算）

**类型三：因果关系**
示例：
- 三角形两边之和大于第三边 → 可以判断能否组成三角形

**练习：发现数学关联**
1. 找一组包含关系的概念：_______
2. 找一组并列关系的概念：_______
3. 找一组因果关系的概念：_______

**关联思考**：
数学和物理有联系吗？
- 数学的____可以帮助理解物理的____
- 物理的____需要数学的____支持

**训练要点**：培养发现知识关联的能力。
