## 一、APP概述

### 1.1 产品简介

🧠 **认知训练门户** 是一款专为12-16岁青少年打造的在线认知能力训练平台。平台整合了AI智能推荐、认知游戏、思维训练、学习计划等多种功能，帮助青少年科学提升记忆力、注意力、逻辑推理等核心认知能力。

### 1.2 适用对象

| 目标用户 | 说明 |
|---------|------|
| 🎓 **中小学生** | 12-16岁，正在建立良好学习习惯的青少年 |
| 👨‍👩‍👧 **家长** | 希望科学辅助孩子认知发展的家长 |
| 📚 **教育工作者** | 需要认知训练资源的老师/教练 |

### 1.3 访问方式

您可以通过以下两种方式访问认知训练门户：

| 平台 | 访问地址 | 说明 |
|------|---------|------|
| 🌐 **GitHub Pages** | https://yuhang87989.github.io/cognitive-training-portal/ | 适合电脑浏览器访问 |
| 🤖 **扣子平台** | https://6mz7txgx3f.coze.site/ | 适合微信内直接打开 |

## 二、安装说明

> ⚠️ **重要提示**：目前平台正在完善PWA（渐进式Web应用）功能，以下介绍的是"添加到主屏幕"的Web App方式。后续将支持完整的PWA安装体验。

### 2.1 iOS设备安装（iPhone/iPad）

#### 方式一：Safari浏览器安装（推荐）

```
步骤1️⃣：打开Safari浏览器
     ↓
步骤2️⃣：访问 https://yuhang87989.github.io/cognitive-training-portal/
     ↓
步骤3️⃣：点击底部 Safari工具栏的「分享」按钮 📤
     ↓
步骤4️⃣：向下滚动，找到「添加到主屏幕」选项
     ↓
步骤5️⃣：点击「添加」，确认添加位置
     ↓
步骤6️⃣：完成！桌面上会出现认知训练图标 🎉
```

#### 效果示意
```
📱 iPhone桌面
┌─────────────────────────┐
│  ⚙️  📱  📷  🎵         │
│                         │
│   ┌─────────┐          │
│   │ 🧠      │          │
│   │认知训练 │  ← 点击打开 │
│   └─────────┘          │
│                         │
└─────────────────────────┘
```

#### 注意事项
- ✅ 必须使用 **Safari浏览器**，Chrome/微信内置浏览器不支持此功能
- ✅ 添加后会像原生APP一样全屏运行
- ✅ 首次打开可能会有加载时间，请耐心等待

## 六、常见问题FAQ

### ❓ 关于安装

#### Q1: 为什么iPhone添加到主屏幕后图标很小？
**A:** 这是iOS系统的正常显示。首次添加时图标可能会显示较小，使用几天后系统会自动调整。如果想立即调整，可以长按图标，选择「编辑主屏幕」，然后拖动图标到其他位置，系统会自动优化显示。

#### Q2: 安卓手机提示「不支持安装」怎么办？
**A:** 请确保：
- 使用Chrome、华为浏览器或系统默认浏览器
- 确认已连接网络
- 尝试刷新页面后重新操作
- 如果仍然不行，可以使用「在浏览器中打开」功能

#### Q3: 微信里无法添加到主屏幕怎么办？
**A:** 微信内置浏览器不支持此功能。请：
1. 点击微信右上角「···」
2. 选择「在浏览器中打开」
3. 选择Chrome或系统浏览器
4. 按照Android安装步骤操作

#### Q4: 每次打开都需要联网吗？
**A:** 是的，目前需要网络连接。后续PWA功能完善后，可以支持离线使用部分功能。

---

### ❓ 关于DeepSeek功能

#### Q5: DeepSeek对话显示「API Key无效」怎么办？
**A:** 请按以下步骤排查：
1. 检查Key是否完整复制（以`sk-`开头）
2. 检查是否有多余空格
3. 确认Key还有额度
4. 重新保存设置

#### Q6: 如何获取DeepSeek API Key？
**A:** 
1. 访问 https://platform.deepseek.com/
2. 注册并登录账号
3. 进入「API Keys」页面
4. 点击创建新Key并复制

#### Q7: API Key需要付费吗？
**A:** DeepSeek API按量计费，约¥1/百万tokens。新用户注册赠送$1额度，普通使用可维持较长时间。

#### Q8: 如何给DeepSeek充值？
**A:** 
1. 登录 DeepSeek开放平台
2. 点击右上角「余额」
3. 选择充值金额和支付方式
4. 完成支付

### ❓ 关于使用体验

#### Q9: 为什么有时候加载很慢？
**A:** 可能原因：
- 网络连接不稳定
- 同时使用人数较多
- 浏览器缓存较多

**建议**：刷新页面、清理浏览器缓存或换个网络环境。

#### Q10: 学习数据会丢失吗？
**A:** 数据存储在浏览器本地存储(localStorage)中，清除浏览器数据可能丢失。建议定期查看「我的报告」功能备份重要数据。

#### Q11: 如何重置学习进度？
**A:** 目前没有重置功能。如果需要重新开始，可以使用浏览器的「清除网站数据」功能，但这会清除所有数据，请谨慎操作。

#### Q12: 训练游戏有哪些？
**A:** 平台提供31+款游戏：
- 认知训练：23款（记忆翻牌、舒尔特方格、数字记忆等）
- 娱乐游戏：8款（2048、俄罗斯方块、贪吃蛇等）
